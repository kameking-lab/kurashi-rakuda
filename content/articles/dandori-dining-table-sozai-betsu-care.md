# ダイニングテーブルのお手入れと洗浄剤使用時の注意

## 全体の流れ

テーブルのお手入れは「毎日の乾拭き・水拭き→汚れに応じた洗浄剤選び→洗浄剤を安全に使う」の3段階で進めると効果的です。

1. 毎日の乾拭き・水拭き — 食べこぼしや水滴を放置しない
2. 汚れに応じた洗浄剤選び — 素材に合わない強い薬剤を避ける
3. 洗浄剤を安全に使う — 種類の異なる洗浄剤を混ぜない

## ステップ別チェックリスト

### 毎日のお手入れ

- [ ] 食事の後は乾いた布またはかたく絞った布でテーブルを拭く
- [ ] 水滴・調味料をこぼした場合はできるだけ早く拭き取る
- [ ] 熱い鍋・食器を直接置かず、鍋敷きやコースターを使う（天板の変色・傷を防ぐ一般的な目安）

### 汚れがひどいときの洗浄剤選び

- [ ] 住宅用・家具用の洗浄剤を使う場合はラベルの「用途」欄に対象素材が記載されているか確認する
- [ ] 消費者庁の表示規程では、住宅用又は家具用の洗浄剤には用途・使用上の注意（子どもの手の届かない場所に置く、誤飲時の応急処置等）の表示が義務づけられている
- [ ] 目立たない場所で試してから全体に使う（変色・傷みを避けるための一般的な目安）

### 洗浄剤を使うときの注意（混ぜない）

- [ ] 酸性タイプの洗浄剤と塩素系（アルカリ性）の洗浄剤を同時に使わない
- [ ] 消費者庁の基準では、定められた試験で1.0ppm以上の塩素ガスを発生する住宅用洗浄剤に「まぜるな危険」表示が義務づけられている
- [ ] 国民生活センターの報道発表では、塩素系洗浄剤と酸性洗浄剤を一緒に使用した実験で、数分後に室内の塩素ガス濃度が16ppmを超えたと報告されている
- [ ] 塩素系洗浄剤を使う場合は換気を十分に行い、他の洗浄剤と併用しない
