# 家計の固定費見直しと食費の週次管理の段取り

## 全体の流れ

家計を整える段取りは「固定費を洗い出す→見直しの優先順位をつける→食費を週ごとに管理する→統計データに基づく目安と比べる」の4段階で進めると効果的です。

1. 固定費を洗い出す — 毎月・毎年決まって出ていく支出を一覧にする
2. 見直しの優先順位をつける — 効果が大きく手間が少ないものから着手する
3. 食費を週ごとに管理する — 月単位より把握しやすく、使いすぎに早く気づける
4. 統計データに基づく目安と比べる — 家計調査のデータをもとにした計算ツールで確認する

固定費の見直しは一度行えば効果が続くため後回しにされがちな変動費（食費など）より先に着手すると効率的です。一方、食費のような変動費は都度の判断が必要なため、週単位で区切って振り返る習慣をつくることが管理のコツです。

## ステップ別チェックリスト

### 固定費を洗い出す

- [ ] 家賃・住宅ローン、保険料、通信費（携帯・インターネット）、サブスクリプションサービスを一覧にする
- [ ] 契約中のプランと実際の利用状況（データ通信量、利用頻度など）を照らし合わせる
- [ ] 家族カード・複数契約など重複している支払いがないか確認する
- [ ] 見直しにかかる手間（解約手続き・違約金の有無）も合わせてメモする

### 見直しの優先順位をつける

- [ ] 削減額が大きく手間が小さいものから優先して着手する（例: 使っていないサブスクの解約）
- [ ] 保険・通信プランなど比較検討が必要なものは、次の見直しタイミングを決めてスケジュールに入れる
- [ ] 違約金・解約金が発生するものは、契約更新月などタイミングを見て動く

### 食費を週ごとに管理する

- [ ] 1週間分の食材をまとめて買う日を決める
- [ ] レシートや家計簿アプリで週ごとの食費合計を記録する
- [ ] 週の途中で使いすぎに気づいたら、残りの日数で調整する
- [ ] 月末にまとめて振り返るのではなく、週ごとに振り返る習慣にする

### 統計データに基づく目安と比べる

- [ ] 総務省統計局「家計調査」に基づく世帯人員別の支出目安を確認する
- [ ] ご自身の手取り収入と世帯人員数を入力し、費目ごとの配分目安を確認する
- [ ] 目安と実際の支出を比べて、大きくずれている費目があれば理由を考える
