# 洗面所の片付け・収納の段取りと誤飲事故の防止

## 全体の流れ

洗面所の片付けは「誤飲事故のリスクを知る→洗剤・芳香剤の保管場所を見直す→洗面台下収納の湿気対策」の3段階で進めると効果的です。

1. 誤飲事故のリスクを知る — 洗面所は洗剤・芳香剤の誤飲事故が起きやすい場所
2. 洗剤・芳香剤の保管場所を見直す — 詰め替えボトルの使用や保管場所に注意する
3. 洗面台下収納の湿気対策 — 通気性を確保してカビを防ぐ

## ステップ別チェックリスト

### 誤飲事故のリスクを知る

- [ ] 国民生活センターには、自宅の洗面所に置いてあった洗濯用パック型液体洗剤を高齢者が誤って食べてしまった事例が報告されている
- [ ] 同様に、洗面台の下の棚に置いてあった洗濯用パック型液体洗剤を子どもが誤って口に入れた事例も報告されている
- [ ] 乳幼児がリードディフューザー（液体芳香剤）の液を誤飲し入院した事例も報告されている
- [ ] 家族に高齢者・乳幼児がいる場合は特に注意して収納場所を見直す

### 洗剤・芳香剤の保管場所を見直す

- [ ] 洗剤・芳香剤は子どもや高齢者の手が届きにくい場所に収納する
- [ ] 洗剤・殺虫剤などを飲料用のペットボトルや別容器に移し替えない
- [ ] 液体芳香剤（リードディフューザー等）は倒れにくい安定した場所に置く
- [ ] 認知症のある家族がいる場合は、ジュース等と見た目が紛らわしい容器を避ける

### 洗面台下収納の湿気対策

- [ ] 洗面台下など空気の流れが悪い収納スペースには、すのこなどを使って通気性を確保する
- [ ] 京都市の案内では、室内の湿度は60%を超えないようにすることがカビ対策の目安とされている
- [ ] 濡れたタオル・スポンジを収納内に入れたままにしない
