# 保活の全手順チェックリスト — 情報収集から入園までの流れ

## 全体の流れ

保活は大きく分けて次の4段階で進みます。いつから始めるか（逆算スケジュールの立て方）は別記事で扱っているため、ここではそれぞれの段階で「具体的に何をするか」を手順として整理します。

1. 情報収集 — 自治体の制度と地域の園を知る
2. 見学・比較 — 候補の園を絞り込む
3. 申込み — 自治体の窓口へ申請し、書類を保管する
4. 入園決定後の準備 — 健康診断・持ち物など、入園までの実務手続き

保活でつまずきやすいのは「見学で何を見ればよいか分からない」「申込み後に必要になる書類を後から取ろうとして取れない」「入園が決まってから何を準備すればよいか分からず慌てる」という3つの場面です。この記事では、各ステップで具体的に確認・準備すべき項目をチェックリストの形でまとめます。自治体によって申込みの受付時期・必要書類・様式は異なるため、時期そのものはお住まいの自治体窓口で必ず確認してください。

## ステップ別チェックリスト

### ステップ1: 情報収集

- [ ] お住まいの自治体の保育案内（利用のしおり等）を入手する（役所窓口・自治体サイトのいずれでも入手可能な場合が多い）
- [ ] 認可保育所・認定こども園・地域型保育（小規模保育等）・認可外保育施設など、園の種類ごとの違いを把握する
- [ ] 保育の必要性の認定区分（就労・妊娠出産・疾病・介護等）のうち、自分がどの区分に該当するかを確認する
- [ ] 自治体の窓口や保活相談会・入園説明会の日程を確認する
- [ ] きょうだいが別の園になる可能性（同一園への優先利用の有無）を自治体に確認する
- [ ] 幼児教育・保育の無償化の対象年齢を把握しておく（全世帯は3歳児クラスから、住民税非課税世帯は0〜2歳児クラスから無償化の対象）

### ステップ2: 見学・比較

- [ ] 候補となる園をリストアップする（自宅からの距離だけでなく、通勤経路上にある園も候補に入れる）
- [ ] 見学の予約を取る（人気園は見学自体の予約が埋まりやすいため、情報収集と並行して早めに動く）
- [ ] 保育方針・1日のスケジュール・園庭や設備の様子を確認する
- [ ] 延長保育の対応可否と、朝夕の預かり体制を確認する
- [ ] 病児・病後児保育や体調不良時の対応方針（お迎えの連絡基準等）を確認する
- [ ] アレルギー対応・給食（自園調理か外部搬入か）を確認する
- [ ] 持ち物のルール（布おむつ・紙おむつの可否、着替えの枚数、名前付けの方法）を確認する
- [ ] 保護者の負担（行事参加の頻度、保護者会・係の有無、送迎ルール）を確認する
- [ ] 保育料以外にかかる実費（給食費・行事費・教材費等）を確認する。無償化の対象になっても給食費等の実費は原則対象外になる
- [ ] 見学時の質問内容と回答をメモに残し、複数園を比較できるようにする
- [ ] 見学後、家庭内で候補園の優先順位をつけておく

### ステップ3: 申込み

- [ ] 申込み書類一式を自治体窓口またはサイトで入手する
- [ ] 就労証明書など、必要な添付書類を勤務先に依頼し、早めに準備する
- [ ] 希望順位をつけて複数の園を申込書に記載する（自治体ごとに記載できる希望園数の上限を確認する）
- [ ] 自宅から通所に時間がかかりすぎる園だけに絞らず、通える範囲で複数の候補を確保しておく。育児休業給付金の延長を申請する可能性がある場合、申込んだ園が自宅から片道30分以上かかる施設のみになっていると、延長の審査で不利になることがある
- [ ] 申込書類を提出したら、必ずコピーを取って保管する（電子申請の場合は申込内容の画面を印刷または保存する）。窓口の受付印は不要で、提出したものと同じ内容であればよい。この写しは提出後に自治体から再発行してもらうことが難しいため、後から必要になっても入手できないことがある
- [ ] 申込内容を後から変更した場合は、変更後の申込書の写しも保管しておく
- [ ] 「入所保留となることを希望する」といった意思表示の記載をしていないか、提出前に自分の記入内容を確認する
- [ ] 申込みの締め切り日を確認し、余裕をもって提出する

### ステップ4: 入園決定後の準備

- [ ] 内定通知（利用調整結果通知）の内容を確認する（入所可能な園・入所可能日）
- [ ] 内定の返答期限を確認し、期限内に意思表示する
- [ ] 園が指定する面談日・健康診断日を確認し、予定を確保する
- [ ] 児童票・健康記録・アレルギーの有無など、入園時に提出を求められる書類を準備する
- [ ] 保育料の決定通知を確認する。無償化の対象年齢（3〜5歳児クラスは全世帯、0〜2歳児クラスは住民税非課税世帯）に該当するかを確認する。認可外保育施設等を利用する場合は、無償化の上限額が3〜5歳児クラスで月37,000円、住民税非課税世帯の0〜2歳児クラスで月42,000円までである点も踏まえて費用を見積もる
- [ ] 給食費・行事費・教材費など、無償化の対象外となる実費の支払い方法（口座振替・現金徴収等）を確認する
- [ ] 持ち物リストを入手し、必要な物品を揃える
- [ ] 持ち物すべてに記名する（衣類・おむつ・コップ・タオル・お昼寝布団など、園から指定されたルールに沿って行う）
- [ ] 慣らし保育の期間と、その間の送迎・就業調整のスケジュールを職場と相談する
- [ ] 復職日を職場に伝え、必要であれば時短勤務等の申請を行う
- [ ] 内定を辞退する場合は、やむを得ない理由がない辞退だと後の育児休業給付金の延長要件に影響することがあるため、辞退前に理由を整理しておく
