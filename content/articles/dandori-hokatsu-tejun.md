# 保活の全手順チェックリスト — 情報収集から入園までの流れ

## 全体の流れ

保活は大きく分けて次の4段階で進みます。

1. 情報収集（自治体の制度・地域の園を知る）
2. 見学・比較（候補の園を絞り込む）
3. 申込み（自治体の窓口へ申請）
4. 入園準備（内定後の手続き）

自治体によって申込みの受付時期や必要書類が異なるため、早めにお住まいの自治体窓口で最新のスケジュールを確認することが最も重要です。

## ステップ別チェックリスト

### ステップ1: 情報収集
- [ ] お住まいの自治体の保育案内（利用のしおり等）を入手する
- [ ] 認可・認可外など、園の種類ごとの違いを把握する
- [ ] 自治体の窓口や相談会の日程を確認する

### ステップ2: 見学・比較
- [ ] 候補となる園をリストアップする
- [ ] 見学の予約を取る
- [ ] 見学時のチェックポイント（保育方針・行事・持ち物など）を決めておく

### ステップ3: 申込み
- [ ] 申込み書類一式を自治体窓口またはサイトで入手する
- [ ] 就労証明書など、必要な添付書類を準備する
- [ ] 申込みの締め切り日を確認し、余裕をもって提出する

### ステップ4: 入園準備
- [ ] 内定通知を確認する
- [ ] 面談・健康診断など、園から指定された手続きを行う
- [ ] 持ち物・名前付けなど、入園までの準備を進める
