# 洗濯機の臭いを防ぐための湿気・カビ対策の段取り

## 全体の流れ

洗濯機まわりの臭い対策は「原因を理解する→毎日の水滴・湿気対策→洗濯機本体の手入れ」の3段階で進めると効果的です。

1. 原因の理解 — カビは湿度が高い場所で発生し、水滴が残った場所に定着しやすい
2. 毎日の水滴・湿気対策 — 洗濯機まわりの水滴拭き取りと換気
3. 洗濯機本体の手入れ — 取扱説明書に沿った清掃コースの実施

東京都保健医療局「室内環境対策」（アレルギー情報navi.）では、カビは換気不足などで湿度が高くなると発生し、洗濯機の周辺などに残った水滴もカビの胞子が定着する場所になるとされています。まずは水滴を残さない習慣と換気を軸に対策を進め、洗濯機本体の清掃は取扱説明書の指示に従うことが重要です。

## ステップ別チェックリスト

### 準備段階

- [ ] 洗濯機の種類（ドラム式か縦型か）と設置場所の換気状況を確認する
- [ ] 取扱説明書で「槽洗浄」「クリーニング」コースの有無と使い方を確認する
- [ ] 洗濯機周辺の湿度計があれば数値を確認する（目安は60%以下）

### 毎日の水滴・湿気対策

- [ ] 洗濯機の周辺やふた・ゴムパッキンに残った水滴をタオルでよく拭き取る
- [ ] 脱水が終わったらふたを開けて内部の湿気がこもらないようにする
- [ ] 天気の良い日は窓を開けて換気し、室内の湿度を下げる
- [ ] 梅雨時期など換気だけで湿度が下がりにくい時期は除湿機の使用も検討する

### 洗濯機本体の手入れ

- [ ] 取扱説明書に記載された「槽洗浄」「クリーニング」コースの手順・頻度を確認して実施する
- [ ] コース終了後は排水口・ゴムパッキンの汚れを確認し、必要に応じて拭き取る
- [ ] 清掃後はふたを開けたまま乾燥させる
- [ ] 洗剤・柔軟剤の使用量は表示された目安量を超えないようにする（残留がカビや汚れの原因になりうるため）

### ドラム式・縦型で異なるポイント

| 項目 | ドラム式 | 縦型 |
|---|---|---|
| 湿気がこもりやすい部分 | ドア・ゴムパッキン周辺 | 洗濯槽と外槽の間（目視しにくい） |
| 日常のケア | ドアを開けてパッキン部分を乾燥させる | ふたを開けて槽内を乾燥させる |
| 清掃コースの有無 | 機種による（説明書で確認） | 機種による（説明書で確認） |

洗濯槽内部の構造や清掃コースの詳細は機種ごとに異なるため、頻度や薬剤量を含めて必ずお使いの機種の取扱説明書・メーカー公式情報で確認してください。
