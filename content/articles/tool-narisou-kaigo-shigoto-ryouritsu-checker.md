# 介護と仕事の両立制度チェッカーの「条件付き（要確認）」はどう決まる？ — 労使協定と法律の除外を読み分ける

## この記事はどのツールと対応している？

本記事は「介護と仕事の両立制度チェッカー」ツール（kaigo-shigoto-ryouritsu-checker）に対応しています。対象家族との続柄・要介護状態の見込み・雇用形態（勤続1年未満か、週の所定労働日数が2日以下か、有期契約か）・深夜に代わって介護できる同居家族の有無を選ぶと、育児・介護休業法が定める介護関係の両立支援制度6本（介護休業・介護休暇・所定外労働の制限・時間外労働の制限・深夜業の制限・所定労働時間の短縮等の措置）について、「対象」「条件付き（要確認）」「対象外」を一覧で表示します。各制度そのものの内容（対象家族の範囲や日数の意味）は別記事「介護と仕事の両立、使える休暇・休業制度は？」で解説していますので、本記事ではそちらと重複させず、ツールがなぜその判定結果を出すのか、判定ロジックに絞って説明します。

## 計算の考え方

### まず「対象家族」と「要介護状態」の2つの前提を確認する

ツールは最初に、選ばれた続柄が「対象家族」の範囲（配偶者・父母・子・配偶者の父母・祖父母・兄弟姉妹・孫）に入っているか、そして要介護状態（負傷・疾病又は身体上若しくは精神上の障害により2週間以上の期間にわたり常時介護を必要とする状態）に該当する見込みかどうかを確認します。

```
baseEligible = (続柄が対象家族の範囲内) かつ (要介護状態に該当する見込みがある)
```

どちらか一方でも満たさない場合、以降の判定はすべて省略され、6制度全部が「対象外」と表示されます。おじ・おば、配偶者の祖父母・配偶者の兄弟姉妹はこの範囲に含まれないため対象外です。また「要介護状態」は、介護保険の要介護認定（要介護1〜5等）とは別の基準であるため、要介護状態の見込みが「わからない」を選んだ場合は判定を続けつつ、この基準の違いを注記として表示します。

### 「労使協定があれば除外できる制度」と「法律で直接除外される制度」を分ける

ここがこのツールの一番の要点です。介護休業・介護休暇（週の所定労働日数が2日以下の場合のみ）・所定外労働の制限・所定労働時間の短縮等の措置の4制度は、勤務先が労使協定を結んでいる場合に**限り**、勤続1年未満や週の所定労働日数が2日以下の労働者を対象から除外できます。労使協定がなければ、その条件に当てはまっても対象になります。ツール側は労使協定の有無を知る手段がないため、これらの条件に該当する場合は「対象外」と断定せず、「条件付き（要確認）」として就業規則・人事担当への確認を促します。

```
if (勤続1年未満 または 週の所定労働日数が2日以下):
    介護休業・所定外労働の制限・所定労働時間の短縮等の措置 → "条件付き（要確認）"
if (週の所定労働日数が2日以下):
    介護休暇 → "条件付き（要確認）"
それ以外 → "対象"
```

一方、時間外労働の制限・深夜業の制限の2制度は、労使協定の有無にかかわらず**法律が直接**、勤続1年未満の労働者を除外の対象と定めています。この場合は労使協定を結んでいるかどうかを確認するまでもなく除外が確定するため、ツールは「条件付き」ではなく「対象外」と表示します。

```
if (勤続1年未満):
    時間外労働の制限・深夜業の制限 → "対象外"（確定。労使協定の有無を問わない）
```

この2つの区別を取り違えると、本来対象になるはずの人に「対象外」と誤って案内してしまう恐れがあるため、ツールの内部では常にこの2系統を分けて扱っています。

### 介護休暇だけ除外条件が違う理由

介護休暇の労使協定による除外要件は、他の制度と少し違います。2025-04-01の育児・介護休業法改正より前は「継続雇用期間6か月未満」の労働者も労使協定で除外できましたが、この改正でこの要件が廃止され、除外できるのは「週の所定労働日数が2日以下の労働者」だけになりました。つまり、介護休暇については勤続1年未満・6か月未満であることは除外の理由になりません。ツールが介護休暇の判定に勤続年数の入力を使わず、週の所定労働日数だけで判定しているのはこのためです。

### 深夜業の制限だけにある独自の除外要件

深夜業の制限には、他の5制度にはない独自の除外要件があります。深夜（午後10時〜午前5時）に代わって対象家族を介護できる同居の家族がいる場合、法律により深夜業の制限は請求できません。ツールは「はい・いいえ・わからない」の3択でこれを尋ね、「はい」なら「対象外」、「わからない」なら「条件付き（要確認）」、「いいえ」なら（勤続1年未満でない限り）「対象」と表示します。この要件は深夜業の制限だけのものなので、他の5制度の判定には使いません。また、週の所定労働日数が2日以下であることは、深夜業の制限の除外要件には含まれていないため、他の労使協定系の制度が「条件付き」になっていても、深夜業の制限だけは「対象」のままになることがあります。

## 結果の読み方

結果画面には6制度それぞれについて、バッジで「対象」「条件付き（要確認）」「対象外」のいずれかが表示され、その下に日数・回数・期間などの制度上の数値と、判定理由の注記が並びます。「対象」は入力からは除外要件に該当しないことが分かったという意味で、「条件付き（要確認）」は労使協定の有無・深夜の同居家族の有無など、ツールの入力だけでは判定できない条件があるという意味です。「条件付き（要確認）」が出た場合は、必ず勤務先の就業規則や人事担当に確認してください。

介護休業（対象家族ごとに通算93日、3回まで分割可）と所定労働時間の短縮等の措置（利用開始日から連続3年以上、2回以上利用可能な措置）は、日数・年数の桁が大きく違うため混同しやすい制度です。介護休業は「介護の体制を整えるためのまとまった休業」、所定労働時間の短縮等の措置は「介護休業を取得していない人が長く働き続けるための勤務時間の調整」という役割の違いを踏まえて読んでください。所定労働時間の短縮等の措置には、介護休業を取得している間は対象にならないという注記が常に表示されます。

時間外労働の制限（1か月24時間・1年150時間まで）と所定外労働の制限（残業そのものの免除）は別の制度で、同時に請求することはできません。ツールはこの2つを別々の行として表示しますが、実際にどちらを選ぶかは、残業をどこまで減らしたいかによって変わります。

介護休暇は、まとまった休業ではなく通院の付き添いなど日常的な用事に使う日数で、1年度に5日、対象家族が複数いる場合は10日まで取得できます。介護休業（対象家族ごとに93日）と桁が違うため、画面の数値だけを見て「介護休業より少ない」と単純比較しないよう注意してください。役割そのものが異なる制度です。

画面の下部には、令和7年（2025年）4月の改正で新設された会社の義務（介護に直面した旨を伝えると個別に周知・意向確認を受けられること、40歳に達すると情報提供を受けられること、会社が雇用環境整備をする義務を負うこと、介護のためのテレワーク導入が努力義務になったこと）も表示されます。これらは労働者が請求する制度ではなく、会社が果たすべき義務なので、チェック結果の「対象／対象外」とは切り離して読んでください。特にテレワークは義務ではなく努力義務のため、「テレワークができるようになった」と断定せず、「会社が導入するよう努めることとされた」という位置づけで案内しています。

## よくある疑問

**「条件付き（要確認）」は結局、対象なの？対象外なの？** どちらとも言い切れません。労使協定の有無という、ツールの入力だけでは分からない条件で結果が変わるため、断定を避けています。就業規則や労使協定の内容を人事担当に確認すれば、白黒はっきりします。

**有期契約（契約社員・パートなど）で働いていますが、介護休業は使えますか？** 「有期契約で働いている」を選ぶと、介護休業の判定は必ず「条件付き（要確認）」になり、独自の追加要件（介護休業開始予定日から起算して93日を経過する日から6か月を経過する日までに労働契約が満了することが明らかでないこと）の注記が表示されます。この要件を満たすかどうかは契約内容次第のため、ツールでは判定せず、原文のまま案内しています。

**要介護状態の見込みが「わからない」場合、判定は止まりますか？** 止まりません。育児・介護休業法の「要介護状態」は、介護保険の要介護認定とは別の基準（2週間以上、常時介護が必要な状態かどうか）で判断されるため、要介護認定を受けていなくても対象になり得ます。ツールは判定を続けたうえで、この基準の違いを必ず注記します。

**時間外労働の制限と深夜業の制限で、なぜ「勤続1年未満」の扱いが他の制度と違うのですか？** 介護休業・所定外労働の制限・所定労働時間の短縮等の措置は「労使協定があれば除外できる」規定であるのに対し、時間外労働の制限・深夜業の制限は法律が直接、勤続1年未満の労働者を除外の対象と定めているためです。前者は労使協定がなければ対象のままですが、後者は労使協定の有無を問わず除外が確定します。
