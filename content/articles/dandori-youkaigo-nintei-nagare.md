# 要介護認定の申請から結果が出るまでの流れ

## 全体の流れ

要介護認定は、介護保険サービスを利用するために必要な手続きです。流れは次の通りです。

1. 申請（お住まいの市区町村の窓口へ）
2. 訪問調査（市町村の認定調査員が本人・家族から聞き取りを行う）
3. 一次判定（訪問調査の結果と主治医意見書をもとにしたコンピュータ判定）
4. 二次判定（保健・医療・福祉の学識経験者で構成される介護認定審査会が、一次判定の結果と主治医意見書をあわせて審査）
5. 認定結果の通知（市町村が認定し、要支援1・2〜要介護1〜5の7区分、または非該当のいずれかが通知される）
6. 結果に応じたケアプランの作成・サービス利用開始

主治医意見書は、申請時に伝えたかかりつけ医に市町村から作成が依頼されます。訪問調査と主治医意見書がそろって初めて一次判定・二次判定に進めるため、かかりつけ医を早めに決めておくことが段取りの起点になります。

認定結果が届くまでには一定の期間がかかります。急ぎで介護サービスの利用を始めたい場合は、申請日にさかのぼって暫定的にサービスを利用できる制度もあるため、地域包括支援センターや市町村の窓口に早めに相談するとよいでしょう。

### 要介護度はどう決まるか（「要介護認定等基準時間」という物差し）

要支援・要介護の区分は、「要介護認定等基準時間」という、介護にどれくらいの手間がかかるかを示す指標（一次判定で推計される時間）によって決まります。厚生労働省の基準では、次のように区分されています。

- 要支援1: 25分以上32分未満
- 要支援2・要介護1: 32分以上50分未満（両者は同じ時間帯にあり、認知症の有無や状態の安定性などで振り分けられる）
- 要介護2: 50分以上70分未満
- 要介護3: 70分以上90分未満
- 要介護4: 90分以上110分未満
- 要介護5: 110分以上

ここで重要なのは、要介護1・2・3…という区分ごとに「要介護1は部分的な介護が必要」「要介護3は全面的な介護が必要」といった、症状の重さそのものを定義した公式な基準は存在しないという点です。厚生労働省は、要介護認定は病気の重さと必ずしも一致するとは限らない介護の必要度を判断するものであり、要介護認定等基準時間は実際に家庭で行われる介護時間そのものとも異なる「ものさし」だと説明しています。区分の重さを実際の症状の重さと同一視しないよう注意してください。

### 「要介護状態」は2つの意味がある

介護に関する制度を調べていると、「要介護状態」という言葉が2つの異なる意味で使われていることに気づきます。

- 介護保険の「要介護認定」: この記事で説明している、要支援1・2〜要介護5の7区分（または非該当）の認定。
- 労働基準法系の「要介護状態」: 介護休業・介護休暇など、仕事を休むための制度を使う条件となる状態。育児・介護休業法では「負傷、疾病又は身体上若しくは精神上の障害により、2週間以上の期間にわたり常時介護を必要とする状態」と定義されており、介護保険の要介護認定を受けているかどうかとは別の基準です。要介護認定を受けていなくても、あるいは要支援・非該当と判定されていても、2週間以上常時介護が必要な状態であれば介護休業・介護休暇等の対象になり得ます。

この2つを混同すると、「要介護認定が下りていないから会社の介護休業は使えない」といった誤解につながるため、区別しておくことが大切です。

## ステップ別チェックリスト

### ステップ1: 申請前の準備
- [ ] お住まいの市区町村の介護保険担当窓口・地域包括支援センターの連絡先を確認する
- [ ] 本人のかかりつけ医（主治医意見書を依頼する医師）を決めておく
- [ ] 介護保険被保険者証・本人確認書類・マイナンバーが分かるものを準備する
- [ ] 「要介護状態」には介護保険の要介護認定と、労働基準法上の介護休業・介護休暇の対象となる状態（2週間以上常時介護を要する状態）の2つの意味があることを確認しておく

### ステップ2: 申請する
- [ ] 市区町村の窓口または地域包括支援センターで要介護認定の申請書を提出する
- [ ] 家族が代理で申請する場合の委任状の要否を窓口で確認する
- [ ] 主治医の氏名・医療機関名を申請書に記入する（窓口から主治医へ意見書作成の依頼が行われる）
- [ ] 申請の代行に付き添う（あるいは自分が代行する）場合、勤務先の介護休暇（年5日、対象家族が2人以上の場合は年10日）が使えないか確認する（介護休暇は要介護認定の申請手続きの代行にも使える制度です）

### ステップ3: 訪問調査に備える
- [ ] 訪問調査（認定調査員による聞き取り）の日程調整の連絡を待ち、本人・家族の同席可能な日を伝える
- [ ] 普段の生活状況（できること・困っていること）をメモにまとめておく
- [ ] できれば家族が同席し、普段の様子を調査員に補足できるようにする
- [ ] 訪問調査への同席にも、上記の介護休暇（年5日／2人以上なら年10日）や時間単位の取得が使えないか勤務先に確認する

### ステップ4: 結果が出るまでにできること
- [ ] 利用を検討しているサービス（デイサービス・訪問介護など）の情報を集めておく
- [ ] ケアマネジャー（居宅介護支援事業所）の候補を調べておく
- [ ] 結果通知が届くまでの間の対応（急ぎでサービス利用を始めたい場合の相談先など）について、地域包括支援センターに相談しておく
- [ ] 要介護度は「要介護認定等基準時間」という介護の手間の量で決まり、症状の重さだけで自動的に決まるものではないことを理解しておく（想定より低い区分が出ても、それが症状の軽さの証明にはならない）

### ステップ5: 認定結果を受け取ったら
- [ ] 通知された要支援1・2〜要介護1〜5（または非該当）の区分を確認する
- [ ] 区分に納得できない場合の不服申立て（審査請求）の窓口を、通知書または市町村窓口で確認する
- [ ] ケアマネジャーと相談してケアプランを作成する
- [ ] サービス利用の開始時期・内容を決める
