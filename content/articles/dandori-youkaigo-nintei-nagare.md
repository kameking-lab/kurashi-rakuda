# 要介護認定の申請から結果が出るまでの流れ

## 全体の流れ

要介護認定は、介護保険サービスを利用するために必要な手続きです。申請してから結果が届くまでには一定の期間がかかるため、全体の流れを先に把握しておくと、その間にできる準備を進めやすくなります。

1. 申請（お住まいの市区町村の窓口へ）
2. 訪問調査（調査員が本人・家族から聞き取りを行う）
3. 主治医意見書の作成（申請時に伝えたかかりつけ医が作成）
4. 一次判定・二次判定（コンピューター判定と介護認定審査会による審査）
5. 認定結果の通知（要支援・要介護の区分が通知される）
6. 結果に応じたケアプランの作成・サービス利用開始

申請から結果通知までは一般的に一定の日数がかかるとされており、申請日にさかのぼってサービスを利用できる場合もあるため、急ぎの場合は窓口で相談するとよいでしょう。訪問調査の日程調整や主治医意見書の依頼には家族の協力が必要になる場面が多く、早めに全体の段取りを共有しておくと当日の対応がスムーズになります。

## ステップ別チェックリスト

### ステップ1: 申請前の準備
- [ ] お住まいの市区町村の介護保険担当窓口・地域包括支援センターの連絡先を確認する
- [ ] 本人のかかりつけ医（主治医意見書を依頼する医師）を決めておく
- [ ] 介護保険被保険者証・本人確認書類・マイナンバーが分かるものを準備する

### ステップ2: 申請する
- [ ] 市区町村の窓口または地域包括支援センターで要介護認定の申請書を提出する
- [ ] 家族が代理で申請する場合の委任状の要否を窓口で確認する
- [ ] 主治医の氏名・医療機関名を申請書に記入する（窓口から主治医へ意見書作成の依頼が行われる）

### ステップ3: 訪問調査に備える
- [ ] 訪問調査の日程調整の連絡を待ち、本人・家族の同席可能な日を伝える
- [ ] 普段の生活状況（できること・困っていること）をメモにまとめておく
- [ ] できれば家族が同席し、普段の様子を調査員に補足できるようにする

### ステップ4: 結果が出るまでにできること
- [ ] 利用を検討しているサービス（デイサービス・訪問介護など)の情報を集めておく
- [ ] ケアマネジャー（居宅介護支援事業所）の候補を調べておく
- [ ] 結果通知が届くまでの間の対応について、地域包括支援センターに相談しておく

### ステップ5: 認定結果を受け取ったら
- [ ] 通知された要支援・要介護の区分を確認する
- [ ] ケアマネジャーと相談してケアプランを作成する
- [ ] サービス利用の開始時期・内容を決める
