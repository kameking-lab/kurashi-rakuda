# 産後手続きの期限はどう決まる？ — 産後手続きリスト生成ツールの起算日ルールを解説

## この記事はどのツールと対応している？

本記事は「産後手続きリスト生成（期限つき）」ツール（sango-tetsuzuki-checklist）に対応しています。出生日（出産日）と国内出生か国外出生かを入力すると、出生届の提出・児童手当の認定請求（15日特例）・出産育児一時金の請求について、それぞれの法定期限日を計算し、期限が早い順に並べて表示します。持ち物や窓口など手続きそのものの詳しい段取りは別記事「産後の手続きは何をいつまでに？出生届から児童手当まで」で解説していますので、本記事ではそちらと重複させず、ツールが期限日を**どう計算しているか（起算日のルール）**に絞って説明します。

## 計算の考え方

### 出生届だけが「出生日を含めて」数える

期間の数え方には、法律上の原則と、個別の法令による例外があります。原則は、期間の初日を数に入れず翌日から数える「初日不算入」という考え方です。ところが戸籍法第43条第1項は、届出期間について「届出事件発生の日からこれを起算する」と定めており、出生届に限っては出生した日そのものを数に含めて数えるという、初日不算入の原則の例外になっています。

このため、出生届（国内で出生した場合）の期限は次のように計算します。

```
期限日 = 出生日 + (14 − 1)日
```

出生日が2026-04-01の場合、期限は2026-04-14になります。出生日そのものを数に含めるため、14日目にあたる日がそのまま期限日です。

国外で出生した場合の期限（3ヶ月以内）も、起算日は同じく出生日そのものです。ただし月単位の期間なので、日数の足し算ではなく、民法第143条が定める「暦に従って計算し、起算日に応当する日の前日に満了する」という考え方で計算します。

```
期限日 = 起算日（出生日）に months か月を加えた年月の「応当する日」の前日
```

出生日が2026-04-01で国外出生の場合、3か月後の応当日は2026-07-01なので、その前日の2026-06-30が期限になります。

起算日の日にちが、加算後の月に存在しないケース（月末近くに生まれた場合など）もあります。その場合は民法第143条のただし書のとおり、応当日を探すのではなく、その月の末日そのものを期限とします。前日への繰り下げをさらに重ねることはしません。

### 児童手当・出産育児一時金は「翌日」から数える

児童手当の15日特例、出産育児一時金の請求期限（2年）には、出生届のような初日算入の特則がないため、民法の原則どおり出生日（出産日）の翌日を起算日として数えます。

```
児童手当の期限日 = (出生日の翌日) + (15 − 1)日
```

出生日が2026-04-01の場合、翌日の2026-04-02が起算日になり、期限は2026-04-16です。この期限内に認定請求をすれば、出生月の翌月分から支給されます。

出産育児一時金の請求期限（2年）も同じく翌日起算で、月・年単位の期間なので出生届の国外出生と同じ「応当日の前日」方式で計算します。出生日が2026-04-01の場合、翌日の2026-04-02を起算日として2年後の応当日は2028-04-02、その前日の2028-04-01が期限です。

### 国内・国外の切り替えでチェックリストの並び順が変わる

3つの手続きは期限が早い順に並べ替えて表示します。国内出生では出生届（14日以内）が児童手当（15日以内）より先に来ますが、国外出生では出生届の期限が3ヶ月に伸びるため、児童手当のほうが先に来る順番に入れ替わります。入力を切り替えると並び順そのものが変わることに注意してください。

## 結果の読み方

結果画面には、まず「次の期限」としてもっとも早い期限のカードが表示され、続けて3つの手続きが期限順にリスト表示されます。それぞれのカードには、期限日・起算日のルール・残り日数（または超過日数）・簡単な手続きの説明・根拠となる法令名が並びます。

期限を過ぎた手続きがあっても、出生届は市区町村長が必ず受理しなければならないと戸籍法第46条で定められているため、届出自体をあきらめる必要はありません。正当な理由なく期限内に届け出なかった場合は5万円以下の過料の対象になることがありますが、これは前科がつく刑事罰ではなく、行政上の秩序罰です。

国外出生を選ぶと、出生届とともに国籍留保届を期限内に済ませないと日本国籍を失う場合があるという法務省の注意も表示されます。出生届自体は遅れても受理されますが、国籍留保届は特に期限厳守が重要とされているため、この警告は必ず確認してください。

なお本ツールは、健康保険の加入手続きのように、依存する制度データから期限日数を機械的に導き出せない手続きは扱っていません。期限を推測して表示することはしないという方針によるものです。

## よくある疑問

**「出生日を含めて数える」と「翌日から数える」でどれくらい結果が変わりますか？** 出生届（国内14日）は出生日そのものを含めて数え、児童手当（15日特例）は出生日の翌日を起算日として数えます。同じ「◯日以内」という期間でも起算日が違うため、期限日は単純に暦の日数だけを比べても分かりません。本ツールはこの2つの起算日ルールを取り違えないよう、それぞれ別の計算式で処理しています。

**国外出生の3ヶ月の期限で、応当日が存在しない場合はどうなりますか？** その月の末日が期限になります。前日への繰り下げをさらに重ねることはしません（民法第143条第2項ただし書）。

**出生届の期限を過ぎてしまったら、もう提出できませんか？** できます。戸籍法第46条により、期限を過ぎた届出であっても市区町村長は必ず受理しなければならないと定められています。ただし正当な理由なく遅れると過料の対象になることがあるため、気づいた時点でできるだけ早く届け出てください。

**健康保険の加入手続きの期限も教えてくれますか？** 教えていません。本ツールが依存する3つの制度データ（出生届・児童手当・出産育児一時金）には、健康保険の加入手続きそのものの期限日数を定めた一次情報が含まれていないため、期限を推測せずに扱いを見送っています。
