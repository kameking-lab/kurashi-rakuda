# 虫の発生源別・安全な対策の段取り

## 全体の流れ

家の中の虫対策は「発生源を知る→環境的対策（整理整頓・清掃）→薬剤を安全に使う→保管・換気に注意する」の4段階で進めると効果的です。

1. 発生源を知る — 虫の種類ごとに好む場所が異なる
2. 環境的対策 — 整理整頓・清掃で餌や隠れ場所をなくす
3. 薬剤を安全に使う — パッケージの適用害虫・用法を確認する
4. 保管・換気に注意する — 防虫剤・殺虫剤による体調不良を防ぐ

厚生労働省の「建築物における維持管理マニュアル」（IPM＝総合的有害生物管理）では、薬剤に頼る前にまず環境的対策（整理整頓・清掃・すき間対策）を行うことが基本とされています。

## ステップ別チェックリスト

### 準備段階

- [ ] 家の中に出ている虫の種類を特定する
- [ ] 発生しやすい場所（台所・水回り・収納など）を確認する
- [ ] 使う薬剤がある場合はパッケージの「適用害虫」表示を確認する

### ゴキブリ対策（台所・水回り）

- [ ] ガスレンジ・調理台・流し台・カウンター裏・冷蔵庫やその周りなど、ゴキブリが潜みやすい場所を確認する（厚生労働省のIPMマニュアルが挙げる主な生息場所）
- [ ] 大阪市西淀川区の案内にある通り「台所・食堂・調理場などの整理整頓、清掃により、ゴキブリの餌になるものがない環境をつくる」ことを基本にする
- [ ] 食品は密閉容器に入れ、出しっぱなしにしない
- [ ] 生ごみは早めに処理する
- [ ] 薬剤を使う場合は毒餌（ベイト剤）や粘着トラップなど、環境的対策と組み合わせる

### 防虫剤・忌避剤を使うときの注意

- [ ] 衣類用防虫剤（パラジクロロベンゼン・樟脳・ナフタリンなど）は成分の異なる種類を同時に使わない（しみ・変色の原因になることがある）
- [ ] 国民生活センターの事例では、室内の防虫剤濃度が高くなると目・鼻・のどの粘膜を刺激することがあると報告されている
- [ ] 防虫剤を使う収納は使用時・衣替え時にしっかり換気する
- [ ] つり下げタイプの虫よけ製品を使うときは、パッケージの説明書で適用害虫（例: ユスリカ・チョウバエ限定など）を確認し、蚊やハエ全般に効くと思い込まない

### 殺虫剤の保管・使用上の注意

- [ ] 殺虫剤・防虫剤は食品・食器・飼料と区別し、子どもの手の届かない直射日光の当たらない涼しい場所で保管する
- [ ] エアゾール剤は人体に直接使用しない
- [ ] 車の中や暖房器具の周囲などの高温になる場所に置かない（破裂の危険があるため）
- [ ] 厚生労働省の集計では、家庭用殺虫剤による中毒事故の相談は5歳以下の子どもに多いとされているため、置き場所には特に注意する
