# 産休・育休の復帰日はどう決める？期限の逆算ルール

## 全体の流れ

育休からの復帰日は、自分の希望日を先に決めてから調整するものではありません。すべての期限は「子の1歳到達日（1歳の誕生日の前日）」を起点に決まっており、そこから逆算して動く必要があります。

まず押さえておきたいのは、育休の延長には次の2種類があり、それぞれ別の制度だという点です。

1. **育休そのものの延長**（育児・介護休業法にもとづき、勤務先に申し出るもの）
2. **育児休業給付金の支給対象期間の延長**（雇用保険法にもとづき、ハローワークが審査するもの）

この2つは要件も審査主体も異なり、片方が認められてももう片方が自動的に認められるわけではありません。育休そのものの延長には「子の1歳到達日の時点で、本人または配偶者がまだ育休を取得していること」が絶対条件です。一度職場復帰してしまうと、原則としてあとから延長を申し出ることはできません。延長できる期間にも上限があり、1歳6か月までの延長、1歳6か月から2歳までの延長を経ても、育休の上限は2歳です。それ以上の延長制度は存在しません。

保育園に入れなかったことを理由に育休を延長する場合、2025年4月から育児休業給付金側の審査が厳格化されました。これは「入所する気がないのに、給付を延ばすためだけに申し込む」といういわゆる落選狙いへの対策です。この改正により、保育園の入所申込みは子が1歳に達する日の翌日以前を入所希望日として終わらせておく必要があり、申し込む園には自宅から通所に片道30分未満で通える園を1園以上含めることが求められます（通勤経路上にある場合や、片道30分未満の園が周辺にない場合などの例外はあります）。自治体窓口で「今の状況では入れない」と言われても、申込み自体を行わなかった場合は延長が認められません。

さらに、入所保留通知書（自治体により入所不承諾通知書などと呼ばれます）は、1歳到達日翌日の2か月前以降の発行日でなければならず、4月入所を希望していた場合に限り3か月前まで遡った発行日でも有効です。古い通知書は使えないため、発行時期にも注意が必要です。

もうひとつ見落としやすいのが、育休関連の給付・免除には「1歳（最長2歳）」と「3歳未満」という2つの異なる期間の区切りがあることです。育児休業給付金は原則1歳まで、延長しても最長2歳までしか支給されません。これに対して、健康保険・厚生年金保険料の免除は3歳未満の子を養育する育休等であれば対象になります。「給付が終わる年齢」と「保険料免除が終わる年齢」がずれているため、2歳から3歳になるまでの間は「給付はないが保険料は免除される」期間が生じます。この期間ギャップは復帰・退職・時短勤務のいずれを選ぶ場合にも影響するため、最初に理解しておく価値があります。

以上を踏まえると、復帰日を決める際の逆算の順番は次のようになります。

1. 子の1歳到達日を確定する
2. 保育園の入所申込みを1歳到達日以前に終わらせる（通勤30分未満の園を含める）
3. 入所保留通知書の発行時期を確認する
4. 育休そのものの延長と、給付金の延長を別々に手続きする
5. 社会保険料免除の期間（3歳未満まで）と給付の期間（最長2歳まで）のギャップを確認する
6. 復帰候補日を固め、勤務先の申請期限までに正式に伝える

## ステップ別チェックリスト

### ステップ1: 子の1歳到達日を確定し、そこから逆算する
- [ ] 子の1歳到達日（1歳の誕生日の前日）をカレンダーに記入し、そこを起点にすべての期限を逆算する
- [ ] 育休をそのまま延長したい場合、1歳到達日の時点で本人または配偶者が育休を取得している必要があることを確認する（一度職場復帰すると原則延長できない）
- [ ] 育休の延長は1歳6か月まで、さらに2歳までの2段階があり、2歳が絶対的な上限であることを押さえる

### ステップ2: 保育園の入所申込みを1歳到達日以前に終わらせる
- [ ] 保育園の入所申込みは、原則として子が1歳に達する日の翌日以前を入所希望日として完了させる
- [ ] 申込む保育園に、自宅から通所に片道30分未満で通える園を1園以上含める
- [ ] 自治体窓口で「入れない」と言われても申込み自体は必ず行う（申込みをしなかった場合は延長が認められない）
- [ ] 入所申込書に「入所保留を希望する」といった意思表示（自治体独自の「育休延長希望」欄へのチェックなど）を絶対にしない
- [ ] 申込書の控え（電子申請の場合は申込内容の印刷またはスクリーンショット）を、申込んだその日のうちに保管しておく

### ステップ3: 入所保留通知書の発行時期を確認する
- [ ] 入所保留通知書（自治体により入所不承諾通知書などと呼ばれる）が、1歳到達日翌日の2か月前以降の日付で発行されることを確認する
- [ ] 4月入所を希望していた場合は、3か月前まで遡った発行日でも有効であることを確認する
- [ ] 通知書の発行が遅れそうな場合は、早めに自治体窓口へ相談する

### ステップ4: 育休の延長と給付金の延長を別々に手続きする
- [ ] 勤務先に、育休そのものの延長（1歳6か月まで、または2歳まで）を書面等で申し出る
- [ ] ハローワークまたは勤務先経由で、育児休業給付金の支給対象期間延長に必要な書類（延長事由認定申告書・保育所等利用申込書の写し・入所保留通知書）がそろっているか確認する
- [ ] 育休の延長と給付金の延長は別制度であり、片方が認められてももう片方が自動的に認められるわけではないことを理解しておく
- [ ] 申込書の写しの内容と実際の申込み内容が異なると不正受給として返還を求められる場合があることを理解しておく

### ステップ5: 社会保険料免除と給付金の期間ギャップを確認する
- [ ] 育児休業給付金は最長2歳までだが、社会保険料免除は3歳未満まで対象になるという期間の差を確認する
- [ ] 2歳到達後も育休や時短勤務を続ける場合、その間の社会保険料の扱いを勤務先の担当者に確認する
- [ ] 育休給付金は休業日数が通算180日に達するまでは67%、それ以降は50%に給付率が下がるため、復帰時期によって受け取れる給付額が変わることも踏まえておく
- [ ] 同じ月の中で14日以上の育休を取得すれば、その月の社会保険料が免除される場合があることも覚えておく

### ステップ6: 復帰日を確定し、勤務先に伝える
- [ ] 保育園の入園時期が固まったら、勤務先の復帰申請期限までに正式な復帰日を伝える
- [ ] 育休の開始（延長ではなく最初の育休）を申し出る場合は、原則として休業開始予定日の1か月前までに勤務先へ申請するルールがあるため、最初の育休取得時にもあわせて期限を確認しておく
- [ ] 慣らし保育の期間が必要な園の場合、その期間も見込んで復帰日に余裕を持たせる
