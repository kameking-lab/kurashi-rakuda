# 産休・育休の復帰日はどう決める？期限の逆算ルール

## 全体の流れ

育休からの復帰日は、自分の希望だけで決められるものではなく、「保育園にいつから入れるか」「勤務先への復帰意向の申請期限」「育児休業給付の取り扱い」など、複数の期限が絡み合います。多くの場合、次の順番で逆算して考えると検討しやすくなります。

1. 保育園の入園可能時期を確認する（自治体の申込みスケジュール・年度途中入園の可否）
2. 勤務先の育休制度・復帰時期の申請ルールを確認する
3. 上記2つを踏まえて、現実的な復帰候補日をいくつか仮決めする
4. 勤務先に復帰意向を伝え、正式な復帰日を確定させる
5. 育児休業給付など、復帰にともなう手続きを進める

保育園は自治体ごとに入園申込みの受付時期や選考基準が異なり、年度途中の入園は枠が限られる場合があるとされています。そのため、復帰日を先に決めてから保活を始めるより、保活の見通しを立てながら復帰日を調整する方が現実的な進め方になりやすいでしょう。

## ステップ別チェックリスト

### ステップ1: 保育園の入園時期を確認する
- [ ] お住まいの自治体の保育園申込みスケジュール（申込み受付時期・結果通知時期）を確認する
- [ ] 年度途中入園の可否・倍率の傾向を自治体窓口や園に確認する
- [ ] 認可・認可外を含めた候補園をリストアップし、見学の予定を立てる

### ステップ2: 勤務先の制度・ルールを確認する
- [ ] 勤務先の育児休業に関する規程（復帰時期の申請期限、時短勤務制度の有無）を確認する
- [ ] 復帰の意向をいつまでに伝える必要があるか、人事・上長に確認する
- [ ] 復帰後の働き方（時短勤務・在宅勤務の可否など）についても合わせて相談しておく

### ステップ3: 復帰候補日を仮決めする
- [ ] 保育園の入園時期と勤務先の申請期限を照らし合わせ、無理のない復帰候補日を複数挙げる
- [ ] 慣らし保育の期間が必要な園の場合、その期間も見込んで候補日を調整する
- [ ] パートナーや家族と、復帰後の送迎・家事分担についても話し合っておく

### ステップ4: 復帰日を確定し、勤務先に伝える
- [ ] 決めた復帰日を勤務先の期限までに正式に伝える
- [ ] 復帰後の勤務条件（勤務時間・部署など）を書面等で確認する

### ステップ5: 給付・保険まわりの手続きを進める
- [ ] 育児休業給付の受給期間や手続きの流れを勤務先またはハローワークに確認する
- [ ] 復帰にともなう社会保険料の手続き（養育期間特例措置など）について勤務先に確認する
