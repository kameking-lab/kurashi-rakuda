# 吹き出物（にきび）と生活習慣の関係・日常でできること

## 全体の流れ

にきびとの付き合い方は「できる仕組みを理解する→洗顔を見直す→生活習慣を整える→悪化させない工夫をする→必要なら皮膚科に相談する」の5段階で進めると効果的です。

1. できる仕組みを理解する — 皮脂の分泌と毛穴の詰まりが重なって始まる
2. 洗顔を見直す — 洗いすぎ・触りすぎに注意する
3. 生活習慣を整える — 食事を極端に制限せず、睡眠を整える
4. 悪化させない工夫をする — 髪や衣類が触れないようにする
5. 必要なら皮膚科に相談する — 自己判断で悪化させないために

日本皮膚科学会のQ&Aでは、にきびは「皮脂（皮膚のあぶら）の分泌が多いことと毛穴の先が詰まることで、毛穴の中に皮脂がたまることで始まります」と説明されています。皮脂と酸素の少ない毛穴の中でアクネ菌が増えると炎症が起き、赤いぶつぶつや膿をともなうにきびになるとされています。

## ステップ別チェックリスト

### 洗顔を見直す

- [ ] 「にきびは不潔にしているからできるわけではありません」という点を踏まえ、洗いすぎない
- [ ] 洗顔は1日2回、洗顔料をよく泡立てて手でやさしく洗い、十分な水で洗い流す
- [ ] 強くこすったり、1日に何度も洗顔したりしない（炎症が起きて悪化することがある）
- [ ] 乾燥が気になる場合は保湿剤を併用する

### 生活習慣を整える

- [ ] 食事はチョコレート・ケーキ・ピーナッツなど特定の食品を一律に禁止する必要はなく、バランスの良い食事を心がける
- [ ] 特定の食品を食べると悪化すると感じる場合は、その食品を控える
- [ ] 睡眠不足や昼夜逆転の生活はホルモンバランスを崩し、にきびの悪化因子になりうるため、規則正しい生活を心がける
- [ ] ストレスの少ない規則正しい生活を意識する

### 悪化させない工夫をする

- [ ] にきびを潰したり、触ったりしない
- [ ] 前髪・マフラー・マスクなど、肌に常に触れるものを避ける（毛先やマフラーなどが常にあたっているとにきびができやすくなるとされる）
- [ ] にきびを隠すためのコンシーラーの塗り重ねは避ける
- [ ] 自宅では髪を束ねる、ヘアバンドを使うなどして前髪が触れないようにする

## 皮膚科への相談が必要な場合

にきびは慢性炎症性疾患に位置づけられており、悪化すると瘢痕（あと）が残ったり、心理面への影響が大きくなったりすることがあります。上記の生活習慣を見直しても改善しない場合や、悪化している場合、痛みを伴う場合は、自己判断で市販薬や民間療法を続けず、早めに皮膚科を受診してご相談ください。
