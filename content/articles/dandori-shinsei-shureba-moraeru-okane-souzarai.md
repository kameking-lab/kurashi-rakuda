# 申請しないと受け取れないお金・行政サービスの総ざらい段取り

## 全体の流れ

見落としやすい公的給付・助成をチェックする段取りは「申請主義であることを理解する→ライフイベント別に洗い出す→制度ごとの窓口・期限を確認する→申請する」の4段階で進めると効果的です。

1. 申請主義であることを理解する — 対象でも自分から申請しないと受け取れない制度が多い
2. ライフイベント別に洗い出す — 子育て・医療費・介護・仕事の変化のタイミングで確認する
3. 制度ごとの窓口・期限を確認する — 所管が制度ごとに異なる
4. 申請する — 必要書類をそろえて窓口・オンラインで手続きする

厚生労働省の「年金生活者支援給付金制度 特設サイト」では、「年金生活者支援給付金を受け取るには、年金生活者支援給付金請求書の提出が必要です」と明記されています。これは一例ですが、多くの公的給付・助成は対象要件を満たしていても、自分から請求・申請しなければ受け取れない「申請主義」で運用されています。制度を知らなかった、申請を忘れていた、という理由で受け取れないことがないよう、定期的に見直す段取りが大切です。

## ステップ別チェックリスト

### 申請主義であることを理解する

- [ ] 「対象なら自動的に振り込まれる」制度と「自分で申請しないと受け取れない」制度があることを理解する
- [ ] 引っ越し・転職・出産・介護の開始など、ライフイベントのタイミングで制度が変わることが多いと意識する
- [ ] 申請には期限があるものが多いため、対象になったら早めに確認する

### 子育てに関する制度を洗い出す

- [ ] お子さんの年齢・ひとり親かどうかなどの状況から、関係しうる子育て支援制度を横断的に確認する
- [ ] 児童手当・児童扶養手当・こども医療費助成・幼児教育無償化・就学援助・高校就学支援金など、制度ごとに所管・申請先が異なる点に注意する

### 医療費・介護に関する制度を洗い出す

- [ ] 1年間の医療費が一定額を超えた場合の医療費控除（確定申告が必要）の対象になっていないか確認する
- [ ] 医療費・介護費の自己負担が高額になった場合の高額療養費・高額介護サービス費、両方が高額になった場合の高額医療・高額介護合算療養費の制度を確認する
- [ ] これらは基本的に本人からの申請が必要な制度である点に注意する

### 仕事・キャリアに関する制度を洗い出す

- [ ] 資格取得やスキルアップの費用を支援する教育訓練給付など、雇用保険に関する給付を確認する
- [ ] 介護のために休業する場合の介護休業給付など、状況の変化に応じた給付を確認する
- [ ] いずれもハローワーク等での手続きが必要な制度である点に注意する

### 申請する

- [ ] 各制度の所管窓口（自治体・年金事務所・ハローワーク・税務署など）と必要書類を確認する
- [ ] 申請期限がある制度は、対象になった時点でスケジュールに入れる
- [ ] 制度の詳細・最新の要件は必ず所管窓口の公式情報で確認する
