# 保活はいつから始める？申込スケジュールの立て方

## 全体の流れ

保育園（保育所等）の入園申込みの締切日は、法律で全国一律に定められているわけではありません。受付期間・締切日・必要書類は市区町村ごとに決められており、同じ「4月入園」でも自治体によって数か月単位で違うことがあります。そのため「保活は◯月◯日までに始めれば大丈夫」という全国共通の答えは、そもそも存在しません。

ただし、手がかりがまったくないわけではありません。次の2点を押さえておくと、動き出す時期を具体的に組み立てられます。

1つめは、多くの自治体が4月入園の一斉申込みを、予算編成や職員配置の年間スケジュールに合わせて前年の秋から冬にかけて締め切る運用をとっていることです。これは法律で決まった期限ではなく自治体ごとの事務スケジュールによるものですが、結果として多くの家庭の保活が前年の初夏から秋ごろにかけて本格化する傾向があります。

2つめは、保活がうまくいかず入園できなかった場合に備えて育児休業を延長する道を残しておきたいときに関わる、実在する国の要件です。育児休業給付金の支給対象期間を延長するには、子が1歳に達する日（1歳の誕生日の前日）以前の日を入所希望日として、市区町村へ保育所等の利用申込みを済ませていることが要件の一つになっています。この申込日そのものが、育休延長というセーフティネットを使えるかどうかの分かれ目になります。単に申込みを忘れていた場合や、窓口で「どうせ入れない」と言われて申込みを見送った場合は、延長が認められません。

つまり、全国共通の確定した締切日は存在しなくても、「1歳に達する日までに申込みを終えている」という一点は、育休延長を選択肢として残すための実質的な期限として機能します。まずはお住まいの自治体が発行する「保育施設利用申込み案内」（名称は自治体により異なります）を早めに確認し、そこに書かれている締切日を基準に自分のスケジュールを逆算するのが、現実的な進め方です。

## ステップ別チェックリスト

### ステップ1: 入園希望時期と申込み区分を決める
- [ ] 4月の一斉入園を目指すか、年度途中の入園も視野に入れるかを決める
- [ ] 育休からの復職時期など、家庭の希望条件を整理する
- [ ] 落選した場合に育休を延長する可能性があるかどうかも、あらかじめ選択肢に入れておく

### ステップ2: 自治体の「保育施設利用申込み案内」を確認する
- [ ] 4月入園を希望する場合は、前年の初夏〜秋ごろに一度、お住まいの自治体の「保育施設利用申込み案内」（名称は自治体により異なる）を確認する
- [ ] 一斉申込みの受付期間・締切日を確認する
- [ ] 年度途中入園の申込み方法（随時受付か、別途締め切りがあるか）を確認する
- [ ] 見学が必須の園があるか、見学の予約方法もあわせて確認する

### ステップ3: 締切日から逆算して動き出す
- [ ] 確認した締切日から逆算し、見学・情報収集にあてる期間を確保する
- [ ] 就労証明書など、準備に時間がかかる書類を早めに用意し始める
- [ ] 候補の園を複数リストアップし、優先順位をつけておく
- [ ] 自宅から片道30分未満で通える園を、候補の中に少なくとも1つ含めておく（育休延長を申請する場合の要件の一つになるため）

### ステップ4: 落選に備え、育休延長の要件と当面の費用も確認しておく
- [ ] 育児休業給付金の延長を申請する可能性がある場合、保育所等の利用申込みを子が1歳に達する日までに終える
- [ ] 申込書の写し（電子申請の場合は申込内容を印刷したもの、または申込み画面を印刷したもの）を、提出した時点で必ず保管しておく
- [ ] 自治体が発行する入所保留通知書は、発行年月日が1歳に達する日の翌日の2ヶ月前（4月入所申込みの場合は3ヶ月前）以降の日付である必要があることを確認しておく
- [ ] 育児休業そのものは2歳になるまで延長できるが、雇用保険の育児休業給付金の延長は別制度であり、審査基準も別であることを理解しておく
- [ ] 入園が決まるまでの間に認可外保育施設等を一時的に利用する場合、無償化の対象になる世帯であれば3〜5歳児クラスは月37,000円、住民税非課税世帯の0〜2歳児クラスは月42,000円までの利用料が無償化の対象になることを確認しておく
