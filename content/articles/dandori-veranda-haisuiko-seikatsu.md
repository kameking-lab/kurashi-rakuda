# ベランダ排水溝の詰まりを防ぐ清掃の段取り

## 全体の流れ

ベランダ排水溝の対策は「詰まりの原因を知る→日常の清掃・ゴミ除去→大雨・台風前後の重点清掃→業者トラブルに注意する」の4段階で進めると効果的です。

1. 詰まりの原因を知る — ゴミ・落ち葉の堆積、取付管への異物混入
2. 日常の清掃・ゴミ除去 — 排水溝にゴミや物を置かない
3. 大雨・台風前後の重点清掃 — 溝・排水溝の掃除を欠かさない
4. 業者トラブルに注意する — 無料点検を装った高額請求に警戒する

世田谷区の案内では、雨水ますの詰まりの原因として「雨水ますにごみなどが溜まっていたり、下水道管とますを連結している取付管に物が詰まっている」ことが挙げられています。ベランダの排水溝も同様に、ゴミや落ち葉の堆積が詰まりの主な原因です。

## ステップ別チェックリスト

### 準備段階

- [ ] ベランダの排水溝の位置を確認する
- [ ] ほうき・ちりとり、ゴム手袋など掃除用具を準備する
- [ ] 排水溝の上や周辺に物を置いていないか確認する

### 日常の清掃

- [ ] 排水溝にたまった落ち葉・砂・ゴミを取り除く
- [ ] 排水溝の上に鉢植えや物を置かない（世田谷区の案内では「雨水ますにごみなどを入れたり、物を置かないでください」とされている）
- [ ] 洗濯機・エアコン室外機の水はけを妨げていないか確認する

### 台風・大雨前後の重点清掃

- [ ] 中央区の案内にある通り、台風が来る前に「バルコニーの溝・排水溝を掃除する」
- [ ] 台風通過後は「バルコニーの排水溝にあるゴミを取り除く」
- [ ] 大雨予報が出たときは事前に排水溝の詰まりがないか確認する

### 業者に依頼する場合の注意

- [ ] 「無料点検」をきっかけに高額な洗浄・修理を勧誘されるトラブルが国民生活センターに報告されている
- [ ] 点検・清掃を依頼する前に料金体系を明確に確認する
- [ ] その場で契約を急かされたら一度持ち帰って検討する
- [ ] 不安がある場合は消費生活センター等の窓口に相談する
