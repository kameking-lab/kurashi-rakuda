# 車内清掃を場所別に進める段取りと洗剤使用時の注意

## 全体の流れ

車内清掃は「大きなゴミを取り除く→場所別に掃除機・拭き取りを行う→仕上げに換気・乾燥させる」の3段階で進めると効率的です。

1. 大きなゴミを取り除く — ゴミ・落ち葉・砂などを先に処分する
2. 場所別に掃除機・拭き取りを行う — 座席・フロアマット・ダッシュボードなど部位ごとに対応する
3. 仕上げに換気・乾燥させる — 湿った状態のまま締め切らない

## ステップ別チェックリスト

### 準備段階

- [ ] 車内のゴミ・落ち葉・砂などの大きなゴミを取り除く
- [ ] 掃除機（車載用またはコイン洗車場の掃除機）、拭き取り用の布、必要であれば内装用洗浄剤を準備する
- [ ] フロアマットは取り外せるものは車外に出しておく

### フロアマット・座席の掃除

- [ ] フロアマットを取り外し、掃除機でホコリ・砂を吸い取る
- [ ] 布製シートは掃除機がけの後、必要に応じて内装用洗浄剤で拭き取る
- [ ] 洗浄剤を使う場合は目立たない場所で変色しないか試してから全体に使う
- [ ] 洗浄後はドアを開けて換気し、湿った状態のまま締め切らない

### ダッシュボード・内装パネルの拭き取り

- [ ] 乾いた布でホコリを払ってから、必要に応じて内装用クリーナーで拭き取る
- [ ] エアコンの吹き出し口などすき間はすき間用ノズルやブラシでホコリを取る

### スプレー缶製品（洗浄剤・消臭剤等）を使うときの注意

- [ ] スプレー缶（エアゾール缶）製品は、気体状態のガスが高温になると圧力が異常に上昇し、缶の破裂の原因になることが国民生活センターの資料で説明されている
- [ ] 使用後のスプレー缶や予備の缶を炎天下の車内に放置しない
- [ ] 換気しながら使用し、火気の近くでは使用しない
- [ ] 使い切ったと思っても中にガスが残っていることがあるため、廃棄時は自治体のルールに従う
