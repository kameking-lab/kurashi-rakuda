# 窓サッシの結露・カビを防ぐ清掃の段取り

## 全体の流れ

窓サッシの結露・カビ対策は「結露の仕組みを理解する→こまめな拭き取り→サッシ溝の清掃・排水→換気で湿度を下げる」の4段階で進めると効果的です。

1. 結露の仕組みを理解する — 空気が冷たい面に触れて水滴になる現象
2. こまめな拭き取り — 結露した水を放置しない
3. サッシ溝の清掃・排水 — 水はけをよくして水を屋外に出す
4. 換気で湿度を下げる — 結露が起きにくい室内環境をつくる

台東区「快適で健康的な住まいを目指して」では、結露を「空気が冷たい面（壁や窓ガラスなど）に触れて温度が下がり、空気中に含まれる水蒸気が水滴となって現れること」と説明しています。壁や窓ガラスに付着した水滴は建材・壁紙を傷めるだけでなく、アレルギー疾患の原因の一つであるカビを発生させやすくするとされています。

## ステップ別チェックリスト

### 準備段階

- [ ] 結露が起きやすい窓（北向き・断熱性の低い窓など）を確認する
- [ ] 雑巾・結露取りワイパーなど拭き取り用具を準備する
- [ ] サッシ下部の溝を掃除するための歯ブラシや綿棒、掃除機のすき間ノズルを用意する

### 結露の拭き取り

- [ ] 朝、窓ガラスやサッシに結露が付いていたらこまめに拭き取る
- [ ] 東京都「室内のカビ対策」指針では、窓に結露した水はこまめに拭き取ることが対策として挙げられている
- [ ] 拭き取った布はよく絞って乾燥させ、湿ったまま放置しない

### サッシ下部の溝を清掃する

- [ ] サッシ下部の排水溝にたまったホコリ・砂を掃除機や歯ブラシで取り除く
- [ ] 東京都「室内のカビ対策」指針・台東区の案内では、サッシ下部の排水溝は流れをよくして結露した水を屋外に出すようにすると説明されている
- [ ] 排水溝の水はけが悪い場合は詰まりの原因物を取り除く

### 換気・湿度管理

- [ ] 1日数回、窓を開けて換気し室内の湿度を下げる
- [ ] 梅雨時期など換気だけで湿度が下がりにくい場合は除湿機の使用も検討する
- [ ] 洗濯物の室内干しが多い部屋は特に結露しやすいため、換気を意識する

## 断熱と結露の関係

消費者庁の資料「『寒い家で我慢』は健康上のリスク！―窓から始める断熱リフォーム」によると、単板ガラス＋アルミサッシのような断熱性能がほとんどない窓は熱ロスが多く室内表面温度も低いため結露しやすい一方、Low-Eガラス＋樹脂サッシなど高断熱な窓は冬の室内表面温度が高く維持されるため結露が生じにくく、カビの予防にも効果的だとされています。日常の清掃・拭き取りに加えて、窓自体の断熱性能も結露の起きやすさに関わることを知っておくと、対策の優先順位をつけやすくなります。
