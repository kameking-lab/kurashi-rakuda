# 掃除・衛生管理の家庭向けガイドラインとは

## 全体の流れ

家庭の掃除・衛生管理は、次の5段階で進めると抜け漏れを防ぎやすくなります。

1. 掃除の優先順位を決める（キッチン・トイレなど水回りを優先）
2. 家族の体調を確認し、消毒が必要な場面かどうかを見極める
3. 場所に合った濃度の消毒液を準備する（次亜塩素酸ナトリウム）
4. 熱湯消毒が向いている道具には熱湯消毒を行う
5. 掃除・消毒を実施し、片付けと振り返りをする

特に嘔吐・下痢などの症状が家族に出たときは、ノロウイルスなどの感染症対策として消毒の濃度を場所ごとに変える必要があります。「なんとなく薄める」のではなく、決められた濃度で作ることが大切です。

## ステップ別チェックリスト

### ステップ1: 掃除計画を立てる
- [ ] 家の中で優先して清潔にしたい場所（キッチン、トイレ、洗面所、床など）をリストアップする
- [ ] 家族に嘔吐・下痢などの症状がないか確認する
- [ ] 症状がある場合は、通常の掃除ではなく感染症対策の消毒が必要と判断する

### ステップ2: 次亜塩素酸ナトリウムの消毒液を準備する
- [ ] 調理器具・まな板・ふきんの消毒には200ppmの次亜塩素酸ナトリウム溶液を使う
- [ ] 床や、嘔吐物・排泄物が付着した場所の消毒にも200ppmの次亜塩素酸ナトリウム溶液を使う
- [ ] 使用済みおむつなど廃棄物を捨てる前の消毒には、より濃い1000ppmの次亜塩素酸ナトリウム溶液を使う
- [ ] 消毒液は使うたびに作り、作り置きせず使い切る（時間が経つと消毒効果が落ちるため）
- [ ] 使い捨て手袋・マスクを着用し、窓を開けて換気しながら作業する

### ステップ3: 熱湯消毒を行う
- [ ] まな板・包丁など煮沸できる道具は、85度以上の熱湯に1分以上つけて消毒する
- [ ] 熱湯消毒のあとはしっかり乾燥させてから片付ける
- [ ] やけどに注意し、作業中は小さな子どもを近づけない

### ステップ4: 場所ごとに掃除・消毒を実施する
- [ ] 汚れがひどい場所や消毒が必要な場所から順に取りかかる
- [ ] ステップ2・3で準備した消毒液・熱湯消毒を、場所に合わせて使い分ける
- [ ] 消毒に使った布・ペーパーは使い捨てるか、他の洗濯物と分けてしっかり洗浄・乾燥させる

### ステップ5: 片付け・振り返り
- [ ] 使用した手袋・マスクなどを処分する
- [ ] 石けんと流水で手をしっかり洗う
- [ ] 消毒液の濃度や場所で迷った点があれば、次回のためにメモしておく
