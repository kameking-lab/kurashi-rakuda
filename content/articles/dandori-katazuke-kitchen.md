# キッチンの片付け・収納の段取りと安全対策

## 全体の流れ

キッチンの片付けは「冷蔵庫・食品ストックの整理→刃物・洗剤の安全な収納→シンク下収納の湿気対策」の3段階で進めると効果的です。

1. 冷蔵庫・食品ストックの整理 — 中身をチェックし、種類ごとに分類する
2. 刃物・洗剤の安全な収納 — 使用後すぐに片付け、指定容器のまま保管する
3. シンク下収納の湿気対策 — 通気性を確保してカビを防ぐ

## ステップ別チェックリスト

### 冷蔵庫・食品ストックの整理

- [ ] 消費者庁の案内にある手順で、まず冷蔵庫や食品棚の中身を全部チェックする
- [ ] 食品を種類（カテゴリ）ごとに分類し、置く場所を決める
- [ ] フリースペースを作る（消費者庁の案内では「冷蔵庫に保存する食材は7割以下に」が目安とされている）
- [ ] ストックのルールを決める（同じ種類の食品を無計画に増やさない）
- [ ] 賞味期限・消費期限を確認し、期限が近いものを手前に置く

### 刃物・洗剤の安全な収納

- [ ] 包丁などの刃物は使用後すぐに収納場所へ片付ける
- [ ] 刃物・洗剤を収納する扉や引き出しにはチャイルドロックを付ける
- [ ] 洗剤・漂白剤は購入時の容器のまま保管し、飲料用ペットボトルなど別の容器に移し替えない
- [ ] 国民生活センターの発表では、洗剤や殺虫剤を飲料用容器に移し替えたことによる誤飲事故が報告されている

### シンク下収納の湿気対策

- [ ] シンク下など空気の流れが悪い収納スペースには、すのこなどを使って通気性を確保する
- [ ] 京都市の案内では、押入れや家具の後ろなど空気の流れの悪い場所にカビが発生しやすいとされ、室内の湿度は60%を超えないようにすることが目安とされている
- [ ] 濡れたスポンジ・布巾を収納内に入れたままにしない
