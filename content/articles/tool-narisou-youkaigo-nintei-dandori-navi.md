# 要介護認定「あと何日で結果が出る？」はこう決まる — 申請段取りナビの目安と根拠条文の読み方

## この記事はどのツールと対応している？

本記事は「要介護認定 申請段取りナビ」ツール（youkaigo-nintei-dandori-navi）に対応しています。要介護認定の申請日（または申請予定日）を入力すると、原則の期限日、認定調査・主治医意見書・審査判定の目安の日程、そして今おおよそどの段階にいそうかをチェックリスト形式で表示します。要介護認定の手続そのものの全体像（申請→訪問調査→一次判定→二次判定→結果通知という流れや、要介護度がどう決まるか）は別記事「要介護認定の申請から結果が出るまでの流れ」で解説していますので、本記事ではそちらと重複させず、ツールが表示する**期間の目安・現在地推定がどういう根拠と計算で出ているか**に絞って説明します。

## 計算の考え方

### 「原則30日以内」は法律上の期限であり、保証ではない

ツールがまず示すのは、申請日から30日後の日付（原則の期限日）です。これは介護保険法第27条第11項本文の「第一項の申請に対する処分は、当該申請のあった日から三十日以内にしなければならない」という規定に基づく計算で、次のように求めています。

```
legalDeadlineDate = 申請日 + 30日
isPastLegalDeadline = 申請日からの経過日数 > 30
```

ただし、同項にはただし書があり、認定調査に時間を要するなど特別な理由があるときは、市区町村が「あとどれくらいかかりそうか（処理見込期間）」と理由を書面で通知したうえで、この期限を延期することが法律上できると定められています。つまり30日という数字は、市区町村が絶対に守らなければならない締切ではなく、「原則としてはこの期間内に」という基準であり、延期の通知が来たとしても、それは制度が想定している正規の手続きです。ツールが「原則の30日を超過」と表示しても、必ずしも自治体の対応が遅れているとは限らない、という点をまず押さえておいてください。

なお同法第27条第12項には、30日を過ぎても処分・延期通知のいずれもない場合等に、被保険者側から「市区町村が申請を却下したものとみなす」ことができるという規定もありますが、本ツールはこの権利を行使するよう助言するものではなく、条文の内容の紹介にとどめています。

### 目安の日程（7日・13日・12日）はどこから来ているか

ツールの「認定調査の実施目安」「主治医意見書の入手目安」「審査会での審査判定の目安」という3つの日数は、厚生労働省が社会保障審議会介護保険部会の資料2で示した「対応（案）」の数字（認定調査は依頼から7日、主治医意見書は依頼から13日、介護認定審査会は調査票・意見書が揃ってから12日）をそのまま使っています。

この3つの数字は、令和5年度の全国実績（全1,559保険者の集計）のうち、認定審査期間の平均が30日以内に収まっているごく一部の保険者に限って集計した、各段階の平均所要日数を踏まえて示された目安です。**2025-02-20の時点ではまだ「対応（案）」の段階であり、全国一律に確定した基準ではありません。** 各保険者が今後、地域の実情に応じて独自の目安を検討・設定していく前提の参考値である、という位置づけをツールも本記事も崩さないようにしています。

### なぜ「7日」と「13日」を単純に足し算しないのか

ツールのタイムラインで少し分かりにくいのが、認定調査の目安（7日）と主治医意見書の目安（13日）を単純に足し算していない点です。これは、認定調査と主治医意見書が申請受付後にそれぞれ別々に依頼され、実務上は並行して進むためです。認定調査は7日ほどで終わる見込みでも、主治医意見書のほうが時間がかかる（13日ほど）ので、「両方がそろうタイミング」はこの2つのうち日数が長いほう、つまり13日を基準にします。そこに、書類がそろってから審査会の事務処理にかかる目安（12日）を足したものが、審査判定までのおおよその目安日数になります。

```
認定調査の目安 = 7日
主治医意見書の目安 = 13日
両方がそろう目安 = 認定調査の目安と主治医意見書の目安のうち大きいほう（＝13日）
審査判定の目安 = 両方がそろう目安 + 12日
```

もし7日と13日を単純に直列で足してしまうと、実際の進み方より遅い見積もりになってしまいます。認定調査と主治医意見書は同時並行で進む、という実務上の前提を反映した計算です。

### 今どの段階かの推定は、あくまで経過日数からの概算

ツールは、申請日から今日までの経過日数と、上で説明した目安の日程を比べることで、「認定調査・主治医意見書の準備中」「主治医意見書の入手待ち」「審査会での審査判定中」「結果通知待ち」といった段階を推定して表示します。これはあくまで目安の日程からの逆算による概算であり、実際の進捗を市区町村のシステムから直接取得しているわけではありません。実際にどこまで進んでいるかは、自治体からの連絡や書類で確認する必要がある、という点を必ず念頭に置いてください。

## 結果の読み方

結果画面では、まず「原則の期限日」（申請日から30日後）が示されます。あわせて、認定調査・主治医意見書・審査判定それぞれの目安の日付と、現在のおおよその段階が表示されます。

ここで見落としてはいけないのが、ツールが必ず一緒に表示する「実態」の数字です。令和5年度の全国実績では、要介護認定の認定審査期間（申請日から二次判定日まで）の全保険者平均は39.8日で、法律上の原則である30日を上回っています。また、申請から30日以内に実際に認定された割合は、全保険者平均でわずか25.1%にとどまります。つまり、「原則30日」どおりに終わるほうが、全国的にはむしろ少数派だという実態です。ツールの目安の日程より結果が遅れても、それ自体が異常事態とは限らないことを、この2つの数字が示しています。

原則の期限を過ぎたと表示された場合は、まず延期の通知が届いていないかを確認し、届いていない・内容がよく分からないという場合は、申請先の市区町村の介護保険担当窓口に直接問い合わせるのが確実です。

## よくある疑問

**「7日」「13日」「12日」という目安を守ってくれる保証はあるの？** ありません。これらは厚生労働省が保険者（市区町村など）に示した「対応（案）」の段階の参考値であり、法律で定められた確定基準ではありません。実際の処理期間は、お住まいの自治体や申請の混雑状況、主治医の対応状況によって大きく変わります。

**申請予定日を先の日付で入力したらどうなる？** 「申請前」として、申請予定日までの残り日数と、その日に申請した場合の原則の期限日を表示します。この場合、現在地の段階推定（準備中・待ち・審査中など）は行いません。

**認定調査と主治医意見書の依頼は、申請したその日にすぐ始まるの？** ツールの目安は「依頼から」の日数なので、実際には申請受付から依頼が出るまでの事務処理の時間が別途かかる場合があります。目安の日程はあくまで依頼後の所要日数であり、申請日当日に依頼が完了することを前提にした最短シナリオである点にご注意ください。

**認定結果が出たあとの、ケアプラン作成やサービス利用開始までの日数は分かる？** 分かりません。ツールは認定後の手続き（ケアプラン作成→サービス利用開始）についても順序は示しますが、この部分に対応する所要日数の一次データは持っていないため、断定的な日数は表示しません。順序の確認や具体的な段取りは、別記事「要介護認定の申請から結果が出るまでの流れ」のチェックリストもあわせてご確認ください。
