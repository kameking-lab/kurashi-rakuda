# 産後の手続きは何をいつまでに？出生届から児童手当まで

## 全体の流れ

産後の手続きは、期限が短いものと比較的余裕があるものが混在しています。まずは「役所への届出」「健康保険の加入」「お金にまつわる手続き」の3つの塊で全体像をつかんでおくと整理しやすくなります。

1. 出生届の提出（最優先。里帰り先など出産した場所の役所でも提出可）
2. 健康保険への加入（子の医療証・医療費助成の前提になるため早めに）
3. 児童手当の申請（自治体窓口。届出が遅れると受給開始が遅れる場合がある）
4. 乳幼児医療費助成の申請（自治体ごとに名称・内容が異なる）
5. 出産育児一時金・出産手当金など、加入していた健康保険側の手続き（勤務先や協会けんぽ等に確認）

多くの場合、出生届の提出と健康保険の加入を先に済ませてから、児童手当・医療費助成の申請に進むとスムーズです。手続きの多くは自治体窓口で同時にまとめて案内してもらえることが多いので、母子健康手帳や印鑑、本人確認書類などを一式持って窓口へ行くと二度手間になりにくいでしょう。

## ステップ別チェックリスト

### ステップ1: 出生届の提出
- [ ] 出生届の用紙を準備する（出産した医療機関で出生証明書欄に記入してもらう）
- [ ] 提出先を確認する（本籍地・所在地・出生地のいずれかの役所で提出可能）
- [ ] 届出の期限を役所または母子健康手帳の案内で確認する（生後まもなくのうちに提出するのが一般的）
- [ ] 母子健康手帳・印鑑・本人確認書類を持参する

### ステップ2: 健康保険への加入
- [ ] 家族の加入している健康保険（勤務先の健康保険または国民健康保険）に子を扶養として加入させる手続きを行う
- [ ] 加入先（勤務先の担当窓口または市区町村の窓口）に必要書類を確認する
- [ ] 保険証の発行を待ち、乳幼児医療証の申請に備える

### ステップ3: 児童手当の申請
- [ ] お住まいの自治体の児童手当担当窓口で申請書を入手する
- [ ] マイナンバー・振込先口座・請求者の健康保険証など、自治体が指定する必要書類を準備する
- [ ] 申請のタイミングによって手当の支給開始月が変わる場合があるため、早めに窓口で確認しながら手続きする

### ステップ4: 乳幼児医療費助成の申請
- [ ] お住まいの自治体の医療費助成の名称・対象年齢・自己負担の仕組みを確認する
- [ ] 子の健康保険証ができ次第、速やかに申請する
- [ ] 医療証が届くまでの受診時の扱い（一旦立て替えて後日申請できるかなど）を窓口で確認する

### ステップ5: 加入健康保険側の給付手続き
- [ ] 出産育児一時金・出産手当金など、対象となりそうな給付を勤務先や加入している健康保険（協会けんぽ等）に確認する
- [ ] 必要書類・申請期限を早めに確認し、里帰り出産の場合は郵送対応が可能か確認する
