# 調味料の「黄金比」はこう計算する — 味付け黄金比計算ツールの結果の読み方

## この記事はどのツールと対応している？

本記事は「味付け黄金比計算（人数・分量スケール）」ツール（ajitsuke-ougonhi）に対応しています。煮物・照り焼き・煮魚・めんつゆ・万能合わせだれ・すき焼きの割り下・天つゆ・酢の物（三杯酢）という和食の定番料理8品目について、料理の種類と人数（または基準となる調味料の分量）を入力すると、しょうゆ・みりん・酒・だしなど各調味料の使用量を大さじ・小さじ・mlで計算します。

## 計算の考え方

このツールがしていることは、実はとてもシンプルです。それぞれの料理には、調味料どうしの「配合比率」があらかじめ決まっています。たとえば煮物なら、だし:しょうゆ:みりん = 8:1:1という比率です。ツールは、この比率のうち基準となる1つの調味料（煮物なら「だし」）の分量が決まれば、残りの調味料の分量を比率どおりに掛け算するだけで計算します。

具体的には、まず基準となる調味料の分量（ml）を決めます。人数を入力した場合は、料理ごとにツール側があらかじめ設定した「ひとりぶんの目安量」に人数を掛けて基準量を求めます（この目安量は入力をわかりやすくするためのツール側の初期値で、公的機関が定めた一人前の基準ではありません）。基準となる調味料の分量を直接ml単位で入力することもでき、その場合はその数値がそのまま基準量になります。

基準量が決まったら、「基準量 ÷ 比率の基準側の数値」で倍率（スケール係数）を求め、他の調味料の比率にこの倍率を掛けて、それぞれの分量（ml）を計算します。たとえば煮物（だし:しょうゆ:みりん = 8:1:1）で「だし400ml」を基準にすると、倍率は400÷8=50。しょうゆは1×50=50ml、みりんも1×50=50mlになります。同じ考え方で、めんつゆ（だし:しょうゆ:みりん = 12:1:1）で「だし120ml」を基準にすると、倍率は120÷12=10。しょうゆは1×10=10ml、みりんも10mlです。比率の「基準側の数値」で割ってから他の数値に掛けるだけなので、ツールの中でブラックボックスな推定や補正は行っていません。

計算した各調味料の量(ml)は、大さじ1杯=15ml以上ならその量に応じて大さじ表記、5ml以上15ml未満なら小さじ表記、5ml未満はml表記でまとめて表示します（詳細画面ではml・大さじ・小さじの3通りをあわせて確認できます）。

## 結果の読み方

このツールが扱う8品目の比率は、出典の確かさが2段階に分かれています。

煮物（だし:しょうゆ:みりん = 8:1:1）・煮魚（酒:しょうゆ:みりん = 6:1:1）・めんつゆ（だし:しょうゆ:みりん = 12:1:1）・万能だれ〈照り焼き用〉（しょうゆ:酒:みりん = 1:1:1）・万能合わせだれ（みそ:酒:みりん:すりごま:水 = 1:1:1:1:1）の5品目は、農林水産省「伝統調味料は『黄金比』で手間いらず」（Let's！和ごはんプロジェクト）にそのまま掲載されている比率です。ツールの結果画面でも、この5品目には特別な注意書きは表示されません。

一方、すき焼きの割り下（しょうゆ:みりん:酒:砂糖 = 1:1:1:1）・天つゆ（だし:しょうゆ:みりん = 4:1:1）・酢の物〈三杯酢〉（酢:砂糖:しょうゆ = 3:2:1）の3品目は、同じ農林水産省の記事に比率の記載がなく、本サイトが出典として使える許可ドメインの範囲内でも、これらの比率を明記した公的機関の一次情報を見つけられませんでした。複数の一般的なレシピサイトで比較的よく紹介されている比率を目安として採用したものであり、地域や家庭によって別の比率（すき焼きの割り下なら関東風・関西風、天つゆなら「5:1:1」、三杯酢なら「1:1:1」など）が紹介されることも珍しくありません。たとえばすき焼きの割り下でしょうゆを基準に30mlにすると、比率1:1:1:1どおりにみりん・酒・砂糖もそれぞれ30mlという計算結果になりますが、この1:1:1:1という比率自体が公的機関の一次情報にもとづくものではない点に注意してください。この3品目を選ぶと、結果画面には「一般的に紹介されている目安（公的機関の一次情報ではない）」という注意書きが常に表示されます。

## よくある疑問

**すき焼きの割り下・天つゆ・酢の物の比率も、農林水産省が決めているものですか？** いいえ。この3品目は農林水産省の記事に比率の記載がなく、本サイトが出典として使える範囲内に公的機関の一次情報を確認できませんでした。複数の一般的なレシピサイトで比較的よく紹介されている比率を目安として採用しているだけなので、「これが正解」という位置づけではありません。

**基準にする調味料はどうやって決まっていますか？** 料理ごとに、比率の中心になっている調味料（煮物なら「だし」、照り焼きのたれなら「しょうゆ」など）があらかじめ決まっています。ツールの入力欄には、選んだ料理の基準調味料の名前がヒントとして表示されます。

**人数をとても多く（作り置きやイベント用の大量調理など）しても計算できますか？** はい。エラーにはならず、比率どおりにそのまま倍率を掛けて計算します。ただし、人数や分量が0以下、または数値になっていない場合は入力エラーになります。

**表示される分量はぴったりの数値ですか？** 表示は小数第1位までの四捨五入です。また、この比率自体、味の好み・地域・家庭によって差があるものなので、実際に作るときは味を見ながら微調整することをおすすめします。
