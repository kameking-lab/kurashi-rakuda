# 玄関の片付け・収納と避難経路・転倒事故の防止

## 全体の流れ

玄関の片付けは「避難経路を塞がない収納にする→転倒しやすい段差・履物に注意する→日常的な整理を続ける」の3段階で進めると効果的です。

1. 避難経路を塞がない収納にする — 出入り口付近に物を置かない
2. 転倒しやすい段差・履物に注意する — わずかな段差やスリッパのすべりに気をつける
3. 日常的な整理を続ける — 出したものは元の場所に戻す習慣をつくる

## ステップ別チェックリスト

### 避難経路を塞がない収納にする

- [ ] 東京消防庁の案内では「避難通路や、出入り口付近には、転倒、移動しやすい家具類を置かないようにしましょう」とされている
- [ ] 東京都防災ホームページでも「部屋の出入り口や廊下には家具類を置かないように、据え付けの戸棚に収納します」と案内されている
- [ ] 玄関に置く傘立て・靴箱などは通路をふさがない位置・サイズのものを選ぶ
- [ ] 来客時以外に靴を出しっぱなしにせず、下駄箱に収納する

### 転倒しやすい段差・履物に注意する

- [ ] 国民生活センターの見守り情報では、わずか1〜2センチの敷居やじゅうたんの端、室内のコードなどに足を引っかける事故が紹介されている
- [ ] スリッパが脱げたり靴下で滑ったりして転倒し、骨折に至った事例も報告されている
- [ ] 玄関の上がりかまちや敷居の段差に物を置かない
- [ ] 滑りにくい靴・スリッパを選ぶ

### 日常的な整理を続ける

- [ ] 靴は使用頻度に応じて下駄箱内の位置を決め、出しっぱなしを防ぐ
- [ ] 傘・レインコートなど季節用品は使う季節が終わったら収納にしまう
- [ ] 玄関マットはずれたりめくれたりしていないか定期的に確認する
