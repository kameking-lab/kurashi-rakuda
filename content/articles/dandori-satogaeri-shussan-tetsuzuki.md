# 里帰り出産で必要になる前後の手続きチェックリスト

## 全体の流れ

里帰り出産では、「出産前に現住所側でやっておくこと」「里帰り先で妊婦健診・出産を受けるための準備」「出産後に現住所側でやること」の3つの時期に分けて考えると整理しやすくなります。

1. 出産前：現在通院中の産科で里帰りの相談・紹介状の依頼
2. 出産前：里帰り先の産科の予約・受け入れ確認
3. 出産前後：加入している健康保険側への里帰り出産の届出確認（出産育児一時金の直接支払制度の扱いなど）
4. 出産後：里帰り先での出生届の提出（出生地の役所でも提出可能）
5. 出産後：現住所に戻ってからの児童手当・乳幼児医療費助成などの申請

里帰り先の産院は分娩予約が埋まりやすいため、多くの場合は妊娠中期までに里帰り先の産院へ連絡し、受け入れ可否を確認しておくことが勧められています。現住所の産科とは、健診記録の紹介状（診療情報提供書）を出産予定の時期から逆算して早めに依頼しておくと安心です。

## ステップ別チェックリスト

### ステップ1: 出産前・現住所側の準備
- [ ] 現在通院中の産科医に里帰り出産の意向を伝え、紹介状（診療情報提供書）の依頼時期を相談する
- [ ] 里帰り先の産院の候補をリストアップし、分娩予約の受付時期を確認する
- [ ] 里帰り先までの移動手段・移動時期（何週目頃に帰省するか）を決める

### ステップ2: 出産前・里帰り先の準備
- [ ] 里帰り先の産院に予約を入れ、必要な検査・持ち物を確認する
- [ ] 母子健康手帳・健診補助券（受診票）を里帰り先の医療機関でも使えるか確認する（自治体によって扱いが異なる場合がある）
- [ ] 産後ケア事業など、里帰り先の自治体独自の支援制度の有無を確認する

### ステップ3: 出産前・お金まわりの確認
- [ ] 加入している健康保険（勤務先の健康保険または国民健康保険）に、里帰り先の医療機関でも出産育児一時金の直接支払制度が使えるか確認する
- [ ] 直接支払制度が使えない産院の場合の立て替え・請求方法を確認する

### ステップ4: 出産後・里帰り先での手続き
- [ ] 出生届を里帰り先（出生地）の役所で提出できることを確認し、必要書類を準備する
- [ ] 里帰り先で発行された出生証明書等の書類を、現住所に戻った後の手続き用に保管しておく

### ステップ5: 現住所に戻った後の手続き
- [ ] 児童手当・乳幼児医療費助成の申請を現住所の自治体窓口で行う
- [ ] 子の健康保険加入手続きを進め、母子健康手帳の産後の記録欄を確認する
