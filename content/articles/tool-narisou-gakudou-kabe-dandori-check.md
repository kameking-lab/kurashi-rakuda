# 学童保育に『全国共通の開所時間・締切』がない理由 — 小1の壁 段取りチェックの時期区分の考え方

## この記事はどのツールと対応している？

本記事は「学童・小1の壁 段取りチェック」ツール（gakudou-kabe-dandori-check）に対応しています。入学（予定）年度を入力すると、今が「入学前年の秋頃」「入学直前（年明け〜3月）」「入学後」のどの時期に当たるかを判定し、時期別のやることチェックリストを表示するツールです。保育園・幼稚園から小学校＋学童保育（放課後児童クラブ）への移行にともなう、いわゆる「小1の壁」への備えを整理するために使います。本記事では、ツールが**時期区分をどう計算しているか**と、**なぜ開所時間やお迎え締切時刻・申込締切日という具体的な数字を一切出さないのか**という2つの点に絞って説明します。

## 計算の考え方

### 学年オフセットの求め方（4月始まりの年度で考える）

ツールへの入力は「入学（予定）年度」（西暦）ひとつだけです。まだ入学していない場合は入学予定の4月を含む年度、すでに入学している場合は入学した年度を入れます。これと今日の日付から、次の計算で「学年オフセット」を求めます。

```
fiscalYearOf(日付) = 4月以降ならその年、1〜3月なら前年（4月始まりの年度）
学年オフセット = 今日が属する年度 − 入学（予定）年度
```

学年オフセットが0〜5の範囲なら、すでに入学しています（学年＝学年オフセット＋1。小学1年生〜小学6年生）。学年オフセットが負の値なら、まだ就学前です。学年オフセットが6以上になったら、本ツールが対象とする学年（小学生）を超えているものとして扱い、エラーにはせず「対象学年の範囲外」という情報として表示します。

入力の年度ひとつだけで「入学予定」と「現在の学年」の両方の考え方をカバーできるのは、この2つが実は同じ値の裏表だからです。今年度入学した子なら今年度の年度を、3年前に入学した子（今の学年オフセットが2で小学3年生）なら3年前の年度を入れれば、同じ計算式で今の学年が求まります。

### 就学前だけ、さらに3段階に分ける

学年オフセットが負の値（就学前）の場合だけ、今日の日付を使ってさらに細かい時期区分を判定します。

```
入学前年の9月に入ったら         → 「入学前年の秋頃」
（それより前は「入学まで時間があります」）
入学年の1月に入ったら           → 「入学直前」
入学年の4月に入ったら（学年オフセット0） → 「入学後」
```

この3区分の境目は月の頭を基準にした単純な比較で、複雑な日付の逆算はしていません。うるう年をまたぐ場合も、年月日の文字列同士をそのまま比較しているだけなので、特別な補正は不要です。

### なぜ開所時間・お迎え締切時刻・申込締切日を計算しないのか

このツールの一番の特徴は、**具体的な時刻・締切日を一切出さない**ことです。理由は、学童保育（放課後児童健全育成事業）の基準の法的な性質にあります。

令和2年4月に施行された第9次地方分権一括法により、放課後児童支援員の員数・資格を含む本基準の全ての項目が「従うべき基準」から「参酌すべき基準」に変わりました。「参酌すべき基準」とは、市町村が条例を定めるときに参考にすべき目安のことで、国が直接強制する全国一律の最低基準ではありません。

具体的には、国の基準（省令）は次のような時間数・日数の目安を示しているだけで、実際の時刻は定めていません。

- 休業日（夏休み等）：開所時間8時間以上を原則
- 休業日以外の日（平日）：開所時間3時間以上を原則
- 年間開所日数：250日以上を原則

いずれも「当該事業所ごとに定める」とされており、実際に何時から何時まで開所しているか、お迎えの締切が何時かは、クラブ・自治体ごとにばらばらです。さらに、学童保育の利用申込みの締切日については、そもそも国の基準に規定そのものがありません。市町村・クラブが独自に定めています。

こうした事情から、本ツールは「◯時までにお迎え」「◯月◯日締切」のような架空の全国共通値を作らず、「入学前年の秋頃」「入学直前」「入学後」という粒度の粗い時期区分でのチェックリストの提示にとどめています。開所時間・締切に関するチェックリストの項目も、必ず「クラブ・自治体ごとに確認してください」という案内で終わるようにしています。

### 待機児童の実態も、隠さず示す

対象学年は制度上、小学1年生から小学6年生までですが、実際にどの学年まで受け入れるかは市町村・クラブごとに異なります。それが数字にも表れていて、1つの「支援の単位」（クラス相当）の定員はおおむね40人が国の参酌基準である一方、令和7年5月時点で学童保育を利用できなかった児童（待機児童）は全国で16,330人おり、学年別に見ると小学4年生が5,589人と最も多くなっています。低学年を優先する選考のしくみにより、学年が上がるほど利用できないケースが増える、いわゆる「小4の壁」が統計にも表れている形です。

一方で、同年10月時点の速報値では待機児童数は7,363人まで減少しています。年度途中の退所などで空きが出るケースが実際にあるためで、4月の時点で利用できなかったとしても、年度途中の入所を継続的に確認する価値があるという実務上の示唆になります（ただし速報値であり確定値ではありません）。ツールはこの実態を、不安をあおる方向にも、大丈夫だと安心させすぎる方向にも寄せず、両方の数字を並べて示しています。

## 結果の読み方

結果画面では、まず「今の時期」（入学まで時間があります／入学前年の秋頃／入学直前／入学後／対象学年の範囲外のいずれか）が表示されます。あわせて、3つの時期別チェックリスト（入学前年の秋頃までに／入学直前に／入学後に）がすべて表示され、今の時期に対応するチェックリストだけに目印が付きます。まだ先の時期のチェックリストも消さずに表示し続けるので、先を見越した準備にも使えます。

チェックリストの下には、対象学年・支援の単位の定員・開所時間の参酌基準・待機児童の実態という4つの制度上の事実が、常に参考情報として表示されます。これらはすべて国の基準・全国統計であり、あなたが住んでいる自治体・利用予定のクラブに直接当てはまる保証はありません。「参酌基準」という言葉が示すとおり、実際の運用は自治体・クラブごとに確認する必要がある、という前提を崩さずに読んでください。

なお、チェックリストの項目そのもの（「◯◯を確認する」等）は、公的な統計調査や法令で定められた基準に基づくものではなく、一般的に知られている段取りをまとめた目安です。この点もツール内に常時表示しています。

## よくある疑問

**入学予定年度と現在の学年、どちらを入れればいいの？** どちらでも構いません。まだ入学していない場合は入学予定の4月を含む年度を、すでに入学している場合は入学した年度を入れてください。今の学年は、入力された年度から自動的に計算されます。

**学童保育の開所時間は、結局何時から何時まで？** 本ツールも本記事も、その具体的な時刻はお答えできません。国の基準は時間数（休業日8時間以上・平日3時間以上）の目安を示すだけで、実際の時刻は事業所ごとに定めるとされているためです。利用予定のクラブ、またはお住まいの市区町村の担当窓口に直接ご確認ください。

**待機児童が多いと聞いて不安です。うちの子は大丈夫でしょうか。** 全国の統計値（学年別の待機児童数）だけでは、個別のクラブの状況は分かりません。地域差も大きく、実際の定員・待機状況は利用予定のクラブ・自治体に確認するのが確実です。ツールが示す16,330人という数字は「入れないかもしれない」という不安を煽るためではなく、年度途中に空きが出る実態とあわせて、早めの情報収集と継続的な確認の重要性を伝えるために表示しています。

**学年オフセットが6以上になったら、エラーになりますか？** いいえ、エラーとしては扱いません。本ツールが対象とする学年（小学1年生〜小学6年生）を超えている、という情報として表示するだけです。中学生以降のお子さんについて誤って入力した場合の確認にも使えます。
