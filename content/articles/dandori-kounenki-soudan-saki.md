# 更年期の相談先はどこ？受診前に準備すること

## 全体の流れ

更年期に関する不調は、体の症状（ほてり・発汗・動悸など）と心の症状（気分の落ち込み・イライラなど）が同時に出ることも多く、「まずどこに相談すればよいのか分からない」という声がよく聞かれます。この記事は症状の診断や治療方針を示すものではなく、あくまで相談先を探すときの段取りと、受診前に準備しておくと話がスムーズになる情報の整理の仕方を扱います。

1. 相談先の候補を知る（婦人科・更年期外来、かかりつけ医、自治体の相談窓口など）
2. 自分に合いそうな相談先を選ぶ（症状の内容や、通いやすさで検討する）
3. 受診前に自分の症状・生活状況をメモにまとめる
4. 予約・受診し、必要に応じて他の診療科や専門外来を紹介してもらう

体調や気分の不調は「更年期のせいかどうか」を自己判断せず、まずは相談できる窓口につながることを優先するとよいとされています。婦人科では更年期に関連する相談を専門的に扱っていることが多い一方、症状によっては他の診療科の受診が適していることもあるため、迷う場合はかかりつけ医や自治体の相談窓口に「どこに行けばよいか」を相談するところから始める方法もあります。

## ステップ別チェックリスト

### ステップ1: 相談先の候補を知る
- [ ] 近隣で更年期外来・女性外来を掲げている婦人科があるか調べる
- [ ] かかりつけ医（内科など）がある場合は、そこに相談して適切な診療科を紹介してもらえるか確認する
- [ ] お住まいの自治体の保健センターや女性の健康相談窓口の有無を調べる

### ステップ2: 相談先を選ぶ
- [ ] 体の症状が中心か、気分の落ち込みなど心の症状が中心かで、婦人科と他の診療科のどちらを先に検討するか考える
- [ ] 通院のしやすさ（自宅や職場からの距離、診療時間帯）を確認する
- [ ] 初診でも相談しやすそうか、電話やウェブサイトで事前に確認する

### ステップ3: 受診前の準備
- [ ] 気になる症状（いつから、どのくらいの頻度で、どんな場面で出るか）をメモにまとめる
- [ ] 直近の月経の状況（周期の変化など）を分かる範囲でメモする
- [ ] 服用中の薬やサプリメント、他に通院している病気があれば書き出しておく
- [ ] 家事・仕事・介護など、生活面で負担に感じていることがあれば合わせてメモしておく

### ステップ4: 受診・相談する
- [ ] 予約が必要かどうかを事前に確認する
- [ ] 準備したメモを持参し、症状や生活状況を伝える
- [ ] 必要に応じて他の診療科や専門外来への紹介を受ける
