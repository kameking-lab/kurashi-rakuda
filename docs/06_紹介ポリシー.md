# 06_紹介ポリシー — アフィリエイト・商品紹介の掲載基準

作成日: 2026-07-17 ／ 適用範囲: 本サイトのすべてのページ・SNS 投稿・メール/Push 通知

本ポリシーは [00_企画書](00_企画書.md) のブランド哲学を商品紹介の場面に適用した**掲載可否の判定規則**である。本ドキュメントに収益試算・売上目標は書かない。今後も書かない。

---

## 1. 大原則

商品・サービスを紹介してよいのは、**その商品で主婦・キャリアウーマンの生活が確実に改善される場合のみ**。

「紹介できる商品が見つからない」は失敗ではなく通常の状態。無理に探さない。紹介ゼロの記事・ツールが大多数であることを健全とみなす。

## 2. 掲載4条件（すべて満たさない限り掲載不可）

1. **家族テスト**: 自分の家族（母・妻・姉妹・娘）に、報酬ゼロでも本気で勧められるか。勧めるときに一瞬でもためらうなら不可
2. **リンク削除テスト**: アフィリエイトリンクをすべて消しても、記事が読み物として完結し、ユーザーの課題が解決するか。リンクがないと成立しない記事は不可
3. **主目的テスト**: 記事の主目的が課題解決であり、商品紹介は解決手段の一つとして従属的に登場するか。「紹介するために書いた記事」は企画段階で却下
4. **欠点開示テスト**: その商品の欠点・向かない人・より安い代替手段（無料の代替、自治体サービス、型落ち品等）を本文中に正直に書いているか。欠点が書けない＝理解が浅い＝掲載不可

## 3. 禁止事項（ブランド哲学からの派生）

- 「おすすめ○○選」型ランキング記事の作成（比較が必要な場合は比較**表**を課題解決記事の中に置く。順位はつけない）
- PR・タイアップ・寄稿による商品紹介の受託（打診が来ても全件断る）
- 成約率を上げるための施策全般: CTA ボタンの色/文言最適化、リンクの複数回設置、離脱防止ポップアップ、限定・緊急を演出する文言（「今だけ」「残りわずか」）
- 商品名を含む煽りタイトル、体験談の演出・創作
- 紹介報酬の高さを理由とした商品選定（選定プロセスに報酬額の情報を持ち込まない。掲載可否を決めた**後**にアフィリエイトプログラムの有無を確認する順序を厳守）
- ASP の「特単」交渉・受諾（単価と編集判断の癒着を構造的に断つ）

## 4. 掲載時の必須表記

- リンクの直前または記事冒頭に、定型文で明示: 「このリンクから購入すると当サイトに紹介料が入ります。紹介料の有無は掲載判断に影響していません（[紹介ポリシー](/policy)）」
- 本ポリシー自体を公開ページ `/policy` として全文掲載する（透明性がブランドの核）
- 景品表示法・ステマ規制（2023年10月施行）に完全準拠。アフィリエイトリンクには `rel="sponsored"` を付与

## 5. 判定フロー（運用手順）

```
記事/ツール企画 → ユーザーの課題解決として完成させる
  → 完成後に「この課題の解決に物理的な商品/サービスが必要か？」を問う
    → 不要（大多数）: 紹介なしで公開
    → 必要: 掲載4条件チェックリストを起票 → 4条件すべて Yes → 掲載
                                        → 1つでも No → 商品名は出さず一般名詞で言及（例:「電動鼻吸い器」とだけ書きリンクなし）
```

- チェックリストはリポジトリ内で管理（`docs/affiliate-checks/` に1商品1ファイル、判定根拠を記録）。誰がいつ何を根拠に掲載を決めたかを追跡可能にする

## 6. 掲載後の見直し

- 掲載中の全商品を**年2回**再判定（値上げ・改悪・より良い代替の登場をチェック）
- ユーザーから「合わなかった」等の声が届いたら都度再判定。再判定で条件を割ったら即時リンク削除（記事はリンク削除テストを通っているので、そのまま成立する）
- リンク切れ・販売終了は検知次第削除（VPS 自律改善エンジンに監視ジョブを追加。[08_タスク分解](08_タスク分解.md) 参照）

## 7. 具体例（判定イメージ）

- ✅ 掲載しうる例: 離乳食の冷凍ストック記事における小分け冷凍トレー。無印・100均の代替も併記し、「週1回まとめて作る人以外には不要」と明記できる場合
- ❌ 掲載しない例: ウォーターサーバー（家計改善に反するケースが多く「確実に改善」を満たさない）／高額な情報商材・講座全般／サプリ等の健康食品（YMYL かつ効果の個人差が大きく「確実」と言えない）／ベビー用品の「おすすめ10選」形式全般
- 保険・金融商品の紹介（保険相談への送客等）は**全面禁止**。お金カテゴリは計算ツールと制度解説のみで完結させる
