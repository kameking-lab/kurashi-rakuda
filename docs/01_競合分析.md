# 01_競合分析 — 主婦・女性向けWebサービス市場（2026年7月時点）

作成日: 2026-07-17 ／ 手法: WebSearch・WebFetch による一次情報確認（媒体資料PDF・公式リリース・IR・アプリレビュー実査）。数値は自社公表値を含む。

## 1. 市場環境サマリ

- 出生数は 2025年 67.1万人（10年連続過去最少・出生率1.14）。年間第1子出生数は約32万人で、妊娠期メディアの実質市場はこの規模しかない
- 2024年以降、広告単価下落がメディア事業を直撃（エキサイトHDが決算で明言、影響約2億円）。AI Overview によるパブリッシャーのクリック減も進行し、**「SEO記事×広告一本足」モデルが構造的に崩れている**
- 撤退・縮小年表: ウィメンズパーク終了(2022/01)／たまひよプレミアム終了(2022/01)／keisan英語版閉鎖(2023/09)／コズレ被買収(2024/09)／ベネッセHD MBO非公開化(2024/05)／**ゼクシィBaby 全サービス終了(2025/12)**／keisan カシオ本体から子会社移管＋広告導入(2025/07)／ベビーカレンダー CFO不正発覚・YouTube収益化停止(2026)
- 生き残り組の共通行動は「広告依存からの脱却」: B2G(ママリ)、DMP(たまひよ)、リアルイベント(ninaru)、課金(ルナルナ・あすけん)、人材紹介(しゅふJOB)
- **本サイトへの含意**: 競合が広告収益の維持に苦しみ広告密度を上げるほど、「完全無料・広告臭ゼロ」の相対価値は上がる。市場縮小はメディア(広告枠)ビジネスには逆風だが、低コスト・ツール型の本サイトには参入好機

## 2. 主要競合の個票

### 2.1 ママリ（コネヒト／KDDI 100%子会社）
- 規模: アプリ会員400万人以上（公称）、総リーチ約898万人。匿名Q&Aが中核
- BM: 広告・タイアップ＋課金(プレミアム月400円)＋B2G(自治体子育てDX)＋データ事業
- 強み: 数百万件のQ&A蓄積（2026年4月からAI要約に活用開始）、KDDI資本、カテゴリ認知No.1
- 弱み: コミュニティ民度問題が慢性化（§3の不満実例）、公称会員数が数年更新されず鈍化の可能性（推定）
- 本サイトとの関係: Q&A(共感・体験談)が主戦場であり、**ツール(即答)領域では直接競合しない**。「ググっても分からないからママリで聞く」の手前を計算ツールで先取りする

### 2.2 たまひよ（ベネッセ／2024年MBOで非公開化）
- 規模: 初産ママの約92%がアプリDL（産院ルートで分娩の60%カバー）。アプリMAU約51万・WEB MAU約381万（2026年版媒体資料）
- BM: 広告(タイアップ・DMP)＋EC・物販。課金と掲示板からは撤退済み
- 強み: 産院という独自チャネルと30年ブランド。弱み: アクティブ規模は見かけより小さく、紙の縮小とPE傘下の再編圧力（推定）
- 本サイトとの関係: 妊娠期の初回接点は取られている前提で、**「アプリを入れるほどではない単発の調べもの」**（週数計算・手続き・お金）を検索経由で取る

### 2.3 コズレ（2024年アイドマHDに100%子会社化）
- 規模: 月間UU300万・会員90万人以上。月齢・日齢セグメント配信が特徴
- BM: 企業向けマーケソリューション（セグメントメール・調査・プレゼントキャンペーン）
- 弱み: 検索流入依存の記事メディアで、単独成長できず被買収に至った（推定）。プレゼント応募→営業電話への不満も（§3）
- 本サイトとの関係: 「会員情報と引き換えの無料」に対し、**登録不要の無料**で対置する

### 2.4 トモニテ（旧MAMADAYS／エブリー、2026年8月上場予定）
- 規模: SNS合計フォロワー200万人（子育てSNS No.1公称）。アプリ会員基盤は薄いSNSリーチ型
- BM: 広告・SNSタイアップ中心。強み: 動画制作力。弱み: 広告品質への不満が最も深刻（中国語の全画面広告・音あり動画広告、§3）
- 本サイトとの関係: SNS上の認知獲得手法は参考になるが、サイト体験では真逆を行く

### 2.5 ウーマンエキサイト（エキサイトHD）
- 規模: 月間約2.7億PVでママ向け最大級。コミックエッセイが中核
- 弱み: 広告単価下落で2024年に大幅減収減益（実績）。グループはメディカル事業へ投資シフト
- 本サイトとの関係: 読み物(暇つぶし)需要であり、課題解決ツールとは市場が異なる。**PVが大きくても広告単価に運命を握られる**という反面教師

### 2.6 しゅふJOB（ビースタイルHD、2024年12月上場）
- 規模: 登録会員145万人・掲載求人10万件超。BM: 求人企業課金（掲載1.2万円〜）
- 強み: 主婦特化の条件検索（扶養枠内・時短・在宅）と毎月の時給調査による専門性ブランド
- 弱み: タイミー等スポットワークの急拡大が競合圧力。増収鈍化・純利益減（2025年3月期）
- 本サイトとの関係: 求人は扱わないが、**「扶養の壁・時短給与の計算」はしゅふJOBユーザーの前段の悩み**。キャリアカテゴリの検索需要で接点があり、将来の提携候補（送客ではなく情報連携）

### 2.7 ベビーカレンダー（東証グロース）
- 専門家監修メディア＋産婦人科IT。2023年赤字→M&Aで回復→**2026年 前CFO不正発覚で決算4.5ヶ月延期、YouTube収益化停止で売上の約17%を喪失**、減収減益予想
- 教訓: 専門家監修という E-E-A-T 資産があっても、収益源の一極依存と統制不備で信頼は毀損する

### 2.8 ninaru シリーズ（エバーセンス、非上場）
- シリーズ累計1,100万DL、「妊婦の86%が利用」。少子化下でも2025年10月にMAU過去最高の逆行成長
- 週数・月齢連動の毎日配信＋パパ・祖父母への拡張＋リアルイベント（2026年〜）が強み
- 本サイトとの関係: **週数・月齢連動 UX の完成形**として最重要ベンチマーク。ただし不安を煽るコラム（「流産かも」等）への不満もあり、「煽らない週数情報」に差別化余地

### 2.9 ke!san — カシオ高精度計算サイト【ツール系の最重要事案】
- **日本語版は稼働中だが、2025年7月にカシオ本体から子会社リプレックスへ移管、ドメインも keisan.casio.jp → keisan.site に変更**。実行履歴・ブックマークはリセット。英語版は2023年9月に閉鎖済み
- **移管と同時にサーバー費用捻出のため Google AdSense を導入** — 「大手の信頼性×広告なし計算ツール」というポジションが2025年に消滅した
- 検索需要調査でも keisan.site は身長予測・基礎代謝・失業保険・介護休業給付金等で依然上位に居るが、生活文脈のUI・スマホ最適化・制度改定の説明は弱い
- **本サイトへの含意（最重要）**: 空いた「広告臭のない高品質計算ツール」の座を、女性ライフステージ領域から取りに行く。kurashi-keisan（汎用生活計算）と本サイト（女性特化）の2面展開はこの空席を面で押さえる戦略になる

### 2.10 その他
- ゼクシィBaby（リクルート）: **2025年12月で全サービス終了**（雑誌休刊・EC含む）。リクルートですら広告×メディア型で撤退した事実は、同モデルの限界の証拠
- ルナルナ: 累計2,300万DL・有料会員200万人。課金型で健在。ただし「いつの間にか課金しないと見れない部分が増えた」不満あり → **無課金で完結する体験**が対抗軸
- あすけん: 会員1,400万人超で急成長。記録・課金型ツールの勝ちパターン
- ママスタ: 月間9.6億PV。2025年にコイン課金導入。掲示板文化はマウント問題と表裏一体

## 3. 利用者の不満（実例・出典つき）

アプリレビュー・知恵袋・掲示板の実査から。**不満は広告の「量」より「質」に集中**している。

1. トモニテ:「全て中国語の全画面広告…閉じるボタンも中国語で殆ど詐欺に近い」「日記をつけると音あり動画広告が流れ、寝た子の横で音が鳴って焦る」（App Store ★2、2024年）
2. ベビーカレンダー:「『とじる』を押すとサイトに飛ぶ詐欺みたいな広告。うっとおしすぎてストレス」＋プレゼント登録後の営業電話報告（レビューまとめサイト実査）
3. たまひよ:「投稿するたびに広告」「スクロール中に誤タップで広告に飛ぶ」「ルーム画面の半分が広告」（2023年）
4. ママリ:「触っていないのに勝手にゲーム広告が出る」（レビュー実査）＋「マウント合戦」「質問を読まない回答」「CM後に民度低下」（知恵袋2024-2025・App Store）
5. ママスタ:「強制30秒動画広告…だんだんママスタから遠のいていく」（本体掲示板、2023年）
6. ウーマンエキサイト:「漫画より広告を見ている時間の方が多い」「アプリで読むとスマホが爆熱」（2023年）
7. ルナルナ:「いつの間にか課金しないと見れない部分が多くなった。乗り換えます」
8. ninaru:「『流産かも』『9週の壁』など不安を煽るコラムが多い」「広告応募は個人情報流出が心配」
9. 検索全般:「育児ジャンルの上位はドメイン優遇の大手の似た記事ばかり」（SEO汚染への不満、はてな匿名ダイアリー等）

**→ 本サイトの設計原則（[04_サイト設計](04_サイト設計.md) UI十原則）は、この不満リストの一つずつを裏返したものになっている**（偽閉じるボタン→広告枠自体を持たない／音あり広告→ポップアップ全面禁止／営業電話→登録不要／煽りコラム→煽り禁止／課金の壁→完全無料）。

## 5. 競合分析の結論

1. 記事×広告のメディアモデルは崩壊過程にあり、大手が撤退・多角化する中で**「低コスト・ツール型・完全無料」は唯一逆風を受けないポジション**
2. 妊娠期の初回接点（アプリ）は、たまひよ・ninaru が抑えており、正面から奪いに行かない。**検索経由の単発課題解決**と、離乳食以降〜介護までの長いステージで信頼を貯める
3. 差別化の本体は機能ではなく**哲学**。競合は構造上（上場・広告主・PE）広告密度を下げられない。個人運営×既存インフラ活用で損益分岐が極端に低い本サイトだけが「広告臭ゼロ」を維持できる
4. リスク: (a) 出生数減による分母縮小 → 介護・キャリア・家事の全世代領域でヘッジ済み (b) AI Overview による検索流入減 → 「AIが答えを要約できない対話的ツール」（入力→計算）はAI要約に食われにくい。ツール中心はこの点でも防御的
