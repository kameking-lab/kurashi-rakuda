需要: 2026-07-17 の SERP 実査による推定（高=1万超目安/中=1千〜1万/低=1千未満。詳細根拠は調査ログ）

重複: kurashi-keisan.jp の既存160ツール（2026-07-17 sitemap 実査）との関係。無印=重複なし

## 棲み分けの基本方針（kurashi-keisan.jp との関係）

sitemap 実査の結果、**単発のお金計算（手取り・ふるさと納税・失業給付・育休給付・出産手当・児童手当・扶養の壁・光熱費ほか）は kurashi-keisan に既に存在する**。そこで:

1. **汎用・単発計算は作らない** → kurashi-keisan へ送客（相互リンク。[05_技術構成](05_技術構成.md) §6）
2. **本サイトが作るのは「女性のライフイベント文脈型」** → ①複数制度をまたぐ複合シミュレーション（例: 産休〜育休の総額タイムライン）②スケジュール・段取り・チェックリスト系（非計算）③検索型データベース（冷凍保存期間・洗濯表示）④週数・月齢連動のパーソナライズ
3. 既存ツールと部分重複する場合は表の「重複」列に対応方針を明記: **リンク**=作らず送客／**移植**=計算エンジンを流用し文脈UIで再構築／**統合**=本サイトの複合ツールの一部品として吸収

## 優先ツール TOP10（決定事項ドラフト）

需要×競合の薄さ×ブランド体現度で選定。

1. **月齢計算**（#17）— SERP 最薄。子育てカテゴリ全ツールのハブ
2. **保育料計算 全国版**（#21）— 空白地帯。データ整備が参入障壁になる本命
3. **予防接種スケジューラー**（#18）— 高需要。Push リマインダーで「アプリ級」体験へ
4. **扶養の壁シミュレーター2026**（#55）— 2025〜26改正ラッシュの追い風
5. **時短勤務給料・給付シミュレーター**（#78）— 2025年新制度で競合最薄
6. **産休育休まるごとお金シミュレーター**（#4）— 単発計算の寄せ集めを統合する独自価値
7. **離乳食の量 早見ツール**（#19）— 大手は静的表のみ、ツール化空白
8. **児童手当計算 2026年度**（#20）— 制度拡充後の専業ツール僅少
9. **介護保険 自己負担シミュレーター**（#90）— シミュレーター不在の空白＋ペルソナ5開拓
10. **献立自動提案**（#40）— Web 完結ツール不在。「考えなくていい」価値の象徴
