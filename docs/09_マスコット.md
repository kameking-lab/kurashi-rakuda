# 09_マスコット — 子ラクダキャラクターブリーフ＆商標予備調査

作成日: 2026-07-17 ／ 決裁事項: マスコットは子ラクダ・かわいい路線で確定（社長承認済み）。キャラ名は候補5案から社長選定。

---

## 1. キャラクターブリーフ

### 1.1 コンセプト

**「あなたの生活の荷物を、代わりにコブへ載せて運ぶ子ラクダ」**

ブランド哲学（完全無料・急かさない・売りつけない）の擬人化。ユーザーが疲れているとき（深夜授乳・通勤・介護の合間）に画面の隅にいても邪魔にならず、そっと手伝う存在。

### 1.2 性格

- のんびり・マイペース。決して急かさない（「早くしないと損」の対極）
- 世話焼きだが押し付けない。頼まれたことだけ手伝う
- 正直。分からないことは「わからない」と言う。出典を添えるのが癖
- 何かを売ろうとしない。おすすめを聞かれても「あなたに必要ないなら、いらないと思うよ」と言える

### 1.3 口調（UIテキスト・SNSで統一）

- 一人称「ぼく」、語尾はやわらかい常体「〜だよ」「〜してね」「〜かも」
- 感嘆符は1文に1個まで。二重感嘆符・急かす命令形（「今すぐ！」）は禁止
- 例: 「この金額は2026年度の税率で計算したよ」「ゆっくりでだいじょうぶ」「それはぼくにも難しいから、専門の窓口を教えるね」

### 1.4 体型・ビジュアル仕様

- **2頭身の子ラクダ**（頭:体=1:1）。丸いシルエット、短い脚、小さな一つコブ
- コブの上に小さな布包み（風呂敷）を載せている＝「荷物を運ぶ」の記号。荷物は小さく、重そうにしない
- 目: 黒くて丸い、たれ気味。頬に薄いピンクの丸。まつげは描かない（性別を固定しない）
- 首元に深緑のスカーフ（ブランドカラーの差し色）
- 線: 太めの均一線（茶系 #7A5F3E）。塗り: フラット＋薄い影1段のみ

### 1.5 カラーパレット（デザイントークンと同期）

- 体: 砂色 #C7AF88 ／ 腹・口元: クリーム #F4EEE3
- コブ・たてがみ: 濃砂色 #B29368 ／ 線: #7A5F3E
- スカーフ・風呂敷: 深緑 #2E5039 ／ 頬: #E8B4A0
- 使用禁止: 彩度の高い赤・黄（セール感の排除。docs/04 UI十原則8と同一ルール）

### 1.6 NG例（やってはいけない描写）

- 写実的な体毛・リアルなラクダ顔（かわいい路線に反する）
- 走る・焦る・汗をかく・驚いて飛び上がる（急かさないキャラ設定に反する）
- 商品を持つ・指し示す・お金やセール札と一緒に描く（売りつけない）
- 泣き顔の多用（困り顔まで。ユーザーの不安を増やさない）
- 帽子・サングラス等の流行アイテム追加（経年劣化するため）、二足歩行の人間化

## 4. 商標予備調査（「くらしのラクダ」）— 2026-07-17 実施

**判断は社長。以下は事実の収集のみ**（Web検索ベースの予備調査であり、弁理士による正式調査の代替にはならない）。

### 4.1 調査手段と限界

- J-PlatPat・Toreru商標検索は動的サイトのため機械取得不可＝**公式DBでの完全一致・称呼検索は未実施**
- 代替として特許庁公報データを収載する民間サイト（patent-i.com 等）とWeb検索で確認（網羅性・最新性の保証なし）

### 4.2 確認できた事実

- 「くらしのラクダ」「暮らしのラクダ」の**完全一致の商標出願・登録情報はWeb上で発見されず**（不存在の証明にはならない）
- 完全一致の既存サービス・企業・店舗も**発見されず**
- ただし称呼「ラクダ」を含む商標は**142件**存在。関係しうる区分の例:
  - 登録6955878「エーギョーラクダ」（9・35・42類、株式会社Tokel）
  - 登録6865944「サイテンラクダ」（9・35・41・42類、共同印刷）
  - 登録6968969「テンショクラクダ」（35類、DSP株式会社）
  - 登録7019246「RAQUDA／ラクーダ」（35類）
  - 登録6835187 サントリーHD保有の「ラクダ」称呼商標（9・41類ほか）
- 楽天「ラクマ」は9・35・36・42類で登録多数（称呼1音違い）
- 名称要素が近い既存事業者: 「らくだ暮らしのサービス」（埼玉の生活支援業）／代行サービス「ラクダ」（株式会社ラクダ、2023年〜）／「RAKUDAクラウド請求」（SoLabo）／PHP研究所の生活情報誌『くらしラク〜る♪』

### 4.3 弁理士確認を推奨する論点（調査エージェントの指摘の要約）

1. J-PlatPat での正式な称呼検索（「クラシノラクダ」「ラクダ」）による先行商標の網羅的洗い出し（9・35・38・41・42・44類）
2. 「くらしの」が識別力の弱い付加語とされ「ラクダ」が要部認定された場合の、上記35類・42類の既登録（エーギョーラクダ、サイテンラクダ等）との類否
3. 「ラクマ」（楽天）との称呼類似の評価
4. 未登録でも先使用が問題になりうる「らくだ暮らしのサービス」「株式会社ラクダ」との関係
5. 出願する場合の指定役務の設計（どの区分まで押さえるか）
