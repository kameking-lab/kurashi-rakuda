# 子どもの身長予測（両親身長法）

## 概要

父親・母親の身長と子の性別を入力すると、広く知られた簡易式「両親身長法（Mid-Parental Height法の日本の育児領域における通称）」を用いて、成人時の予測身長のおおよその目安を計算するツール。

- 対象ペルソナ: ペルソナ1（佐藤美咲）を主対象。子の将来の身長が気になる保護者一般。
- docs/02_ツール一覧.md **#22**（カテゴリ2: 子育て、優先度 **S**、難易度 B）。
- YMYL区分: **中**（成長・発育に関する話題であり、断定的な予測と受け取られないよう配慮が必要。医療的な低身長判定等は一切行わない）。

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| 子の性別（sex） | 選択（男児／女児） | 必須 | 男児・女児のいずれかを選択。未選択はエラー |
| 父親の身長（fatherHeightCm） | 数値（cm、小数第1位まで） | 必須 | 100.0〜250.0の範囲。範囲外・未入力はエラー |
| 母親の身長（motherHeightCm） | 数値（cm、小数第1位まで） | 必須 | 100.0〜220.0の範囲。範囲外・未入力はエラー |

※本ツールは「両親の身長」を入力とする計算式であり、家族構成（実親・養親・ひとり親家庭・同性カップル等）を問わず「生物学的な遺伝情報としての父親役・母親役の身長」として入力する設計とする。該当する数値が分からない場合は入力不要（結果は表示しない）とし、無理な入力を求めない。

## ロジック仕様

両親身長法（父母の身長の平均をベースに、性差の補正値を加減する簡易式）を採用する。

```
if 性別 == 男児:
    予測身長(cm) = (父親の身長 + 母親の身長 + 13) / 2
elif 性別 == 女児:
    予測身長(cm) = (父親の身長 + 母親の身長 − 13) / 2
```

- 誤差範囲の表示: この式は統計的な目安であり、実際の成人身長は個人差が大きい。育児・小児科領域で「誤差はおよそ±9cm程度とされることが多い」と紹介されるため、本ツールでは参考レンジとして `予測身長 ± 9cm` を併記する（「必ずこの範囲に収まる」という保証ではなく、あくまで紹介されている目安幅であることを明記する）。
- 小数の扱い: 計算結果は小数第1位まで表示する（四捨五入しない。単純な割り算の結果をそのまま表示）。

## エッジケース・注意事項

- **父親・母親どちらかの身長が未入力**: 計算不可としエラー表示（両方必須）。
- **極端な入力値（例: 父親250cm・母親100cm等、現実的にまれな組み合わせ）**: エラーにはしないが、結果画面に「入力値をご確認ください」の注意表示を出す。
- **養子・非血縁の家庭等、生物学的な父母の身長が分からない場合**: 無理に入力させない。入力欄の説明文に「わかる範囲で構いません。無理に埋める必要はありません」と明記する。
- **性別が男女以外（インターセックス等）の子**: 本ツールの式は「男児／女児」の二値の統計式に基づくものであり、それ以外の分類には対応しない。該当する場合は目安として近い方を選ぶか、利用を控えてよい旨を注記する（強制しない）。
- **多胎児（双子等）**: 両親の身長は共通のため、同じ入力から複数回計算しても結果は同じになる（式自体は出生順位や一卵性/二卵性を考慮しない）。
- **成長期前の乳幼児への適用**: 本ツールは「成人時の予測」であり、現在の身長・成長曲線（パーセンタイル、#25）とは別物である旨を明記し、混同を避ける。

## YMYL配慮事項

- 医学的な低身長・成長障害の診断や判定は一切行わないことを明記する。
- 固定文言案（結果画面下部・常時表示）: 「この結果は両親の身長から計算した統計的な目安であり、実際の身長を保証するものではありません。栄養・睡眠・運動・個人差など様々な要因で変化します。成長について心配な場合は、乳幼児健診や小児科医にご相談ください。」
- 断定的な表現（「〜になります」等）を避け、「〜cm前後の目安です」「〜とされています」等のヘッジ表現を用いる。
- 最終更新日／準拠年度／出典（統計的経験式である旨）の3点セットを表示する。
