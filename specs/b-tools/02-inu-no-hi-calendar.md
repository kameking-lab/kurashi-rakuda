# 戌の日計算・安産祈願カレンダー

## 概要

最終月経開始日（または既知の出産予定日）を入力すると、日本の伝統的な安産祈願の風習である「妊娠5ヶ月に入って最初の戌の日」を自動計算し、あわせてその前後数回分の戌の日カレンダーを表示するツール。

- 対象ペルソナ: ペルソナ1・2・4（妊娠中の全ペルソナ）。腹帯を巻く安産祈願（帯祝い）で神社にお参りする日取りを決める際に使われる
- 分類: [docs/02_ツール一覧](../../docs/02_ツール一覧.md) カテゴリ1 #2「戌の日計算・安産祈願カレンダー」。優先度 **S**（Phase 1 MVP）。「専用ツールの質が低く狙い目」領域
- YMYL区分: **低**（[docs/03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.1）。十二支に基づく暦計算のみで、医学的判断・宗教的な優劣判断（大安・仏滅等の吉凶）は行わない
- ツール1「出産予定日・妊娠週数計算」と最終月経開始日の入力・週数変換ロジックを部品共有する（[docs/04_サイト設計](../../docs/04_サイト設計.md) §3.2 回遊設計「同じ悩みの人が次に使うツール」に両ツールを相互表示）

## 入力仕様

| フィールド名 | 型 | 必須/任意 | 説明・バリデーション |
|---|---|---|---|
| `inputMode`（入力方式） | 選択（`lmp` / `edd`） | 必須 | 「最終月経開始日から計算」または「出産予定日（医師の診断済み）から計算」のどちらかを選択 |
| `lmp`（最終月経開始日） | 日付（YYYY-MM-DD） | `inputMode=lmp` の場合必須 | 当日以前の日付のみ有効。基準日より330日以上前はエラー |
| `edd`（出産予定日） | 日付（YYYY-MM-DD） | `inputMode=edd` の場合必須 | 妥当な日付であること |
| `baseDate`（基準日） | 日付（YYYY-MM-DD） | 任意（デフォルト＝実行時の当日） | 「もう妊娠5ヶ月の戌の日を過ぎているか」の判定に使用 |
| `calendarCount`（表示する戌の日の件数） | 整数 | 任意（デフォルト6） | 1〜12の範囲。妊娠5ヶ月最初の戌の日を含め、以降の戌の日を何件カレンダー表示するか |

## ロジック仕様

十二支は12日周期で暦日に循環的に割り当てられる伝統的な体系であり、「戌の日」は12日に1度巡ってくる（国立天文台暦計算室が示す干支の60日周期の下位区分。十二支のみでは12日周期）。医学的判断は一切含まない。

### 1. 妊娠5ヶ月に入る日の算出

```
if inputMode == "lmp":
    LMP = 入力値
elif inputMode == "edd":
    LMP = EDD − 280日        // ツール1と同じネーゲレ概算法の逆算

妊娠5ヶ月開始日 month5Start = LMP + 112日   // 16週0日（ツール1のデータ表「5ヶ月＝16〜19週6日」に一致）
```

### 2. 戌の日の判定（12日周期）

```
定数 ANCHOR_INU_DATE = 2026-01-12   // 実在の戌の日（複数の社寺公開カレンダーで確認済み。データ表参照）

isInuNoHi(date):
    diffDays = date − ANCHOR_INU_DATE   // 日単位の整数（負値も可）
    return (diffDays mod 12 == 0)       // mod は負値でも0以上11以下に正規化して判定
```

### 3. 妊娠5ヶ月最初の戌の日の探索

```
d = month5Start
while not isInuNoHi(d):
    d = d + 1日
firstInuNoHi = d
```

- `month5Start` 自身が戌の日であるケースはそのまま採用する（探索ループの0回目で条件成立）

### 4. カレンダー表示用の戌の日リスト

```
list = []
current = firstInuNoHi
repeat calendarCount times:
    list.append(current)
    current = current + 12日
```

### 5. 経過判定（baseDate との比較）

```
if baseDate > firstInuNoHi:
    status = "妊娠5ヶ月最初の戌の日はすでに過ぎています"
    // このときも計算結果自体は変更せず、次の戌の日（リストの2件目）を「次にお参りしやすい戌の日」として案内
else:
    status = "これからです"
```

## データ表

| 定数・区分 | 値 | 出典・根拠 |
|---|---|---|
| 十二支の循環周期 | 12日 | 十二支は暦日に対して連続的に循環割当される伝統的な体系（干支＝十干十二支の60日周期のうち、十二支単独では12日周期）。国立天文台暦計算室「暦Wiki/干支」「暦Wiki/十二支」の解説に基づく |
| 戌の日の基準日（アンカー日） | 2026年1月12日（月曜） | 複数の社寺・出産準備専門店が公開する2026年（令和8年）戌の日カレンダーで実在確認済みの日付。同カレンダーに掲載される2026年の全戌の日（1/12, 1/24, 2/5, 2/17, 3/1, 3/13, 3/25, 4/6, 4/18, 4/30, 5/12, 5/24, 6/5, 6/17, 6/29, 7/11, 7/23, 8/4, 8/16, 8/28, 9/9, 9/21, 10/3, 10/15, 10/27, 11/8, 11/20, 12/2, 12/14, 12/26）はすべて12日間隔であり、本ツールの12日周期モデルと整合する |
| 妊娠5ヶ月の週数区分 | 16週0日〜19週6日 | ツール1（出産予定日・妊娠週数計算）のデータ表と共通。「1ヶ月＝4週」の慣用区分による |
| 妊娠期間の基本日数 | 280日（`edd` 入力時の LMP 逆算に使用） | ネーゲレ概算法。出典はツール1と同じ日本産科婦人科学会の定義 |
| 安産祈願（帯祝い）の風習 | 妊娠5ヶ月に入って最初の戌の日に腹帯を巻き、安産を祈願する | 日本の伝統的な風習（社寺・出産準備関連事業者の公開情報で広く共通する説明）。医学的な安産効果を示すものではなく、風習・文化的行事として紹介する |

**アンカー日の検証方法（実装時の確認事項）**: 2026年の公開戌の日カレンダーから任意の2日を選び、その日数差が12の倍数であることを確認する（本仕様書では上記30件すべてが12日間隔であることを確認済み）。年をまたぐ場合も同じ12日周期をカレンダー日数ベースでそのまま延長すればよく、うるう年による補正は不要（暦日数の実加算のため）。

## 出力仕様

| 項目 | 表示例 |
|---|---|
| 妊娠5ヶ月最初の戌の日 | 「2026年4月30日（木）」を大きく表示 |
| 妊娠5ヶ月に入る日 | 「2026年4月23日〜」（参考情報として） |
| 経過状況バッジ | 「これからです」／「もう過ぎています。次の戌の日は◯月◯日です」 |
| 戌の日カレンダー（複数件） | 表形式で `calendarCount` 件分の日付・曜日を列挙 |
| 計算根拠の明示 | 「最終月経開始日（2026年1月1日）+ 112日で妊娠5ヶ月開始日を算出し、その日以降で最初に巡ってくる戌の日を表示しています」 |
| 固定文言（YMYL配慮） | 下記の文言を常設 |
| 関連導線 | ツール1（出産予定日・妊娠週数計算）へのリンク |

## エッジケース・注意事項

| ケース | 扱い |
|---|---|
| `month5Start` 自身が戌の日 | そのまま採用（探索の0日目）。例: LMP=2026-05-20 の場合、妊娠5ヶ月開始日=2026-09-09 がそのまま戌の日 |
| `baseDate` が `firstInuNoHi` より後 | エラーにはせず「すでに過ぎています」バッジ＋次の戌の日を案内する（安産祈願は5ヶ月の間であれば別の戌の日にお参りする人も多いため） |
| 妊娠5ヶ月開始日がうるう年の2月をまたぐ | 日付演算は暦日数の実加算で行うため追加補正は不要。2月29日を含む期間でも12日周期の判定は変わらない |
| 年をまたぐ場合（例: 12月LMPで5ヶ月開始日が翌年1月） | アンカー日からの経過日数計算は年をまたいでも単純な日数差でよく、年ごとの補正は不要 |
| `inputMode=edd` かつ `lmp` も入力されている | `edd` を優先し、`lmp` は無視する（画面上は片方のみ入力可能なUIとする） |
| 六曜（大安・仏滅等）との重複希望 | 本ツールは十二支の戌の日のみを扱い、六曜との組み合わせ判定は将来検討（Phase 1スコープ外）である旨を注記する |
| 双子・多胎妊娠 | 単胎を前提とした一般的な目安である旨を注記（ツール1と同一の注記文を流用） |
| 未来日の `lmp` / 不正な `edd` | 入力エラーとしてツール1と同じバリデーションメッセージを表示 |

## YMYL配慮事項

YMYL区分は**低**（[docs/03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.1）だが、妊娠週数の算出を内部で伴うため、ツール1に準じた配慮を行う。

- 本ツールが行うのは**暦上の十二支の周期計算と最終月経開始日からの日付演算のみ**であり、安産祈願の効果や妊娠の経過・健康状態についていかなる保証・判断も行わない
- 結果画面に以下の固定文言を常設表示する:
  > 「戌の日は、妊娠5ヶ月に入って最初に巡ってくる十二支の日を計算したものです。安産祈願（帯祝い）は日本の伝統的な風習であり、この日にお参りすることが医学的な安産を保証するものではありません。妊娠経過に関して気になることがある場合は、必ず担当医にご確認ください。また、神社によっては予約制・混雑状況が異なるため、事前に参拝先へご確認ください。」
- 出典（十二支の暦法解説・社寺公開カレンダーによる実在日検証）と最終更新日をページ下部に表示
- 計算式（12日周期・妊娠5ヶ月開始日の算出式）をページ内に明示し、ブラックボックス化しない
- 神社・腹帯の特定商品・特定祈願所への誘導リンクは掲載しない（[docs/06_紹介ポリシー](../../docs/06_紹介ポリシー.md) の「生活が確実に改善される場合のみ」の条件に該当しないため、広告臭ゼロの原則を優先）
