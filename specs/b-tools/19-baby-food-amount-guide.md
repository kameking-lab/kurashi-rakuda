# 離乳食の量・固さ早見（月齢パーソナライズ）

## 概要

生年月日（または直接入力した月齢）から赤ちゃんの離乳食の進行段階を自動判定し、その段階における「食べ物の固さ」「1回あたりの目安量（食品群別）」「食事回数」をパーソナライズして一覧表示するツール。

- **対象ペルソナ**: ペルソナ1（佐藤美咲・下の子の離乳食の量が合っているか不安）が中核
- **docs/02 の該当番号**: #19（カテゴリ2「子育て」）。需要=中〜高／難易度=A／YMYL=中／優先度=S
- **差別化**: 既存の大手メディアは月齢区分ごとの静的な早見表（記事）のみを掲載している。本ツールは月齢を入力すると該当区分の情報だけを絞り込んで表示する「ツール型」であり、この形式は競合が手薄
- **スコープ**: 「量・固さの目安」という制度的・公的ガイドラインの転記＋月齢による絞り込み表示に限定する。個別の栄養指導・アレルギー診断・体重増加の適否判定は行わない（docs/03 §4.1「症状・治療・薬は扱わない」の原則に従う）

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| `birth_date`（生年月日） | 日付（YYYY-MM-DD） | 必須（`age_months` と排他、どちらか一方を入力） | 本日以前の日付であること。本日から起算して0〜36ヶ月前の範囲外（3歳より前・生まれる前）はエラー「生年月日をご確認ください」 |
| `age_months`（月齢を直接入力） | 整数（0〜36） | 任意（`birth_date` の代替入力） | 0〜36の範囲。`birth_date` が入力されている場合はそちらを優先し、この項目は自動計算表示用に使う |
| `is_premature`（早産児として修正月齢を使う） | boolean | 任意（デフォルト false） | true の場合、`gestational_weeks_at_birth` の入力が必須になる |
| `gestational_weeks_at_birth`（出生時の在胎週数） | 整数（22〜41） | `is_premature=true` の場合必須 | 22〜41の範囲。40週を満期の基準とする |
| `weaning_started`（離乳食を開始済みか） | boolean | 任意（デフォルト true） | false の場合、「まだ離乳食を始めていない場合の目安」メッセージのみ表示し、量・固さの表は非表示にする |

## ロジック仕様

### 月齢の算出

```
function calcAgeMonths(birth_date, today):
    # 「生後◯ヶ月」は暦月で切り捨てて算出する（例: 生後6ヶ月20日 → 6ヶ月として扱う）
    months = (today.year - birth_date.year) * 12 + (today.month - birth_date.month)
    if today.day < birth_date.day:
        months -= 1
    return max(months, 0)
```

### 修正月齢の算出（早産児オプション有効時）

```
function calcCorrectedAgeMonths(actual_age_months, gestational_weeks_at_birth):
    weeks_early = 40 - gestational_weeks_at_birth       # 満期(40週)より何週早く生まれたか
    if weeks_early <= 0:
        return actual_age_months                        # 正期産・過期産はそのまま実齢を使用
    months_early = round(weeks_early / 4.345)            # 週→月の概算換算（1ヶ月≈4.345週）
    return max(actual_age_months - months_early, 0)
```

修正月齢は「暦年齢から早産分の週数を差し引いた月齢」。医学的には修正月齢の使用可否・いつまで使うか（一般に1歳半〜2歳頃まで）は個別差があるため、本ツールでは参考値としてのみ提示し、「医師・乳幼児健診の判断を優先してください」と明記する（後述YMYL配慮事項）。

### 離乳段階の判定

```
function determineStage(age_months):
    if age_months < 5:
        return STAGE_BEFORE          # 離乳食開始前の目安
    elif age_months in [5, 6]:
        return STAGE_GOKKUN          # 離乳初期（ゴックン期）
    elif age_months in [7, 8]:
        return STAGE_MOGUMOGU        # 離乳中期（モグモグ期）
    elif age_months in [9, 10, 11]:
        return STAGE_KAMIKAMI        # 離乳後期（カミカミ期）
    elif age_months in [12, 13, ..., 18]:
        return STAGE_PAKUPAKU        # 離乳完了期（パクパク期）
    else: # age_months >= 19
        return STAGE_AFTER           # 幼児食への移行期（離乳完了、目安データ範囲外）
```

- 早産児オプションが有効な場合は `age_months` の代わりに `calcCorrectedAgeMonths()` の返り値を判定に用いる
- `weaning_started = false` の場合は判定処理をスキップし、開始前メッセージのみ表示する

### 表示データの絞り込み

判定された `stage` をキーに §データ表 の該当行のみを抽出して表示する。月齢が段階の境界に近い場合（例: 6ヶ月と7ヶ月の境目）は、次の段階のプレビューも「もうすぐ次の段階です」として簡易表示する（本表示ではなく参考リンク程度に留める）。

| 食品群 | 離乳初期（ゴックン期） | 離乳中期（モグモグ期） | 離乳後期（カミカミ期） | 離乳完了期（パクパク期） |

## 出力仕様

- 月齢サマリー: 「生後◯ヶ月（実齢）」／早産児オプション時は「修正月齢◯ヶ月（実齢◯ヶ月）」を併記
- 該当段階の名称・アイコン・固さの説明・食事回数
- 食品群別の目安量テーブル（表Bの該当列を抽出表示）
- 段階共通の注意事項（表C、常時表示）
- 次の段階までの残り目安（例: 「あと1ヶ月ほどで離乳後期（カミカミ期）の目安です」）
- 出典表示: 「厚生労働省『授乳・離乳の支援ガイド（2019年改定版）』を基に作成」＋最終更新日

## エッジケース・注意事項

- **境界値の扱い**: 月齢は暦月で切り捨てる（生後6ヶ月29日でも「6ヶ月」＝ゴックン期のまま。7ヶ月になった時点でモグモグ期に切り替わる）。この切り捨てルールは実装・表示の両方で明示する
- **個人差の強調**: 月齢はあくまで目安であり、実際の進み方は「首すわり」「支えると座れる」「食べ物への興味」「口に入れたものを舌で押し出す反応（哺乳反射）の減少」等の発達サインによって判断すべきものである旨を、結果画面の目立つ位置に明記する
- **早産児**: 修正月齢の使用有無・使用期間の判断は医学的な個別差が大きいため、本ツールでは参考値の提示に留め、「詳しくは乳幼児健診・かかりつけ医にご相談ください」と明記する
- **体調不良時**: 発熱・下痢・嘔吐等の体調不良時は離乳食を中断・元の段階に戻すことが一般的だが、本ツールは体調に関する入力・判断は行わない（症状・治療領域はスコープ外）
- **食物アレルギー**: 症状の有無・重症度の判定は一切行わない。「アレルギーが心配な場合は医師に相談」という誘導のみ
- **双子・多胎児**: 月齢は同一だが体格・発達に個人差があるため、表示上「お子さまによって個人差があります」を強調する（個別計算はしない）
- **1歳を過ぎても離乳が完了しない場合**: STAGE_AFTER 到達後も表Bのパクパク期データを参考値として引き続き参照可能とする旨を案内し、「無理に進める必要はありません」と補足する

## YMYL配慮事項

docs/02 の YMYL 区分「中」（月齢という日付演算＋公的ガイドラインの目安値の転記であり、個別の医学的判断・栄養指導ではない）。docs/03 §4.1「症状・治療・薬は扱わない」の原則に従い、本ツールは以下を徹底する。

- 医学的な適否判定（「この量で足りているか」「体重増加は適切か」等）は一切行わない
- アレルギーの発症リスク判定・除去食の指導は行わない
- 固定文言案（結果画面下部に常時表示）:
  > 「表示している量・固さは、厚生労働省『授乳・離乳の支援ガイド（2019年改定版）』に基づく目安です。赤ちゃんの発育・発達には個人差があります。食べる量や進み方が心配な場合、食物アレルギーが疑われる場合は、自己判断で進めず、乳幼児健診やかかりつけ医・栄養士にご相談ください。」
- 早産児向けの追加文言:
  > 「修正月齢は参考値です。修正月齢をいつまで使うか、離乳食の進め方をどうするかは、お子さまの状態によって個別に異なります。必ず主治医・乳幼児健診の指示に従ってください。」
