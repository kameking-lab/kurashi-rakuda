# 炊飯の水の量・合数⇄g換算

## 概要

「お米○合」「お米○g」「計量カップ○ml」のいずれかを入力すると、炊飯に必要な水の量（ml）と米の重量（g）を相互に換算して表示するツール。米の種類（白米／無洗米／玄米／もち米）による加水量の違いにも対応する。

- **対象**: [docs/02_ツール一覧](../../docs/02_ツール一覧.md) #39「炊飯の水の量・合数⇄g換算」。カテゴリ3「家事・料理」
- **優先度**: S（Phase 1 MVP）／**需要**: 中／**難易度**: B／**YMYL**: 低
- **対象ペルソナ**: ペルソナ1 佐藤美咲（献立を考える負担を減らす。子どもの人数に合わせて炊飯量を変えたい）、ペルソナ3 山本遥（一人暮らしで少量だけ炊きたいが、計量カップしかない/合数が分からない）
- **提供価値**: 「炊飯器の内釜の目盛りを失くした」「レシピはgで書いてあるのに家に計量カップしかない」「玄米や無洗米は普通の白米と水加減が違うと聞いたが具体的にどれくらい？」を即座に解決する
- **前提となる基準値**:
  - 米 1合 = 180ml（体積の定義。日本の伝統的な体積単位「合」）
  - 米 1合 ≈ 150g（精白米の重量目安。精米の密度は水よりやや高く、180mlあたり概ね150g程度になる）
  - 炊飯器メーカーの内釜に印字された目盛り線がある場合は、そちらが最も正確である旨を必ず案内する（本ツールは目盛りがない場合・計量カップやキッチンスケールで代用する場合の目安を提供するもの）

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーション |
|---|---|---|---|
| `input_mode` | 選択式（`合数`/`米の重量(g)`/`計量カップ(ml)`） | 必須 | 3種類のいずれか |
| `value` | 数値 | 必須 | 0より大きい数値。`合数`指定時は0.1〜20（0.5合刻みが一般的だが0.1刻みまで許容）、`g`指定時は1〜3000、`ml`指定時は1〜3600 |
| `rice_type` | 選択式（`白米`/`無洗米`/`玄米`/`もち米`） | 任意（未指定=白米） | 4種類のいずれか |

- `value` が0以下、非数値、または上限超過の場合はエラーメッセージを表示（上限超過時は「家庭用炊飯器の最大容量を超える可能性があります。分けて炊くことをおすすめします」も併記）

## ロジック仕様

```
定数:
  ML_PER_GO = 180     # 1合の体積(ml)
  G_PER_GO  = 150     # 1合の精白米重量(g)目安

function normalizeToGo(value, input_mode):
    switch input_mode:
        case "合数":         return value
        case "米の重量(g)":   return value / G_PER_GO
        case "計量カップ(ml)": return value / ML_PER_GO

function calcRiceWeightAndVolume(go):
    return {
        rice_g:  round(go * G_PER_GO, 0),
        rice_ml: round(go * ML_PER_GO, 0)
    }

function calcWater(go, rice_g, rice_type):
    switch rice_type:
        case "白米":
            # 方式1: 重量ベース（米の重量の1.2倍）
            water_weight_method_ml = round(rice_g * 1.2, 0)
            # 方式2: 合数簡易法（合数×200ml、炊飯器の目盛りに近い簡易目安）
            water_simple_method_ml = round(go * 200, 0)
            return { weight_method: water_weight_method_ml, simple_method: water_simple_method_ml }

        case "無洗米":
            # 白米の簡易法をベースに、大さじ1〜2(15〜30ml)/合を加算
            base = go * 200
            return { min_ml: round(base + go * 15, 0), max_ml: round(base + go * 30, 0) }

        case "玄米":
            # 体積ベースで米の1.5倍の水（浸水30分〜一晩を推奨）
            return { ml: round(go * ML_PER_GO * 1.5, 0) }

        case "もち米":
            # 体積ベースで米とほぼ同量の水（炊飯器で炊く場合。おこわ・赤飯は蒸す場合が多く別途注記）
            return { ml: round(go * ML_PER_GO * 1.0, 0) }

function convert(input_mode, value, rice_type):
    go = normalizeToGo(value, input_mode)
    weight_volume = calcRiceWeightAndVolume(go)
    water = calcWater(go, weight_volume.rice_g, rice_type)
    return {
        go: round(go, 2),
        rice_g: weight_volume.rice_g,
        rice_ml: weight_volume.rice_ml,
        water: water,
        rice_type: rice_type
    }
```

- 白米のみ2方式（重量ベース／合数簡易法）を併記する。両者は「米の重量×1.2」（180g@1合)と「合数×200ml」（200ml@1合）で数値が異なるため、ツール上では両方を提示したうえで「どちらも目安であり、炊飯器の内釜の目盛りがあればそちらを優先してください」と明記する
- 無洗米・玄米・もち米は白米と加水量の考え方（重量ベース vs 体積ベース）が異なるため、それぞれの一般的な算出方法に合わせて別ロジックとする

## 出力仕様

| 表示項目 | 内容 |
|---|---|
| 米の合数 | 入力値から換算した合数（例: 「2合」） |
| 米の重量 | g表示（例: 「300g」） |
| 米の体積 | ml表示（例: 「360ml」） |
| 必要な水の量 | 米の種類に応じた計算結果。白米は「重量ベース目安」「合数簡易法目安」の2値を併記、無洗米は範囲（min〜max）、玄米・もち米は単一値 |
| 炊飯器の目盛りに関する注記 | 「お使いの炊飯器の内釜に目盛り線がある場合は、そちらに従うのが最も正確です」の固定文言 |
| 米の種類ごとの調理メモ | 無洗米＝ぬかがない分やや多めの水／玄米＝浸水30分〜一晩推奨／もち米＝加水控えめ、蒸す場合は別調理法 |
| 出典・前提条件 | 「1合=180ml、精白米1合≈150gという一般的な計量の目安に基づく」の固定注記 |

## エッジケース・注意事項

- **無洗米の加水量差**: 無洗米は精米時にぬか層をあらかじめ除去しているため、家庭で研ぐ工程がなく、通常の白米より水がやや多く必要になる（目安として大さじ1〜2/合を加算）。無洗米専用の計量カップが付属している製品もあるため、その場合はメーカー付属の目盛りを優先する旨を注記する
- **玄米の浸水時間**: 玄米は表皮が硬く吸水に時間がかかるため、加水量だけでなく浸水時間（30分〜一晩）も重要である旨を必ず併記する。浸水なしで炊くと芯が残りやすい
- **もち米の用途差**: もち米は「赤飯・おこわ」として蒸す調理法と、「もち米ご飯」として炊飯器で炊く調理法があり、必要な水分量が大きく異なる。本ツールは炊飯器で炊く場合の目安のみを扱い、蒸す調理法はスコープ外である旨を明記する
- **重量ベースと合数簡易法の数値差**: 白米で「米の重量×1.2」（1合=180ml）と「合数×200ml」（1合=200ml）は約10%の差がある。これは実務でよく使われる2つの経験則であり、どちらか一方が誤りというわけではないため、両論併記し「炊飯器の目盛りが最優先」と案内することでユーザーの混乱を避ける
- **大量炊飯時の上限**: 家庭用炊飯器は5合炊き・1升(10合)炊きが一般的で、それを超える量（例: 15合）を指定した場合は「分けて炊くことをおすすめします」という注意を表示する
- **極小量（0.1〜0.3合等）**: 一人暮らしの少量炊飯にも対応するが、計量誤差が大きくなりやすいため「少量の場合はキッチンスケールでの計量を推奨」という注記を添える
- **ゼロ・負の値・非数値**: バリデーションエラーとして「0より大きい数値を入力してください」を表示し計算しない

## YMYL配慮事項

- YMYL区分は**低**。炊飯の水加減という生活情報であり、健康・医療・金銭的リスクを伴わない
- 食品衛生上の重大リスクは想定しにくいが、「水加減の目安はあくまで一般的な目安であり、米の産地・精米からの経過日数・好みの硬さによって最適な量は変わります」という趣旨の免責的注記を出力欄に固定表示し、断定的な「これが絶対に正解」という表現は避ける
- 玄米・無洗米に関する情報は、精米業界・農林水産省が公開する一般的な調理知識に基づく内容とし、特定商品・特定炊飯器メーカーの宣伝的な訴求は行わない（[00_企画書](../../docs/00_企画書.md) §3.3「広告臭ゼロ」に整合）
