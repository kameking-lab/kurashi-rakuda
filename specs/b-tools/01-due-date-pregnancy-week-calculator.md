# 出産予定日・妊娠週数計算

## 概要

最終月経開始日（LMP）を入力すると、ネーゲレ概算法に基づく出産予定日と、指定した基準日（通常は「今日」）時点の妊娠週数・妊娠月数・妊娠期（初期/中期/後期）を自動計算するツール。

- 対象ペルソナ: ペルソナ4（鈴木奈々・妊活〜プレママ）を中心に、ペルソナ1・2の妊娠中の全員
- 分類: [docs/02_ツール一覧](../../docs/02_ツール一覧.md) カテゴリ1 #1「出産予定日・妊娠週数計算（統合）」。優先度 **S**（Phase 1 MVP）
- YMYL区分: **中**（[docs/03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.1「妊娠・出産の週数/日付計算」）。医学的判断は一切含まず、日付演算のみを行う
- サイト内の位置づけ: `/week/[n]` 妊娠週数別ガイド（0〜42週・43ページ）へのハブ。本ツールの結果画面から該当週のガイド記事へ遷移する動線を持つ（[docs/04_サイト設計](../../docs/04_サイト設計.md) §3.1）

## 入力仕様

| フィールド名 | 型 | 必須/任意 | 説明・バリデーション |
|---|---|---|---|
| `lmp`（最終月経開始日） | 日付（YYYY-MM-DD） | 必須 | カレンダー入力。当日以前の日付のみ有効。基準日より330日以上前の日付はエラー（「入力範囲外」注意書きを表示、下記エッジケース参照） |
| `cycleLength`（平均月経周期日数） | 整数（日） | 任意（デフォルト28） | 20〜45の範囲でのみ有効。範囲外はバリデーションエラーとし、計算を実行しない |
| `baseDate`（基準日） | 日付（YYYY-MM-DD） | 任意（デフォルト＝実行時の当日） | 妊娠週数を計算する基準の日。`lmp` より前の日付はエラー |
| `knownEdd`（医師から伝えられた出産予定日） | 日付（YYYY-MM-DD） | 任意 | 入力された場合はネーゲレ概算法の計算をスキップし、この日付をそのまま出産予定日として週数計算に使う（下記ロジック仕様 4.） |

## ロジック仕様

すべて日付の加減算のみで構成し、医学的な予後判定・診断要素は含めない。

### 1. 出産予定日（EDD）の算出（ネーゲレ概算法）

```
基本式:
  EDD = LMP + 280日        // 280日 = 40週0日 = 28日×10ヶ月

周期日数による補正（周期が28日と異なる場合の一般的な補正）:
  EDD = LMP + 280日 + (cycleLength − 28)日
```

- 280日固定加算がネーゲレ概算法の基本形。実務では平均周期が28日でない場合、差分日数を加減して補正する運用が広く行われている（下記データ表・出典参照）
- `knownEdd` が入力された場合はこの計算をスキップし、`knownEdd` をそのまま EDD として扱う（医師の超音波診断等による訂正値を尊重するため）

### 2. 妊娠週数・日数の算出

```
経過日数 diffDays = baseDate − LMP   （日単位、整数）

妊娠週数 weeks = floor(diffDays / 7)
妊娠日数 days  = diffDays mod 7

表示形式: "{weeks}週{days}日"
  例: diffDays = 90 → 90/7 = 12 余り 6 → 「12週6日」
```

- `knownEdd` が指定されている場合、算出用の LMP を逆算する: `LMP' = knownEdd − 280日` とし、上記と同じ式で `diffDays = baseDate − LMP'` を計算する

### 3. 妊娠月数（いわゆる「妊娠◯ヶ月」表記）

```
if weeks <= 39:
    month = floor(weeks / 4) + 1     // 1〜10ヶ月
else:
    month = null   // 「出産予定日を超えています」表示に切り替え（エッジケース参照）
```

### 4. 妊娠期（初期・中期・後期）— 日本産科婦人科学会の用語定義に準拠

```
if diffDays <= 97:                  // 〜13週6日
    period = "初期"
elif diffDays <= 195:                // 14週0日〜27週6日
    period = "中期"
else:                                 // 28週0日〜
    period = "後期"
```

### 5. 残り日数

```
remainingDays = EDD − baseDate   （負の場合は「出産予定日を経過」として絶対値を表示）
```

## エッジケース・注意事項

| ケース | 扱い |
|---|---|
| `baseDate` < `lmp` | 入力エラー。「基準日は最終月経開始日より後の日付を指定してください」 |
| `lmp` が `baseDate` から330日以上前 | 通常の妊娠期間（最大でも42週=294日程度）を大きく超えるため、「入力範囲外です。日付をご確認ください」と表示し計算を行わない |
| `cycleLength` が20〜45の範囲外 | 入力エラーとして計算をブロック |
| `diffDays` が294日（42週0日）を超える | 「出産予定日を超えています。週数の表示は目安です。次の健診で医師にご確認ください」の注意書きを追加表示（医学的判断はせず、期日超過の事実のみ提示） |
| `weeks` が40以上（EDD到達後） | 「妊娠◯ヶ月」表記は10ヶ月を上限とし、それ以降は月数表示を行わず週数のみ表示 |
| うるう年をまたぐ場合 | 日付演算はカレンダー日数ベース（うるう年の2月29日を含む実日数）で行うため、追加補正は不要。280日固定加算は暦日数として扱う |
| 月末日を LMP に指定（例: 1月31日） | 「280日後」の計算は暦日数の加算であり、月の日数差による繰り上げ・繰り下げ処理は不要（例: 2026年1月31日 + 280日 = 2026年11月7日） |
| 双子・多胎妊娠 | 本ツールは単胎を前提とした一般的な目安であることを注記する（多胎は経過が異なるため個別に医師へ確認、の文言を添える） |
| `knownEdd` と `lmp` を同時入力 | `knownEdd` を優先し、`lmp` は「参考値」として扱う（医師による超音波所見等での訂正を尊重） |
| 未来日の `lmp` | 「最終月経開始日は本日以前の日付を入力してください」のエラー |

## YMYL配慮事項

[docs/03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.1「妊娠・出産の週数/日付計算＝中リスク」の方針に従う。

- 本ツールが行うのは**最終月経開始日を起点とした日付演算のみ**であり、超音波検査や医師の診察に基づく妊娠週数の確定・修正、および出産の可否・健康状態に関する判断は一切行わない
- 結果画面に以下の固定文言を常設表示する（変更不可の定型文としてテンプレートに組み込む）:
  > 「この予定日・週数は最終月経開始日をもとにした計算上の目安です。実際の妊娠週数・出産予定日は、超音波検査などにより医師が診察のうえ判断します。体調に不安がある場合や、この計算結果と健診での説明が異なる場合は、必ず担当医にご確認ください。」
- 出典（日本産科婦人科学会）と最終更新日・準拠年度をページ下部に常時表示（[docs/03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.2「全ツールに最終更新日／準拠年度／出典の3点セットを強制表示」）
- 計算式をページ内に明示し、ブラックボックス化しない（同§4.1「計算式そのものをページ内に明示」原則）
- 広告・商品紹介・アフィリエイトリンクは本ツール結果画面に一切配置しない（[docs/00_企画書](../../docs/00_企画書.md) §3.2）
