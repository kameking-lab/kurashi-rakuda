# 手取りからの予算配分計算（費目テンプレ）（P2-T29）

## 概要

世帯人員数（1人・2人・3人・4人・5人・6人以上）と手取り月収を入力すると、総務省統計局「家計調査（家計収支編）」の世帯人員別・費目別の支出構成比（全国平均）に基づいて、10大費目（食料・住居・光熱水道・家具家事用品・被服及び履物・保健医療・交通通信・教育・教養娯楽・その他の消費支出）にそれぞれどれくらい配分するのが平均的かの目安金額を表示するツール。

- docs/02 **#62**（カテゴリ4: お金、優先度 中、難易度B）。BACKLOG.md queueId: `P2-T29`。
- データ: `data/tables/yosan-haibun-hiritsu.json`（総務省統計局「家計調査（家計収支編）」2025年（令和7年）平均、2026年2月6日公表）を単一の情報源（SSOT）とする。金額・構成比はすべて同JSONの実額から計算し、本ツール・本仕様書に直接ハードコードしない（可読性のため転記している数値は「データ表・出典」節を参照）。世帯人員区分は `data/tables/shokuhi-meyasu.json`（P2-T28・食費の目安計算）と完全に同じ区分・同じ出典テーブルであり、`食料`・`消費支出`の実額はP2-T28と同一の値である。
- YMYL区分: 中（お金に関わる目安。個別の家計診断・節約指導は行わない）。表示する金額はあくまで統計上の平均的な配分比率を機械的に当てはめた目安であり、実際に望ましい配分は世帯ごとに大きく異なる旨を常時表示する。

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| 世帯人員数（setaininzuu） | 整数 | 必須 | 1以上の整数であること。小数・0以下・非数値は入力エラー。UI上の入力欄は1〜10の範囲を想定するが、ロジック自体は1以上の任意の整数を受け付ける（6以上はすべて同じ区分にまとめる） |
| 手取り月収（tedoriGesshuu） | 数値（円） | 必須 | 0より大きい有限数であること。0以下・非数値は入力エラー。整数であることは要求しない（小数入力があっても計算は通すが、最終的な金額表示は円未満を四捨五入する） |

世帯構成（単身/夫婦のみ/夫婦+子どもなど）は入力項目としない。理由はP2-T28と同じ（総務省統計局の当該統計表は世帯人員のみで区分されており、世帯構成別の実額データは公表されていないため、補間・按分による推測値の表示を避ける）。

## ロジック仕様

### 1. 世帯人員区分の判定（バケット化）

P2-T28（`ShokuhiMeyasu.calc.ts`）と全く同じ判定ロジックを用いる。

```
if setaininzuu が整数でない、または 1未満:
    → 入力エラー
elif setaininzuu == 1:
    bucket = tanshin_setai（単身世帯）
elif setaininzuu == 2:
    bucket = futari_ijou_setai.ninin（2人世帯）
elif setaininzuu == 3:
    bucket = futari_ijou_setai.sannin（3人世帯）
elif setaininzuu == 4:
    bucket = futari_ijou_setai.yonin（4人世帯）
elif setaininzuu == 5:
    bucket = futari_ijou_setai.gonin（5人世帯）
else:  # setaininzuu >= 6
    bucket = futari_ijou_setai.rokunin_ijou（6人以上世帯。実際の平均世帯人員は6.3人）
```

入力した世帯人員数が6を超える場合（例: 7人、10人）も、すべて「6人以上」区分の同じ構成比を返す（総務省統計局の集計自体が「6人以上」までしか区分していないため、7人・10人の按分値を作らない）。

### 2. 手取り月収のバリデーション

```
if tedoriGesshuu が有限数でない:
    → 入力エラー
elif tedoriGesshuu <= 0:
    → 入力エラー
```

### 3. 費目別構成比の算出

各世帯人員区分のバケットには、10大費目の実額（円）と消費支出合計（円）が格納されている。費目 `i` の構成比は次の式で算出する。

```
ratio(i) = bucket[i] / bucket.shouhi_shishutsu
```

10費目は次の順に固定する（家計調査の費目番号1〜10の順）。

1. `shokuryou`（食料）
2. `juukyo`（住居）
3. `kounetsu_suidou`（光熱・水道）
4. `kagu_kaji_youhin`（家具・家事用品）
5. `hifuku_hakimono`（被服及び履物）
6. `hoken_iryou`（保健医療）
7. `koutsuu_tsuushin`（交通・通信）
8. `kyouiku`（教育）
9. `kyouyou_goraku`（教養娯楽）
10. `sonota_shouhi_shishutsu`（その他の消費支出）

### 4. 目安金額の算出

```
amount(i) = round(tedoriGesshuu × ratio(i))
```

各費目ごとに独立して四捨五入するため、10費目の `amount` の合計は `tedoriGesshuu` と厳密には一致しない場合がある（統計上の丸め誤差と同種の、独立丸めによる数円程度のズレ。合計を強制的に一致させる補正〈端数を最後の費目に押し付ける等〉は行わない。存在しない精度の演出を避けるという方針に基づく）。

### 5. 入力値と実際に使われた区分が異なる場合の注記

`setaininzuu >= 6` かつ `setaininzuu !== 6.3`（常に真。データの6人以上区分は平均6.3人のため）の場合、「入力した人数（例: 8人）そのものの統計はなく、『6人以上』区分（平均世帯人員6.3人）の構成比を使っている」旨をUIに明示する。1〜5人の場合はこの注記は不要（統計上の区分とちょうど一致するため）。

## エッジケース

- **世帯人員6人を超える入力（例: 8人、20人）**: 総務省統計局の統計表自体が「6人以上」までしか区分していない。7人・10人などに応じた按分値を作らず、すべて「6人以上」区分（構成比は6人〜6.3人相当の平均値）をそのまま使い、入力した人数と統計区分の人数が一致しない旨を明記する。
- **世帯構成別（単身/夫婦のみ/夫婦+子どもなど）の入力は提供しない**: P2-T28と同じ理由（総務省統計局「家計調査（家計収支編）」の世帯人員別集計は、世帯人員数のみで区分されており、同じ人数でも世帯構成による実額の違いは同統計からは取得できない）。
- **0人・負の数・小数の世帯人員数**: 入力エラーとして扱う。
- **0円・負の手取り月収・非数値**: 入力エラーとして扱う（「手取り月収は1円以上の金額で入力してください」等）。
- **消費支出（構成比の分母）と手取り月収（可処分所得）は異なる概念であること**: 家計調査の「消費支出」は実際に使われた生活費の平均額であり、貯蓄に回した分は含まれない。本ツールは「消費支出の費目別構成比」を、入力された手取り月収にそのまま当てはめて按分するという簡易な計算方法を採用しており、構成比の元になった世帯の実際の貯蓄率・可処分所得水準（例えば単身世帯の消費支出173,042円が、その世帯の手取り収入の何%に当たるか）とは無関係に按分する。この限界は仕様書・ツールのCalloutの双方に明記する。
- **費目別金額の合計が手取り月収と厳密には一致しない**: 各費目を独立して四捨五入するため、10費目の合計が入力額と数円程度ズレることがある。強制的な補正は行わない。
- **住居費が世帯人員によって単調増加しない**: 家計調査の実額を見ると「住居」は世帯人員が増えるほど持家率が上がる傾向があり、必ずしも人数に比例して増加しない（むしろ5人世帯で最も低い11,968円）。これは統計上の実態であり、按分ロジックの誤りではないため補正しない。
- **物価変動・調査年の違い**: 表示する構成比は2025年（令和7年）平均の実額に基づくものであり、翌年以降の物価変動・消費行動の変化を反映していない。次回の「◯◯年平均」公表時にデータを更新する（`next_check_due` 参照）。

## YMYL配慮事項

- 結果画面には常時、次のCalloutを表示する: 「この金額は総務省統計局『家計調査』の世帯人員別・費目別の支出構成比（全国平均）を、入力した手取り月収にそのまま当てはめた目安です。実際の家計調査の『消費支出』は貯蓄に回した分を含まない実際の支出額であり、あなたの手取り収入や貯蓄の状況とは無関係に計算しています。地域・生活スタイル・住居費（持家か賃貸か）などによって、実際に望ましい配分は大きく異なります。『この通りに配分すべき』という指導ではありません。」
- 節約・貯蓄・家計改善を指導・断定する表現（「◯円以内に抑えましょう」「この配分を目指しましょう」等）は一切行わない。あくまで全国平均の統計上の目安を「知る」ためのツールとして位置づける。
- 準拠年（2025年（令和7年）平均）・出典（総務省統計局）・最終確認日（2026-07-18）を画面内に表示する。
- 「消費支出」と「手取り収入（可処分所得）」が異なる概念であることを明示し、按分の前提（構成比をそのまま手取り額に適用する簡易計算であること）を隠さない。
