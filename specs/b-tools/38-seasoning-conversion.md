# 調味料換算

## 概要

調味料を選び、「大さじ／小さじ／カップ」「g（重量）」「ml（体積）」のいずれかで数量を入力すると、他の単位への換算結果を即座に表示するツール。調味料ごとに比重（同じ体積でも重さが異なる）が違うため、単純な体積⇄重量の一律換算ではなく、調味料別の実測値ベースの換算表を用いる。

- **対象**: [docs/02_ツール一覧](../../docs/02_ツール一覧.md) #38「調味料換算（大さじ⇄g⇄ml、食材別比重）」。カテゴリ3「家事・料理」
- **優先度**: S（Phase 1 MVP）／**需要**: 中／**難易度**: B／**YMYL**: 低
- **対象ペルソナ**: ペルソナ1 佐藤美咲（献立を考えるのが苦痛。レシピの単位表記がバラバラで戸惑う）、ペルソナ3 山本遥（一人暮らしで少量ずつ計量する機会が多い）
- **提供価値**: 「レシピは『しょうゆ大さじ1』なのに、うちの計量スプーンはgでしか測れない／逆にレシピがgで自宅は大さじしかない」を解消する。会員登録不要、入力してすぐ答えが出る
- **前提**: 計量単位の基準は日本の標準計量スプーン・カップ（大さじ1=15ml、小さじ1=5ml、カップ1=200ml）とする。調味料ごとの重量目安は、女子栄養大学・香川調理科学研究所の計量スプーン・カップ重量表に代表される、日本の家庭料理で広く使われている一般的な目安値を採用する

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーション |
|---|---|---|---|
| `seasoning_id` | 選択式（データ表の `id`） | 必須 | データ表に存在する調味料のみ選択可（プルダウン or 検索候補） |
| `unit_from` | 選択式（`大さじ`/`小さじ`/`カップ`/`g`/`ml`） | 必須 | 上記5種類のいずれか |
| `amount` | 数値 | 必須 | 0より大きい数値。上限1000（大さじ・小さじ・カップ換算時は換算後gが極端にならないよう目安として上限を設ける）。小数第2位まで許容（例: 1.5杯） |
| `unit_to` | 選択式（`大さじ`/`小さじ`/`カップ`/`g`/`ml`／未指定=全単位を一括表示） | 任意 | 未指定の場合、残り4単位すべてを結果一覧に表示する（デフォルト動作） |

- `amount` が0以下、または非数値の場合はエラーメッセージ「0より大きい数値を入力してください」を表示し計算しない
- `unit_from` と `unit_to` が同一の場合は変換不要としてそのまま `amount` を返す

## ロジック仕様

調味料ごとに「大さじ1あたりの重量（g）」を基準値として保持し、そこから比重（g/ml）を算出して全単位間の相互変換を行う。

```
定数:
  ML_PER_OOSAJI = 15
  ML_PER_KOSAJI = 5
  ML_PER_CUP = 200

function getDensity(seasoning):
    # 比重(g/ml) = 大さじ1の重量(g) ÷ 大さじ1の体積(15ml)
    return seasoning.g_per_oosaji / ML_PER_OOSAJI

function toMl(amount, unit_from):
    switch unit_from:
        case "大さじ": return amount * ML_PER_OOSAJI
        case "小さじ": return amount * ML_PER_KOSAJI
        case "カップ": return amount * ML_PER_CUP
        case "ml":    return amount
        case "g":     return null  # g→mlは比重を使うため別処理

function convert(seasoning_id, amount, unit_from, unit_to):
    seasoning = DATA.find(d => d.id == seasoning_id)
    density = getDensity(seasoning)  # g/ml

    # ステップ1: 入力を「重量g」と「体積ml」の両方に正規化
    if unit_from == "g":
        grams = amount
        ml = amount / density
    else:
        ml = toMl(amount, unit_from)
        grams = ml * density

    # ステップ2: 目的単位に変換
    results = {}
    results["g"] = round(grams, 1)
    results["ml"] = round(ml, 1)
    results["大さじ"] = round(ml / ML_PER_OOSAJI, 2)
    results["小さじ"] = round(ml / ML_PER_KOSAJI, 2)
    results["カップ"] = round(ml / ML_PER_CUP, 2)

    if unit_to is not null:
        return results[unit_to]
    else:
        return results  # 全単位一覧を返す
```

- 丸め方針: g・mlは小数第1位まで、大さじ・小さじ・カップは小数第2位まで表示（例: 「0.33カップ」）。実用上は分数表記（例:「大さじ1/3」）の方が読みやすいため、表示層では0.33→「約1/3」、0.5→「1/2」、0.25→「1/4」等の代表分数に丸めて併記する
- 比重は「大さじ1の重量」を基準に算出するため、`g_per_oosaji` の値を唯一のマスタ値としてデータ表で管理する（小さじ・カップの重量はこの比重から自動計算され、個別に持たない）

## エッジケース・注意事項

- **同じ調味料でも銘柄・状態でばらつきがある**: みそは種類（米みそ／麦みそ／豆みそ、粒/こし）によりg/mlが多少変動する。しょうゆも濃口/薄口で若干差がある。本ツールの値はあくまで「一般的な目安」であり、正確な計量が必要な製菓等の用途では商品パッケージの表示を優先するよう注記する
- **粉状調味料は「すりきり」前提**: 小麦粉・片栗粉・パン粉は、計量スプーンに山盛りにすると重量が変わる。本データは「すりきり（表面を水平にならす）」での計量を前提とする旨を明記する
- **塩は精製塩と天然塩で比重が異なる**: 精製塩（サラサラ・粒が細かい）は大さじ1=18gだが、あら塩・天然塩（粒が粗い・湿り気がある）は大さじ1=15g程度と軽くなる。ユーザーが誤った塩の種類を選ばないよう、選択肢を明確に分ける
- **極端な入力値**: 1000大さじ（=15L相当）等、現実的な調理量を超える入力は上限バリデーションで弾く
- **g→他単位への変換で比重0近い調味料**: 本データ表の範囲では比重0のデータは存在しないため、ゼロ除算は発生しない設計（`g_per_oosaji` は必ず正の値としてデータ検証する）
- **液体と粉体の直感的な違い**: 「同じ大さじ1でも、水は15gなのに小麦粉は9gしかない」という結果に驚くユーザーがいるため、比重の参考値を必ず併記して納得感を持たせる

## YMYL配慮事項

- YMYL区分は**低**。調理の分量換算であり、健康・医療・金銭リスクを伴わない生活情報のため、出典明記以外の特別な配慮は不要
- ただし食品衛生・栄養に関する断定（「この塩分量なら健康に問題ない」等）は一切行わない。本ツールは単位換算のみを提供し、栄養価・塩分量の評価は行わないスコープとする（[03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.1 の「症状・治療・薬は扱わない」方針を踏襲し、調味料の摂取量に関する健康助言はしない）
- 数値は一般に公開されている計量換算の目安値のみを用い、特定メーカー・特定商品の広告的な数値は使用しない（[00_企画書](../../docs/00_企画書.md) §3.3「広告臭ゼロ」に整合）
