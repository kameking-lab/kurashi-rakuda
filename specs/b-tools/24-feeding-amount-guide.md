# 授乳・ミルク量の目安（月齢・体重別）

## 概要

赤ちゃんの月齢・体重・栄養方法（母乳／ミルク／混合）から、1日あたりの授乳・ミルク量とその回数の一般的な目安を表示するツール。

- 対象ペルソナ: ペルソナ1（佐藤美咲・下の子1歳の離乳食・授乳量が不安）を主対象。
- docs/02_ツール一覧.md **#24**（カテゴリ2: 子育て、優先度 **S**、難易度 B）。「ツール形式ほぼ皆無」の空白領域。
- YMYL区分: **中**（乳児の栄養に関わるため、個人差が大きいことと医療判断ではないことを強く明記する）。

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| 月齢（ageMonths） | 整数（0〜11） | 必須 | 0〜11の整数のみ受け付ける。0は「生後0〜1か月（新生児期）」を表す。12以上は対象外としてエラーではなく案内メッセージを表示 |
| 体重（weightKg） | 数値（kg、小数第1位まで） | 必須（月齢0の計算式に使用するため） | 1.0〜20.0の範囲。新生児の一般的な体重（目安 2.0〜5.0kg）から外れる値は警告表示（エラーにはしない。低出生体重児等を排除しないため） |
| 栄養方法（feedingType） | 選択（母乳のみ／ミルクのみ／混合） | 必須 | いずれかを選択。未選択はエラー |

## ロジック仕様

ブラックボックス化を避けるため、計算式・参照表の値をそのまま結果画面に表示する。

### 1. 月齢0（生後0〜1か月・新生児期）: 体重ベースの計算式

```
1日哺乳量目安(ml) = 体重(kg) × 150
1日の回数目安 = 8回
1回量目安(ml) = 1日哺乳量目安 ÷ 1日の回数目安
```

（体重1kgあたり120〜160ml/日で紹介されることが多く、本ツールは目安として150を採用する。実際は個人差があるため出力時にレンジも併記する: `体重(kg) × 120 〜 体重(kg) × 160`）

### 2. 月齢1〜11: 月齢帯ごとの参照表を使用（下記データ表）

```
if 1 <= 月齢 <= 11:
    表の該当行（月齢帯）から「1回量目安レンジ」「回数目安」を取得
    1日量目安レンジ = 1回量目安レンジ × 回数目安（下限×下限、上限×上限で概算）
```

### 3. 栄養方法による出力分岐

```
if 栄養方法 == "母乳のみ":
    ml換算の目標値は表示しない（母乳は哺乳量を正確に測定できないため）
    代わりに「回数目安」と「欲しがるサインに応じた授乳」の考え方を表示
elif 栄養方法 == "ミルクのみ":
    上記1・2で算出したml目安をそのまま表示
elif 栄養方法 == "混合":
    ml目安を「ミルクを足す場合の合計目安」として表示し、
    「母乳を飲んだ分はミルクの量から差し引いて調整する」という考え方を注記
```

### 4. 対象月齢を超える場合（月齢12以上）

```
if 月齢 >= 12:
    数値計算は行わず、案内メッセージのみ表示:
    「生後12か月以降は離乳食が中心になる時期です。ミルク量の目安ツールの対象外です。離乳食の量・固さ早見（#19）をご覧ください」
```

## データ表

| 表示項目 | 内容 |

| 1回量目安 | 「◯◯ml前後」 |
| 回数目安 | 「1日◯回程度」 |
| 母乳のみの場合の表示 | ml目標は出さず「欲しがるサインに応じて授乳（1日8回程度が目安）」という趣旨の文章を表示 |

## エッジケース・注意事項

- **月齢0×極端な体重（低出生体重児等）**: 計算自体は行うが、「低出生体重児や医療的なフォローが必要なお子さまは、医師・助産師・保健師の指示を優先してください」と注記する。エラーにはしない（排除しない設計）。
- **体重が新生児として非現実的な値（極端に大きい/小さい）**: ソフトバリデーションとして注意表示のみ行い、計算はブロックしない。
- **母乳のみでml目標を求められた場合**: あえてml換算値を出さない設計とし、「母乳の量は目分量では測れないため、体重の増え方や機嫌・おしっこの回数で判断するのが一般的です」という趣旨を表示する。
- **混合栄養の場合の按分**: 母乳を飲んだ量を正確に把握できないため、厳密な按分計算はせず「ミルクは様子を見ながら調整する」という考え方に留める（過剰な精緻化をしない）。
- **月齢12以上の入力**: エラーではなく離乳食ツール（#19）への案内に誘導する（スコープ外を明示）。
- **双子・多胎児**: 本ツールは1人分の計算。各子の体重が異なる場合は個別に入力する。
- **体重増加不良等の懸念がある場合**: 本ツールは体重増加の適否判定は行わない（それは成長曲線・パーセンタイル評価の領域であり#25や乳幼児健診の役割）。

## YMYL配慮事項

- 乳児の栄養量に関する情報のため、断定的な指示（「◯◯mlあげてください」等の命令形）は避け、「◯◯ml前後が目安です」という表現に統一する。
- 固定文言案（結果画面下部・常時表示）: 「これは一般的な目安であり、個人差があります。赤ちゃんの機嫌・体重の増え方・おしっこやうんちの様子を見ながら調整してください。心配な場合や体重の増えが気になる場合は、医師・助産師・保健師にご相談ください。」
- 母乳育児について「母乳が足りているか」の医学的判定は行わないことを明記する固定文言案: 「母乳の量そのものは正確に測れません。心配な場合は乳幼児健診や母乳外来でご相談ください。」
- 最終更新日／準拠年度／出典（一般的な育児用ミルク製品の目安表・こども家庭庁「授乳・離乳の支援ガイド」の考え方を参考にしている旨）の3点セットを表示する。
