# 洗濯表示検索

## 概要

衣類タグの洗濯表示記号（新JIS記号）から、その記号の意味・取り扱い方法を検索できるツール。記号の見た目（形状・数字・線の本数）やキーワード（例:「手洗い」「アイロン不可」「漂白剤」）から該当する記号を絞り込み、意味と具体的な取り扱い方を平易文で表示する。

- **対象**: [docs/02_ツール一覧](../../docs/02_ツール一覧.md) #41「洗濯表示検索（新JIS記号、写真対応は将来）」。カテゴリ3「家事・料理」
- **優先度**: S（Phase 1 MVP）／**需要**: 高／**難易度**: B／**YMYL**: 低
- **対象ペルソナ**: ペルソナ1 佐藤美咲（子ども服・デリケートな衣類の洗濯で失敗したくない）、ペルソナ2 田中絵里（時間がない中で素早く正しい取り扱いを知りたい）
- **提供価値**: 「このマーク、見たことあるけど意味が分からない」を検索1回・登録不要で即答する。将来的にカメラで記号を撮影して自動判定する機能（画像認識）を検討するが、**Phase 1では写真対応はスコープ外**とし、テキスト検索・カテゴリ一覧からの選択のみを実装する
- **準拠規格**: 家庭用品品質表示法に基づく新JIS表示（JIS L 0001）。**2016年12月1日改正・施行**。`factory/data/seido/sentaku-hyouji.json` の `shiko_nengetsu: "2016-12"` と整合させ、記号総数は同ファイルの `kigou_sousuu: 41`（全41記号）と一致させる
- **将来拡張**: 本仕様のデータ表は5カテゴリの代表記号（計23種）のサンプル。docs/02 の計画どおり最終的に新JIS全41記号を整備する（`factory/data/seido/sentaku-hyouji.json` の `$comment` に明記されている「#41 実装時に packages/data 側で41記号ぶん整備する」に対応する初期セット）
- **出典**: 消費者庁「洗濯表示（衣類等の取扱い表示）」案内（`factory/data/seido/sentaku-hyouji.json` と同一出典）

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーション |
|---|---|---|---|
| `query` | 文字列（フリーテキスト） | 任意 | 1〜30文字。「手洗い」「40度」「アイロン不可」「ドライクリーニング」等、記号の意味・見た目の特徴語での検索を許容 |
| `category` | 選択式（`洗濯`/`漂白`/`乾燥`/`アイロン`/`クリーニング`のいずれか、未選択＝全件） | 任意（絞り込みフィルタ） | データ表の `category` 列挙値のみ |
| `symbol_id` | 文字列（記号マスタの`id`） | 任意（`query`と排他的だが両方指定時は`symbol_id`優先） | データ表の`id`に存在するもののみ許可 |

- 記号の「見た目」からの絞り込みを助けるため、カテゴリ選択画面では各カテゴリの記号の基本形（桶／三角／四角／アイロン／丸）をアイコンとテキストで併記した一覧をまず表示し、そこから該当しそうな記号を選ばせるUXとする（フリーテキスト検索が苦手なユーザーへの導線）
- **写真アップロードによる記号自動判定は本仕様のスコープ外**（[00_企画書](../../docs/00_企画書.md)・docs/02 の「写真対応は将来」に準拠。将来のPhaseで別仕様として検討する）

## ロジック仕様

計算ロジックは存在しない（固定データの検索・表示のみ）。疑似コードは以下。

```
function search(query, category, symbol_id):
    if symbol_id is not null:
        item = DATA.find(d => d.id == symbol_id)
        if item is null:
            return NOT_FOUND
        return renderDetail(item)

    results = DATA
    if category is not null:
        results = results.filter(d => d.category == category)
    if query is not null and query != "":
        normalized = normalize(query)  # ひらがな化・全角半角統一・空白除去
        results = results.filter(d =>
            normalize(d.meaning).includes(normalized) OR
            d.keywords.any(k => normalize(k).includes(normalized))
        )

    if results.length == 1:
        return renderDetail(results[0])
    else if results.length == 0:
        return NOT_FOUND
    else:
        return renderList(results)  # 候補一覧（記号名・カテゴリ・意味の要約）

function renderDetail(item):
    return {
        category: item.category,
        symbol_shape: item.symbol_shape,     # 見た目の説明文（例:「洗濯桶の中に『40』、下に線1本」）
        meaning: item.meaning,               # 正式な意味
        handling_tip: item.handling_tip,     # 実際の取り扱い方の平易な言い換え
        related_symbols: item.related_symbols, # 見間違えやすい・対になる記号への内部リンク
        source: "消費者庁（家庭用品品質表示法・JIS L 0001 新JIS表示）",
        effective_date: "2016年12月1日改正・施行"
    }
```

- 「洗濯桶に40、下線1本」のように、記号は「基本形（桶/三角/四角/アイロン/丸）」＋「数字（温度上限）」＋「線の本数（処理の強さ）」の組み合わせで構成されるため、データ表の `symbol_shape` は視覚的特徴を文章化して保持する（実際のSVGアイコン表示は別途フロントエンド実装で対応し、本仕様ではデータ構造とテキスト説明のみを定義する）

## エッジケース・注意事項

- **旧JIS表示（2016年11月以前）との混同**: 2016年11月30日以前に購入した衣類には旧JIS表示（22記号）が付いていることがあり、見た目が似ていても意味が異なる記号がある（例: 旧表示の水温は「洗濯機マーク＋液温の数字表記」が異なる形式）。本ツールは**新JIS表示（2016年12月以降）のみに対応**し、「お手持ちの衣類が古い場合は表示が異なることがあります」という注記を検索結果に固定表示する。旧JIS対応は将来検討事項としてスコープ外に明記する
- **記号の組み合わせ（タグ1枚に複数記号）**: 実際の衣類タグには洗濯・漂白・乾燥・アイロン・クリーニングの5記号が並んで表示されるのが一般的。本ツールは記号1つずつの意味を検索する設計とし、タグ全体の一括読み取り（写真対応）は将来機能とする
- **デリケート素材の扱い**: 「非常に弱い」処理指定やドライクリーニングのみ可の表示がある衣類（シルク・ウール・スーツ等）は、家庭洗濯を試みると型崩れ・縮み・色落ちのリスクが高いため、「表示に忠実に、専門のクリーニング店へ相談することを推奨します」という注記を該当記号（`dry_*`, `wet_*`, `sen_tearai`）の詳細表示に付与する
- **記号が読み取れない・タグが切れている場合**: データ検索の対象外となるため、「表示が読み取れない場合は、無理に家庭で洗濯せず、クリーニング店に相談することをおすすめします」という一般的な注意喚起を用意する
- **温度表記の誤読**: 「40」と「140」など、桁数を見誤りやすい表示は見間違えやすい記号として `related_symbols` にまとめ、並べて比較できるようにする
- **未登録記号**: 41記号への拡張までは一部記号が未登録。0件時は「まだ登録がありません。今後追加予定です」を表示し、フィードバック導線（[03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §6）へ誘導する

## YMYL配慮事項

- YMYL区分は**低**（生活情報）だが、誤った取り扱いにより衣類の損傷（縮み・色落ち・型崩れ）という実害につながりうるため、以下を必須表示とする
  - 「記号の意味は一般的な目安です。実際の取り扱いは、衣類ごとのタグ表示を優先してください」の固定注記
  - デリケート素材・クリーニング専用表示については「不安な場合は自己判断で洗わず、クリーニング店に相談してください」という安全側の案内を明記し、断定的に「家庭で洗っても大丈夫」という表現は避ける
- 出典は消費者庁の公式案内（家庭用品品質表示法・JIS L 0001）に限定し、準拠年度（2016年12月改正・施行）を全記号詳細ページに固定表示する（[03_コンテンツ戦略](../../docs/03_コンテンツ戦略.md) §4.2「全ツールに最終更新日／準拠年度／出典の3点セットを表示」に整合）
