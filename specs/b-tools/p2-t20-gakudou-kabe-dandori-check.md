# 学童・小1の壁 段取りチェック（P2-T20）

## 概要

保育園・幼稚園から小学校＋学童保育（放課後児童クラブ）への移行に伴う「小1の壁」に備えるための、段取りチェックリスト・ナビゲーションツール。入学（予定）年度を入れると、今が「入学前年の秋頃」「入学直前（年明け〜3月）」「入学後」のどの時期に当たるかを判定し、時期別にやることチェックリストを表示する。

- docs/02_ツール一覧.md **#28**（カテゴリ2: 子育て、優先度 中、難易度B、YMYL 低）。BACKLOG.md queueId `P2-T20`。
- 依存データ: `data/seido/gakudou-hoiku-kijun.json`（放課後児童健全育成事業の設備及び運営に関する基準。既存・✔）。
- YMYL区分: **低〜中**。自治体差を煽らず、正直に「全国一律の開所時間・締切日は存在しない」ことを伝える。

★本ツールの最重要制約★: `data/seido/gakudou-hoiku-kijun.json` の `data.overview.standardsNature.note` にあるとおり、放課後児童健全育成事業の基準（開所時間・職員配置・面積等）はすべて市町村が条例を定めるに当たっての「参酌基準」であり、全国一律の確定値ではない。実際の開所時間・お迎え締切時刻・申込締切日は市町村・クラブごとに条例・運営規程で決まるため、本ツールは**架空の「◯時まで」「◯月◯日締切」という具体的な時刻・日付を一切生成しない**。日付演算の対象は「入学（予定）年度」という粒度の粗い時期区分のみである。

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| 入学（予定）年度（entryFiscalYear） | 整数（西暦・4桁） | 必須 | 1990〜2100の範囲の整数であること。まだ入学していない場合は入学予定の4月を含む年度、既に入学している場合は入学した年度（＝現在の学年から逆算可能な値）を入力する。範囲外・非整数は入力エラー |

「入学予定年」と「現在の学年」のどちらから考えても同じ値になる（例: 今日が2026年度で、今年の4月に入学したなら2026、来年度に入学予定なら2027、3年前に入学したなら2023）ため、単一のフィールドで両方の入力ニーズを満たす。複雑な日付逆算（申込締切からの逆算等）は行わない。

## ロジック仕様

### 1. 年度・学年オフセットの算出

```
fiscalYearOf(dateIso) = 月が4以上ならその年、そうでなければ前年　（4月始まりの年度）
todayFiscalYear = fiscalYearOf(today)
gradeOffset = todayFiscalYear - entryFiscalYear
```

- `gradeOffset < 0`: まだ入学していない（就学前）
- `0 <= gradeOffset <= 5`: 入学済み。学年 = `gradeOffset + 1`（小1〜小6）
- `gradeOffset >= 6`: 対象学年（小学生）を超えている（中学生以降。異常値ではなく「対象範囲外」として扱う）

### 2. 時期区分（就学前のみ）の判定

```
prepAutumnStart = "{entryFiscalYear-1}-09-01"   # 入学前年の秋頃の開始目安
justBeforeStart = "{entryFiscalYear}-01-01"      # 入学直前（年明け）の開始
entryDate        = "{entryFiscalYear}-04-01"      # 入学（4月）

gradeOffset >= 6            → phase = "outOfRange"
gradeOffset >= 0            → phase = "afterEntry"（入学後。学年 = gradeOffset+1）
today < prepAutumnStart     → phase = "tooEarly"（入学まで時間がある。情報収集期）
today < justBeforeStart     → phase = "prepAutumn"（入学前年の秋頃）
それ以外（today < entryDate） → phase = "justBefore"（入学直前）
```

ISO形式（YYYY-MM-DD）の文字列同士は辞書式比較がそのまま日付の前後判定になるため、`Date` オブジェクトへの変換は行わずそのまま文字列比較する。

### 3. チェックリストの提示

`phase` が `tooEarly` / `prepAutumn` / `justBefore` / `afterEntry` のいずれの場合も、3つの時期別チェックリスト（入学前年の秋頃／入学直前／入学後）をすべて表示し、現在の `phase` に対応するチェックリストだけを「いまの時期」として強調する（`isCurrent` フラグ）。`tooEarly` の場合はどのチェックリストも強調しない（まだ先であることを示す注記のみ表示）。`outOfRange` の場合はチェックリストを表示せず、対象学年外である旨のみ表示する。

### 4. 参考情報（制度上の事実）の提示

以下はすべて `data/seido/gakudou-hoiku-kijun.json` から取得し、断定的な「全国一律の値」としてではなく「国の参酌基準・全国統計」として提示する。

- 対象学年（小学1年生から小学6年生まで）
- 支援の単位の定員（おおむね40人以下。参酌基準）
- 開所時間の参酌基準（休業日は1日8時間以上、それ以外の日は1日3時間以上、年間250日以上を原則とする参酌基準。**実際の時刻・開所日は自治体・クラブごとに異なる**旨を必ず併記）
- 待機児童数（全国16,330人、学年別内訳。小学4年生が最多〈5,589人〉＝「小4の壁」の実態）

## エッジケース・注意事項

- **就学前（gradeOffset < 0）**: 時期区分（`tooEarly`／`prepAutumn`／`justBefore`）を日付ベースで判定し、対応するチェックリストを強調する。
- **入学年度＝今年度（gradeOffset = 0）**: 小学1年生・`phase = "afterEntry"`。
- **入学6年前（gradeOffset = 5）**: 小学6年生・`phase = "afterEntry"`。境界値として重要（`gradeOffset` が5から6に変わる境目）。
- **入学7年以上前（gradeOffset >= 6）**: `phase = "outOfRange"`。エラー（入力ミス）としてではなく、「学童保育の対象学年（小学生）を超えています」という情報として扱う。
- **時期区分の境界値**: `today` が `prepAutumnStart`（entryFiscalYear-1年9月1日）ちょうどの場合は `prepAutumn`（`tooEarly` ではない）。`today` が `justBeforeStart`（entryFiscalYear年1月1日）ちょうどの場合は `justBefore`。`today` が `entryDate`（entryFiscalYear年4月1日）ちょうどの場合は `gradeOffset = 0` となり `afterEntry`（`justBefore` ではない）。
- **入力範囲外（1990年未満・2100年超・非整数）**: 入力エラーとして結果を表示しない。
- **極端に先の入学予定年度（`gradeOffset` が -10 未満、すなわち入学の10年以上前）**: エラーにはしないが、入力ミスの可能性を示すソフト警告を表示する。
- **うるう年をまたぐ判定**: 比較する日付は年月日文字列（YYYY-MM-DD）の辞書式比較のみで行っており、うるう年による特別な補正は不要（2月29日を含む・含まないに関わらず文字列比較の大小関係は変わらない）。
- **開所時間・締切時刻・申込締切日を尋ねない**: 本ツールはこれらの自治体差が大きい具体的な時刻・日付を入力にも出力にも一切含めない。

## ★データの限界（全国一律の開所時間・締切日が存在しない理由）★

`data/seido/gakudou-hoiku-kijun.json` の `amendments[1].impact` に記録されているとおり、令和2年（2020年）4月1日施行の第9次地方分権一括法により、放課後児童支援員の員数・資格を含む本基準の全ての項目が「従うべき基準」から「参酌すべき基準」に変更された。「参酌すべき基準」とは、市町村が条例を定める際に**参考にすべき目安**であって、国が直接強制する最低基準ではない。そのため:

1. **開所時間・お迎え締切時刻**は、省令が「1日8時間以上（休業日）」「1日3時間以上（平日）」という時間数の目安を示すのみで、具体的な開所・閉所の時刻は「当該事業所ごとに定める」（省令第18条第1項）とされている。実際の時刻はクラブ・自治体ごとにばらばらであり、全国共通の「◯時まで」という値は存在しない。
2. **申込締切日**は、そもそも本基準（省令）に規定がない。学童保育の利用申込みの受付時期・方法は市町村・クラブが独自に定めており、本データセットにも申込締切日の一次情報は収録されていない。
3. **対象学年の実際の受け入れ状況**も、制度上は小学6年生まで対象とされているが、実際にどの学年まで受け入れるかは市町村・クラブごとに異なり、待機児童が最も多いのは小学4年生（5,589人）である（低学年優先の選考により高学年で外れる、いわゆる「小4の壁」）。

以上の理由により、本ツールは**具体的な時刻・締切日を一切出力せず**、「入学前年の秋頃」「入学直前」「入学後」という粒度の粗い時期区分でのチェックリストの提示にとどめる。チェックリストの各項目でも、開所時間・締切時刻に関する項目は必ず「クラブ・自治体ごとに確認してください」という案内で終える。

## YMYL配慮事項

- 「全国一律の開所時間・締切日は存在しない」ことを、根拠・計算式セクションおよびツール本文の両方で明示する。
- 待機児童の学年別内訳（小学4年生が最多）を隠さず提示しつつ、「必ず入れない」という不安を煽る表現は避け、年度途中に空きが出る実態（5月16,330人→10月速報値7,363人）もあわせて提示することで、事実に基づいた前向きな情報提供にする。
- チェックリストの項目自体（「◯◯を確認する」等）は一次情報（統計・法令）に基づくものではなく、一般的に知られている段取りをまとめた自作リストである旨を明記する（ShussanJunbiChecklistと同様の扱い）。
- 準拠年度（2026年度）・出典（こども家庭庁・e-Gov法令検索）・最終確認日（2026-07-17）を画面内に表示する。
