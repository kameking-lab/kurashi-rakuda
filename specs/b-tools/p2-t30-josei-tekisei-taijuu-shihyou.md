## 概要

身長（cm）・体重（kg）・任意で年齢（歳）を入力すると、BMI（体格指数）を計算し、日本肥満学会が定める肥満度分類（低体重〜肥満4度）に基づく現在の区分を表示する。年齢を入力した場合は、厚生労働省「日本人の食事摂取基準（2025年版）」が定める年齢別の「目標とするBMIの範囲」と比較し、範囲の下・範囲内・範囲の上のいずれかを表示する。

※ P2-T15（妊娠中の体重増加チェッカー）は別ツールのため、本ツールでは妊娠中の体重増加指導の目安（表8）は参考情報として同じ肥満度区分の目安値に触れる程度に留め、妊娠週数や目標体重の専用計算は行わない。

データのSSOTは `data/tables/tekisei-taijuu-kijun.json`（BMI計算式=厚生労働省 e-ヘルスネット、肥満度分類=日本肥満学会「肥満症診療ガイドライン」図表 chart_A、目標とするBMIの範囲=厚生労働省「日本人の食事摂取基準（2025年版）」、妊娠中の体重増加指導の目安=こども家庭庁「妊娠前からはじめる妊産婦のための食生活指針 解説要領」表8）。数値はすべて同JSONを参照し、本ツール・本仕様書に直接ハードコードしない。

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| `heightCm`（身長） | 数値（cm） | 必須 | 50以上250以下の数値。範囲外・0・負数・非数値は入力エラー |
| `weightKg`（体重） | 数値（kg） | 必須 | 10以上300以下の数値。範囲外・0・負数・非数値は入力エラー |
| `ageYears`（年齢） | 整数（歳） | 任意 | 0以上120以下の整数。入力時のみ「目標とするBMIの範囲」の判定を追加で行う。未入力ならBMI・肥満度分類のみを表示する |

身長・体重の許容範囲（50〜250cm・10〜300kg）は一次情報に基づく基準値ではなく、明らかな誤入力（例: 身長を m 単位で入力してしまった、体重の桁を1つ多く打った）を防ぐための実装上のガードである。この範囲外の値は「入力内容をご確認ください」というエラーメッセージにとどめ、医学的な上限・下限を意味する表示はしない。

## ロジック仕様

### 1. BMI計算式

```
BMI = 体重(kg) ÷ 身長(m)の2乗
```

`data/tables/tekisei-taijuu-kijun.json` の `bmiFormula`（出典: 厚生労働省 e-ヘルスネット）。身長はcmではなくmで計算する（cm入力を100で割ってから2乗する）。

### 2. 肥満度分類の境界値判定

`obesityClassification.categories`（日本肥満学会の肥満度分類。出典: e-ヘルスネット・こども家庭庁・日本肥満学会）の各区分は次の境界（下限を含み上限を含まない半開区間）で判定する。

| 区分コード | ラベル | 範囲 |
|---|---|---|
| UNDERWEIGHT | 低体重（やせ） | 〜18.5未満 |
| NORMAL | 普通体重 | 18.5以上〜25.0未満 |
| OBESE_1 | 肥満（1度） | 25.0以上〜30.0未満 |
| OBESE_2 | 肥満（2度） | 30.0以上〜35.0未満 |
| OBESE_3 | 肥満（3度） | 35.0以上〜40.0未満 |
| OBESE_4 | 肥満（4度） | 40.0以上 |

判定は `bmiFrom <= BMI < bmiTo` で行う（`bmiFrom` が `null` の区分は下限なし＝`-Infinity`相当、`bmiTo` が `null` の区分は上限なし＝`+Infinity`相当として扱う）。境界値そのもの（18.5・25.0・30.0・35.0・40.0ちょうど）は、その値を`bmiFrom`に持つ上位側の区分に分類する（例: BMI=25.0 ちょうどは「肥満（1度）」）。

### 3. 標準体重（参考値。BMI22基準）

```
標準体重(kg) = 身長(m)の2乗 × 22
```

`bmiAndMortality.idealBmi22`（出典: 日本肥満学会の肥満度分類表 注1）。★データの `note` にある通り、これは「標準体重の計算基準」であり、個人の「理想はBMI22」という断定には使わない。表示は「標準体重の計算基準（BMI22）」というラベルにとどめ、目標値・達成目標として提示しない。

### 4. 目標とするBMIの範囲（年齢入力時のみ）

`targetBmiRange.ranges`（出典: 厚生労働省「日本人の食事摂取基準（2025年版）」表1。18歳以上が対象）の年齢区分は次のとおり。

| 年齢区分 | 目標とするBMIの範囲 |
|---|---|
| 18〜49歳 | 18.5〜24.9 |
| 50〜64歳 | 20.0〜24.9 |
| 65〜74歳 | 21.5〜24.9 |
| 75歳以上 | 21.5〜24.9 |

判定は年齢の`ageFrom <= age`かつ（`ageTo`が`null`でなければ`age <= ageTo`）で該当区分を選ぶ。年齢が18歳未満の場合は、この表の対象外（食事摂取基準の対象は18歳以上）であるため、目標BMI範囲の判定自体を行わず「18歳未満はこの表の対象外」の注記のみを表示する。

BMIと目標範囲を比較し、次の3値のいずれかを返す。

- `belowRange`: BMI < 目標範囲の下限
- `withinRange`: 目標範囲の下限 <= BMI <= 目標範囲の上限
- `aboveRange`: BMI > 目標範囲の上限

### 5. 妊娠中の体重増加指導の目安（参考情報のみ・専用計算はしない）

`pregnancyWeightGain.categories`（出典: こども家庭庁「妊娠前からはじめる妊産婦のための食生活指針 解説要領」表8）は、日本肥満学会の肥満度分類に準じた4区分（低体重／普通体重／肥満1度／肥満2度以上）で体重増加量の目安を示している。本ツールの肥満度分類（6区分）とのマッピングは次のとおり。

| 本ツールの区分 | 表8の区分 |
|---|---|
| UNDERWEIGHT | UNDERWEIGHT |
| NORMAL | NORMAL |
| OBESE_1 | OBESE_1 |
| OBESE_2 / OBESE_3 / OBESE_4 | OBESE_2_PLUS |

現在の区分に対応する表8の目安（例: 普通体重なら「10〜13kg」）を、常に「医師が妊娠中の体重増加の指導を行うときの目安であり、個人差を考慮した指導が必要」という`cautionQuote`とともに参考情報として1行表示する。妊娠週数・出産予定日からの目標体重の逆算、体重増加ペースの計算などの専用ロジックは実装しない（P2-T15の範囲）。

## エッジケース

- **身長・体重が範囲外（0・負数・極端値）**: 入力エラーとして扱い、計算そのものを行わない（NaN表示・無言失敗を避ける）。
- **境界値ちょうど（BMI=18.5・25.0・30.0・35.0・40.0）**: 上位側の区分に分類する（「3. 肥満度分類の境界値判定」参照）。テストケースで全境界を検証する。
- **年齢未入力**: BMIと肥満度分類のみを表示し、「目標とするBMIの範囲」セクションは表示しない（「年齢を入れると比較できます」という案内のみ表示）。
- **年齢18歳未満**: 目標BMI範囲の判定を行わず、対象外である旨を表示する（食事摂取基準の対象外・小児は成長曲線で評価するため）。
- **年齢75歳以上（上限なし区分）**: `ageTo: null` を「上限なし」として扱い、75歳以上はすべて同じ区分（21.5〜24.9）で判定する。
- **BMIが極端に高い・低い値になる組み合わせ（例: 身長250cm・体重10kg）**: 入力自体は許容範囲内でも極端なBMIになりうるが、計算はそのまま行い、肥満度分類の最上位（肥満4度）・最下位（低体重）に分類する。クラッシュしない。

## YMYL配慮事項（煽らない表現の徹底）

- 結果画面には常時、次の趣旨のCalloutを表示する: 「BMIは体格を表す指標のひとつで、筋肉量・体脂肪の分布・年齢によって同じBMIでも健康リスクは異なります。基準の範囲から外れていることは、直ちに病気であることを意味しません。肥満（BMI25以上）だけでは治療の対象にはならず、治療が必要な『肥満症』とは区別されます。」（`data/tables/tekisei-taijuu-kijun.json` の `disclaimer` に基づく）
- 「痩せましょう」「太りすぎです」等の指導・断定・煽り表現は一切行わない。判定ラベルは「低体重（やせ）」「普通体重」「肥満（◯度）」という分類名の提示にとどめ、行動を指示しない。
- 標準体重（BMI22基準）は「標準体重の計算基準」というラベルで表示し、「あなたの理想体重は◯kgです」という断定表示はしない。
- 妊娠中の体重増加の目安を表示する場合も、「医師が指導を行うときの目安であり、個人差を考慮した指導が必要」という原文の注意書きを必ず併記し、専用の計算（増加ペース等）は行わない。
- 減量・体重管理が必要かどうかの判断は必ず医師に相談するよう案内する（disclaimer準拠）。
