# P4-T05・P4-T06 犬猫の年齢換算（人間年齢換算）＋食事量目安

## 概要

犬・猫の年齢を入力すると人間年齢の目安を計算し（P4-T06）、体重とフードの代謝エネルギー（ME）を入力すると1日の食事量の目安を計算する（P4-T05）、需要検証枠のツール。2つのBACKLOGキューを1本のツールに統合して実装した。

- BACKLOG.md queueId: `P4-T05`（犬猫の食事量計算）・`P4-T06`（犬猫の年齢換算）。registry.json 上は `P4-T06` を代表queueIdとして登録（1エントリ1queueIdの制約のため）。
- slug: `inuneko-nenrei-shokujiryou`
- category: `kaji`（「暮らし全般」枠。BACKLOG.mdの指示どおりカテゴリは新設せず、audience は universal のまま需要検証する。需要検証の結果〈アクセス・未解決クエリ〉を見てカテゴリ新設を再判断する）
- 実装: `components/tools/impl/InunekoNenreiShokujiryou.calc.ts`（ロジック）／`components/tools/impl/InunekoNenreiShokujiryou.tsx`（UI）
- テスト: `tests/inuneko-nenrei-shokujiryou.test.ts`
- SSOT: `data/tables/inuneko-nenrei-shokujiryou.json`（本ツールが使う唯一のデータソース）
- YMYL区分: 低〜中（獣医療判断は行わない。食事量は「目安」であり、パッケージ表示・獣医師の指導を優先する旨を常時表示する）

## 出典（一次情報）

環境省 自然環境局 総務課 動物愛護管理室「飼い主のためのペットフード・ガイドライン ～犬・猫の健康を守るために～」（平成30年8月改訂版）。

- ランディングページ: https://www.env.go.jp/nature/dobutsu/aigo/2_data/pamph/petfood_guide_1808.html（`checkedAt: 2026-07-21`）
- 年齢換算表・計算式（2.市販のペットフードについて、9ページ「犬と人間、猫と人間の年齢の目安」）: https://www.env.go.jp/nature/dobutsu/aigo/2_data/pamph/petfood_guide_1808/pdf/4.pdf（`checkedAt: 2026-07-21`）
- 食事量の計算式（4.体調管理について、13ページ「食事量を計算してみよう」MEMO欄）: https://www.env.go.jp/nature/dobutsu/aigo/2_data/pamph/petfood_guide_1808/pdf/6.pdf（`checkedAt: 2026-07-21`）

このガイドラインは環境省の「飼い主のためのペットフード・ガイドライン」紹介ページ（petfood_guide.html）が「改訂されました」として案内する改訂版であり、旧版（petfood_guide.html直下のfull.pdf）ではなく、この改訂版（petfood_guide_1808配下）を出典として採用した。

上記PDFはテキスト抽出できない埋め込みフォントのため、ページを画像化して目視で内容を確認した（本文中の表・数値は原典の表記をそのまま転記し、数値の改変は行っていない）。年齢換算の計算式は、表に掲載された全ての数値点（小〜中型犬・猫9点、大型犬8点）と実際に照合し、以下の1点を除いてすべて一致することを確認済み。

- 小〜中型犬・猫の1歳（表の値=15歳）は、計算式 `24+(年齢-2)×4` をそのまま1歳に当てはめると20歳になり、表の値と一致しない。原典の説明文「最初の2年で24歳、3年目からは1年に4歳ずつ年をとる」を踏まえ、1歳は表の値（15歳）による特例、2歳以降は計算式、という原典の構成をそのまま実装した（捏造ではなく原典の構成の忠実な再現）。
- 大型犬は1歳から計算式 `12+(年齢-1)×7` がそのまま全ての表の値と一致するため、特例は設けていない。

## ロジック仕様

### 1. 年齢換算（`calcNenreiKansan`）

```
if species === "cat" && size === "ougata":
    → エラー（原典に猫の大型区分は存在しない）
if 年齢が非数値、0以下、または30超:
    → 入力エラー
if 0 < 年齢 < 1:
    → 「1歳未満は原典に換算根拠がないため計算できません」エラー

if size === "shouChuugata":
    humanAge = 年齢 === 1 ? 15 : 24 + (年齢 - 2) × 4
else:  # ougata（大型犬）
    humanAge = 12 + (年齢 - 1) × 7

isBeyondTableRange = 年齢 > その区分の表の最大掲載年齢（小〜中型=20、大型=15）
```

`isBeyondTableRange` が真の場合、UIに「表に直接掲載されている範囲を超えた計算式の延長値」である旨を明示する（表の実測点を超えた外挿であることを隠さない）。

### 2. 食事量目安（`calcShokujiryou`）

```
if 体重が非数値、2kg未満、または45kg超:
    → 入力エラー（原典の計算式の有効範囲外）
if ME入力あり かつ（非数値 または 0以下）:
    → 入力エラー

RER = 体重(kg) × 30 + 70
係数 = species === "dog" ? 1.6 : 1.2   # 原典の計算例（避妊・去勢済み成犬・成猫）のみ
DER = RER × 係数

if ME入力あり:
    食事量(g) = DER ÷ ME × 100
else:
    食事量 = null（RER・DERまでを表示し、食事量(g)は表示しない）
```

## エッジケース

- **猫に「大型犬」区分を選ぼうとする**: UI側でも猫選択時は体格セレクトを表示せず、常に「小〜中型犬・猫」の式を使う（原典に猫の大型区分の表・式が存在しないため）。
- **1歳未満（生後1年未満）の年齢入力**: 原典の表・式は1歳から始まっており、0〜1歳の換算根拠（子犬・子猫期の人間換算年齢）は記載がない。補間や外挿で数値を作らず、明確なエラーメッセージで非対応であることを示す。
- **表の最大掲載年齢（小〜中型20歳・大型15歳）を超える入力**: 原典の計算式は連続関数のため機械的には計算できるが、表に直接掲載されている実測点の範囲外であることを`isBeyondTableRange`で明示し、UIにも注記を出す。
- **食事量の体重が2kg未満・45kg超**: 原典が「RERは体重2〜45kgの場合のみ計算可能」と明記しているため、範囲外はエラーとして扱い、範囲外でも計算式を無理に適用しない。
- **DERの係数（成長期・妊娠授乳期・未避妊去勢・シニア期等）**: 原典に具体的な数値が明記されているのは「避妊・去勢済み成犬・成猫」の計算例（犬1.6・猫1.2）のみで、他のライフステージの係数は原典に表がなく「詳しくは獣医師に相談しましょう」とだけ案内されている。本ツールはこの2つの係数のみを扱い、他のライフステージ向けの係数を一般的な獣医栄養学の知見（WSAVA等）から補って表示することはしない（本サイトの一次情報主義・捏造禁止の方針に従う）。
- **フードのMEを入力しない場合**: RER・DERまでを結果として表示し、食事量（g）は表示しない（原典の式の後半＝ME依存部分だけを未入力のまま計算しない）。

## YMYL配慮事項

- 結果画面には常時、次のCalloutを表示する: 「まずはフードのパッケージに表示されている給餌量を目安にしてください。ここでの計算はあくまで参考値で、季節・運動量・体調によって必要な量は変わります。体重は現在の体重ではなく理想体重を使うのが望ましいとされており、理想体重が分からない場合は獣医師にご相談ください。」
- 食事量セクションの冒頭に「避妊・去勢済みの成犬・成猫を対象にした計算例に基づく」旨と、他のライフステージには使えない旨を明示する。
- 獣医療上の診断・治療方針の判断は一切行わない。
- 出典（環境省）・最終確認日（2026-07-21）を画面内（根拠・計算式セクション、出典3点セット）に表示する。
