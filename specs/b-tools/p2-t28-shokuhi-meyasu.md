# 食費の目安計算（P2-T28）

## 概要

世帯人員数（1人・2人・3人・4人・5人・6人以上）を入力すると、総務省統計局「家計調査（家計収支編）」の世帯人員別の実額データに基づく、1か月当たりの食費（食料費）の目安と、その内訳（外食・調理食品〈中食〉・その他）を表示するツール。

- docs/02 **#61**（カテゴリ4: お金、優先度 中、難易度B）。BACKLOG.md queueId: `P2-T28`。
- データ: `data/tables/shokuhi-meyasu.json`（総務省統計局「家計調査（家計収支編）」2025年（令和7年）平均、2026年2月6日公表）を単一の情報源（SSOT）とする。金額はすべて同JSONを参照し、本ツール・本仕様書に直接ハードコードしない（可読性のため転記している数値は「データ表・出典」節を参照）。
- YMYL区分: 中（お金に関わる目安。個別の家計診断・節約指導は行わない）。表示する金額はあくまで統計上の平均値であり、実際の食費は世帯ごとに大きく異なる旨を常時表示する。

## 入力仕様

| フィールド名 | 型 | 必須/任意 | バリデーションルール |
|---|---|---|---|
| 世帯人員数（setaininzuu） | 整数 | 必須 | 1以上の整数であること。小数・0以下・非数値は入力エラー。UI上の入力欄は1〜10の範囲を想定するが、ロジック自体は1以上の任意の整数を受け付ける（6以上はすべて同じ区分にまとめる） |

世帯構成（単身/夫婦のみ/夫婦+子どもなど）は入力項目としない。理由は「データ表・出典」節および「エッジケース」節を参照（総務省統計局の当該統計表は世帯人員のみで区分されており、世帯構成別の実額データは公表されていないため、補間・按分による推測値の表示を避ける）。

## ロジック仕様

### 1. 世帯人員区分の判定（バケット化）

```
if setaininzuu が整数でない、または 1未満:
    → 入力エラー
elif setaininzuu == 1:
    bucket = tanshin_setai（単身世帯）
elif setaininzuu == 2:
    bucket = futari_ijou_setai.ninin（2人世帯）
elif setaininzuu == 3:
    bucket = futari_ijou_setai.sannin（3人世帯）
elif setaininzuu == 4:
    bucket = futari_ijou_setai.yonin（4人世帯）
elif setaininzuu == 5:
    bucket = futari_ijou_setai.gonin（5人世帯）
else:  # setaininzuu >= 6
    bucket = futari_ijou_setai.rokunin_ijou（6人以上世帯。実際の平均世帯人員は6.3人）
```

入力した世帯人員数が6を超える場合（例: 7人、10人）も、すべて「6人以上」区分の同じ実額を返す（総務省統計局の集計自体が「6人以上」までしか区分していないため、7人・10人の按分値を作らない）。

### 2. 表示する金額

各区分から次の4項目をそのまま表示する（丸め・加工なし。データ表の値は既に整数円）。

- `shouhiShishutsu`: 消費支出（参考情報として表示。食費以外を含む生活費全体）
- `shokuryou`: 食料（食費の目安・メイン表示）
- `gaishoku`: 外食
- `chouriShokuhin`: 調理食品（惣菜・弁当などの中食）

加えて、内訳の残り（食料のうち外食・調理食品を除いた部分＝食材購入・飲料・酒類などを含む）を次の式で導出する。

```
sonota = shokuryou - gaishoku - chouriShokuhin
```

### 3. 入力値と実際に使われた区分が異なる場合の注記

`setaininzuu >= 6` かつ `setaininzuu !== 6.3`（常に真。データの6人以上区分は平均6.3人のため）の場合、「入力した人数（例: 8人）そのものの統計はなく、『6人以上』区分（平均世帯人員6.3人）の実額を表示している」旨をUIに明示する。1〜5人の場合はこの注記は不要（統計上の区分とちょうど一致するため）。

## エッジケース

- **世帯人員6人を超える入力（例: 8人、20人）**: 総務省統計局の統計表自体が「6人以上」までしか区分していない。7人・10人などに応じた按分値を作らず、すべて「6人以上」区分（実額は6人〜6.3人相当の平均値）をそのまま表示し、入力した人数と統計区分の人数が一致しない旨を明記する。
- **世帯構成別（単身/夫婦のみ/夫婦+子どもなど）の入力は提供しない**: 総務省統計局「家計調査（家計収支編）」の世帯人員別集計は、世帯人員数のみで区分されており、同じ人数でも世帯構成（例: 夫婦のみの2人世帯 と 親1人子1人の2人世帯）による実額の違いは同統計からは取得できない。存在しない粒度のデータを補間・按分で作成しないという方針（BACKLOG.md P2-T28の指示）に従い、本ツールは世帯人員数のみを入力とする。
- **0人・負の数・小数の入力**: 入力エラーとして扱う（「世帯人員数は1人以上の整数で入力してください」等）。
- **単身世帯と二人以上世帯の集計方法の違い**: 家計調査は単身世帯（約750世帯規模）と二人以上の世帯（約8,000世帯規模）を別々に調査・集計しており、公表上も別の統計表である。本ツールは「1人」の結果として単身世帯の実額をそのまま表示するが、これは総務省統計局が定義する連続した「世帯人員1〜6人以上」という単一の系列ではなく、別集計を便宜的につないでいる点をデータ表の `$comment` に明記している（ツールの表示自体では「統計上の平均・目安」であることのみを明示し、集計方法の技術的差異はデータ表側の注記に留める）。
- **物価変動・調査年の違い**: 表示する金額は2025年（令和7年）平均の実額であり、翌年以降の物価変動を反映していない。次回の「◯◯年平均」公表時にデータを更新する（`next_check_due` 参照）。

## YMYL配慮事項

- 結果画面には常時、次のCalloutを表示する: 「この金額は総務省統計局『家計調査』による世帯人員別の平均値（目安）です。地域・生活スタイル・自炊の頻度などによって実際の食費は大きく異なります。この金額に合わせることを目指す必要はありません。」
- 節約や達成を煽る表現（「◯円以内に抑えましょう」等の指導・断定）は一切行わない。あくまで統計上の目安を「知る」ためのツールとして位置づける。
- 準拠年（2025年（令和7年）平均）・出典（総務省統計局）・最終確認日（2026-07-17）を画面内に表示する。
