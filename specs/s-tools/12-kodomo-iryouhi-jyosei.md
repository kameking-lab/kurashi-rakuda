## 概要

子ども医療費助成は国の法定制度ではなく市区町村ごとの地方単独事業のため、**個別自治体の助成額は算出しない**。全国調査の統計値をもとに「この学齢を対象にしている自治体が全国で何割か」という早見・傾向のみを示し、実際の内容は市区町村で確認するよう促す。数値は捏造しない。

## 入力仕様

| フィールド | 型 | 必須 | ルール |
|---|---|---|---|
| `stage`（学齢区分） | 0〜4 | 必須 | 0=未就学児/1=小学生/2=中学生/3=高校生年代/4=18歳超 |
| `care`（通院/入院） | enum | 必須 | outpatient / inpatient |

## ロジック仕様

```
buckets = 対象年齢の分布（preschool→maxStage0, age12→1, age15→2, age18→3, age20/age22→4）
          ※通院は6バケット、入院はage18までの4バケットのみ調査に存在

notCovered = Σ bucket.count（bucket.maxStage < stage）
covered    = TOTAL_MUNICIPALITIES(1741) − notCovered   // ★total差で算出★
```

★total差で算出する理由★ 入院の対象年齢分布は age18 までしか調査に内訳がなく、列挙合計は1,724で総数1,741に17足りない（＝18歳超を対象にする17自治体の内訳が非公表）。バケットを足し上げると入院の高年齢分を取りこぼすため、「対象外＝自分の学齢より上限が低い自治体」を総数から引く方式にする。これにより stage4（18歳超）の入院covered＝1741−1724＝17 が正しく回収される。

所得制限・一部自己負担の分布は `incomeLimit` / `copayment` から care 別に読む。

## このツールが行わないこと

- 個別自治体の助成額・対象年齢・所得制限額の表示（データが存在しない＝捏造しない）。
- 医療費の窓口負担・自己負担額の計算。
