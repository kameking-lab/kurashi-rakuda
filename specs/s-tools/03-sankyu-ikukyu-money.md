## 1. 概要

出産手当金 ＋ 出産育児一時金 ＋ 育児休業給付金 ＋ 出生時育児休業給付金 ＋ 出生後休業支援給付金 ＋ 社会保険料免除を、**産休前から復職までの時系列で1本のタイムラインとして総額表示**するツール。

kurashi-keisan.jp の既存 `shussan-teate` / `ikukyu-kyufu` / `shussan-ikuji-ichijikin` を**統合**する（[02_ツール一覧](../../docs/02_ツール一覧.md) の棲み分け方針: 統合＝本サイトの複合ツールの一部品として吸収。単発計算はリンクで送客）。

### 差別化の核心

競合は**単発計算の寄せ集め**。「出産手当金はいくら」「育休給付はいくら」を別々のページで計算させ、利用者が自分で足し算し、期間の重なりを自分で考える必要がある。

本ツールの独自価値は **①時系列で並べること ②2025年4月創設の新給付に対応していること**。

とくに **出生後休業支援給付金**（2025年4月創設、雇用保険法第61条の10）は、育児休業給付金67%に13%を上乗せして**合計80%**にする制度で、競合ツールの多くが未対応。

### このツールが答える問い

1. 産休・育休の間、**いつ・いくら**入ってくるのか（タイムライン）
2. **合計いくら**もらえるのか
3. 「手取り10割」と聞いたが、**私の場合は本当か**（上限に達しないか）
4. 夫婦でどう取ると**世帯の総額が最大化**するか

### 入力設計の判断

- **#2 を任意にする理由**: 利用者の大半は出産前に試算する。「実際の出産日」を必須にすると使えない。予定日ベースで試算し、「出産日が予定日とずれた場合は結果が変わります」と注記する。
- **#11 が本ツールの隠れた価値**: 「夫が育休を取らないから13%上乗せは無理」と諦めている利用者が、例外7類型（配偶者が無業者・自営業者・産後休業中など）で**対象になる**ケースは多い。ここを掘り起こすのは競合にない体験。

## 4. 計算式

すべて `ikukyu-kyufu.json` の値のみを使う。**ハードコード禁止。**

### 4.1 出産手当金

```
標準報酬日額 = （支給開始日以前の直近12か月の各月の標準報酬月額の平均） ÷ 30
支給日額     = 標準報酬日額 × 2/3
支給総額     = 支給日額 × 支給日数
```

**根拠**: `data.shussanTeateKin.baseFormula` / `.rate`
**出典**: 健康保険法第102条・第99条第2項（e-Gov 法令検索）

#### 支給期間

| 区分 | 日数 | 根拠 |
|---|---|---|
| 産前（単胎） | 42日 | `periodBeforeBirth` |
| 産前（多胎） | 98日 | `periodBeforeBirthMultiple` |
| 産後 | 56日 | `periodAfterBirth`（多胎でも同じ） |

起算点は「出産の日（**出産の日が出産の予定日後であるときは、出産の予定日**）以前42日」。

> ⚠️ **端数処理（法令で規定）**: `rate` の note より
> - 標準報酬日額（30分の1の額）: **5円未満切捨・5円以上10円未満は10円に切上**
> - 2/3 後の額: **50銭未満切捨・50銭以上1円未満は1円に切上**
>
> ⚠️ **`rate` の value は 0.6667（近似値）**。法令上は「3分の2」。**計算は必ず `× 2 ÷ 3` の有理数で行うこと。**`0.6667` を乗じてはいけない（`rate.note` に明記済み）。

#### 被保険者期間が12か月未満の場合

①直近の継続した各月の標準報酬月額の平均の30分の1、②**前年度9月30日における全被保険者の標準報酬月額の平均の30分の1** のいずれか**少ない額** × 2/3。

> ⚠️ **②の令和8年度の具体値は未収集**（`baseFormulaUnder12Months.note`）。→ #6 が `under12months` の場合は「勤務先・加入健康保険にご確認ください」と表示し、**数値を出さない**。

### 4.2 出産育児一時金

| 条件 | 金額 | 根拠 |
|---|---|---|
| 産科医療補償制度加入機関・在胎22週以降 | **500,000円** | `amount` |
| 上記以外 | **488,000円** | `amountWithoutCompensation` |

多胎の場合は**児の数だけ支給**（双子なら100万円）。
支給要件: 妊娠85日（4か月）以降の生産・死産・流産・人工妊娠中絶（`eligibilityWeeks`）。

**出典**: 全国健康保険協会「出産育児一時金」／健康保険法施行令第36条

> **正確な理解**（`statutoryBase.note`）: 「50万円 = 48.8万円 + 1.2万円」の 1.2万円は**法令が直接定めた額ではない**。施行令第36条ただし書は「**3万円を超えない範囲内で保険者が定める金額**」と規定し、実際に1.2万円となるのは産科医療補償制度の掛金が1万2千円だから。→ コンテンツで「1.2万円は法定額」と書かないこと。

### 4.3 育児休業給付金

```
休業開始時賃金日額 W = （休業開始前 直近6か月に支払われた賃金の総額） ÷ 180
  ※ 臨時の賃金・賞与は除く
  ※ 上限 16,110円 / 下限 3,014円 でクランプ

支給額（支給単位期間ごと）= W × 支給日数（原則30日）× 給付率
  給付率 = 67%（通算180日目まで）／ 50%（181日目以降）
```

**根拠**: `data.ikujiKyugyoKyufuKin`
**出典**: 雇用保険法第61条の7／厚生労働省「育児休業等給付の内容と支給申請手続（令和7年8月1日改訂版）」

| 項目 | 値 | 根拠 |
|---|---|---|
| 賃金日額 上限 | 16,110円 | `wageDailyMax` |
| 賃金日額 下限 | 3,014円 | `wageDailyMin` |
| 支給上限額（67%・30日） | 323,811円 | `monthlyMax67`（検算: 16,110×30×67% ✅） |
| 支給下限額（67%・30日） | 60,581円 | `monthlyMin67` |
| 支給上限額（50%・30日） | 241,650円 | `monthlyMax50`（検算: 16,110×30×50% ✅） |
| 支給下限額（50%・30日） | 45,210円 | `monthlyMin50` |

> 🚨 **これらの額は「令和8年7月31日までの額」。2026年8月1日に改定される。**
> `amendments[0]` の `expiresOn: 2026-07-31`。2026-07-17 時点で新額は**未公表**（例年7月下旬公表）。
> `scripts/verify-seido.mjs` が期限切れをエラーとして検出する。**2026年8月1日以降に公開する場合、公開前に必ず差し替えること。**（[queue §8](../../queue/hoikuryo-backlog.md) ★最優先★）

> ⚠️ **母親の賃金日額の起算点**（`wageDailyFormula.note`）: 育休開始前は産休中で賃金がないため、**産前休業開始日の前6か月**が基準になる。「育休開始前6か月」で計算すると賃金ゼロの月が入り、**給付額を大幅に過小評価する**。★最頻出のバグ★

#### 受給要件

- **被保険者期間**: 休業開始日前2年間に、賃金支払基礎日数11日以上（ない場合は80時間以上）の完全月が**12か月以上**（`insuredPeriodRequirement`）
- **就業日数**: 一支給単位期間中の就業日数が**10日以下**（10日超なら就業時間**80時間以下**）（`workDaysLimit` / `workHoursLimit`）
- **回数**: 同一の子について**2回**まで（3回目以後は不支給）（`maxLeaveCount`）

#### 支給単位期間

「育児休業を開始した日から起算した**1か月ごとの期間**」（休業開始応当日から各翌月の休業開始応当日の前日まで）。

> ⚠️ **暦月ではない**（`supportUnitPeriod.note`）。1/15開始なら 1/15〜2/14 が第1支給単位期間。★実装で最も間違えやすい★

#### 就業して賃金が支払われた場合の調整

| 賃金額 ÷ (W × 日数) | 支給額 | 根拠 |
|---|---|---|
| ≦ 13%（181日目以降は30%） | 全額（減額なし） | `wageAdjustment.thresholdNoReduction` |
| 13%（30%）超 〜 80%未満 | **W × 日数 × 80% − 賃金額** | 雇用保険法第61条の7第7項 |
| ≧ 80% | **不支給** | `wageAdjustment.thresholdZero` |

### 4.4 出生後休業支援給付金（★競合が対応していない領域★）

```
支給額 = W × 対象期間内の休業日数（上限28日）× 13%
```

**根拠**: `data.shusshoGoKyugyoShienKyufuKin`
**出典**: 雇用保険法第61条の10

| 項目 | 値 | 根拠 |
|---|---|---|
| 支給率 | 13% | `rate` |
| 上限日数 | 28日 | `maxDays` |
| 支給上限額（28日） | 58,640円 | `monthlyMax`（検算: 16,110×28×13% = 58,640.4 → 58,640 ✅） |
| 支給下限額（28日） | 10,970円 | `monthlyMin` |

#### 受給要件

```
① 被保険者期間: 休業開始日前2年間にみなし被保険者期間12か月以上
② 本人の出生後休業が通算 14日以上                      … ownLeaveRequirement
③ 配偶者が、子の出生日から8週間を経過する日の翌日までの期間内に
   通算 14日以上の出生後休業をしたこと                   … spouseLeaveRequirement
```

#### ③の例外（7類型）— ★このツールの隠れた価値★

**判定時点は「子の出生日の翌日において」**（`spouseRequirementExceptions.note`）:

1. 配偶者がいない
2. 配偶者が被保険者の子と法律上の親子関係がない
3. 被保険者が配偶者から暴力を受け別居中
4. 配偶者が無業者
5. 配偶者が自営業者やフリーランスなど雇用される労働者でない
6. 配偶者が産後休業中
7. 1〜6以外の理由で配偶者が育児休業をすることができない

> ⚠️ **母親の場合の重要な注意**（`spouseRequirementExceptions.note`）:
> 母親は通常「6. 配偶者が産後休業中」に該当するのではなく、**母親自身の産後休業が要件②（本人の14日以上）を満たす**形になる。
> → UI では「**夫が育休を取らなくても、あなたが対象になる場合があります**」と明示し、7類型を選択させる。

#### 賃金が支払われた場合の調整（育休給付との差異）

育児休業給付金が減額される場合でも**出生後休業支援給付金は減額されない**。ただし賃金が `W × 休業日数` の**80%以上**となり育児休業給付金が支給されない場合は、出生後休業支援給付金も**支給されない**（`wageAdjustment`）。

### 4.5 出生時育児休業給付金（産後パパ育休）

```
支給額 = W × 休業日数（上限28日）× 67%
```

対象期間: 子の出生日から起算して**8週間を経過する日の翌日**までの期間内。当該期間内に**4週間以内**の期間を定めた休業。分割2回まで。

支給上限額（28日）: **302,223円**（`monthlyMax`。検算: 16,110×28×67% ✅）

> ⚠️ **就業日数の比例縮小**（`workDaysLimit.note`）: 「最大10日」は28日休業の場合。休業期間が短ければ比例して短くなる。
> - 14日間の休業 → 最大5日（5日超なら40時間）
> - 10日間の休業 → 最大4日（10日×10/28 ≒ 3.57 を**切上**）、80時間×10/28 ≒ **28.57時間（端数処理なし）**

### 4.6 社会保険料免除

#### 産前産後休業

「産前産後休業を開始した日の属する月から、**産前産後休業が終了する日の翌日が属する月の前月**まで」（`maternityLeave.rule`）。

> **産休には14日ルールはない**（条文上、月末在籍ベースの原則のみ）。

#### 育児休業等

| ルール | 内容 | 根拠 |
|---|---|---|
| 月末在籍 | 開始月と終了日翌日の属する月が**異なる**場合、開始月〜終了日翌日の属する月の前月まで免除（実質「月末時点で育休中」） | `childcareLeave.monthEndRule` |
| 同月内14日以上 | 開始月と終了日翌日の属する月が**同一**でも、その月に**14日以上**取得していれば当該月を免除 | `childcareLeave.fourteenDayRule` |
| 賞与の1か月超 | 育休期間が**1か月以下**の者は標準報酬月額に係る保険料のみ免除＝**賞与保険料は免除されない**。賞与保険料は「賞与を支払った月の末日を含んだ**連続した1か月を超える**育休」で免除。1か月超かは**暦日**で判断 | `childcareLeave.bonusRule` |

**免除の効果**: 被保険者・事業主の**両方**の負担が免除され、かつ**保険料を納めた期間として扱われる**（年金額計算上不利にならない）（`effect`）。

> ⚠️ **対象年齢の差異**（`childcareLeave.targetAge.note`）: 雇用保険の育休給付は原則1歳（最長2歳）までだが、**社会保険料免除は3歳未満まで**対象。

> ⚠️ **自動ではない**（`procedure.note`）: 事業主が「育児休業等取得者申出書」「産前産後休業取得者申出書」を提出する必要がある。→ UI に「勤務先に申出をしてもらう必要があります」と明示。

### 4.7 「手取り10割相当」の説明（★誤解を解く責務★）

```
育児休業給付金 67% + 出生後休業支援給付金 13% = 合計 80%（最大28日間のみ）
```

**根拠**: `data.handoriJuwariSoutou`

> 🚨 **最重要の誤解ポイント**（`combinedRate.note`）:
> **雇用保険法第61条の7の給付率が80%に引き上げられたわけではない。** 80%は「67% + 13%」という**2つの別給付の合算**であり、出生後休業支援給付金の対象となる**最大28日間のみ**。29日目以降は67%（181日目以降50%）に戻る。

**手取り10割相当となる理由**（`reason`）: ①健康保険・厚生年金保険料が免除され、②勤務先から給与が支給されない場合は雇用保険料の負担がなく、③**育児休業等給付は非課税**であるため。

> ⚠️ 厚労省自身が「**相当**」と留保している概算。**賃金日額の上限額（16,110円）の制約があるため、賃金水準が高い人は手取り10割に届かない。**
> → **月給が約48万円を超える利用者には「上限に達しています」の警告を必ず出す**（16,110円 × 30日 ÷ 0.67 ≒ 721,000円の賃金月額で67%給付が上限に張り付く。実際には W = 月給×6÷180 = 月給÷30 なので、**月給 ≒ 483,300円で W = 16,110円に到達**）。

**非課税の副次効果**（`taxExempt.note`）: 育児休業等給付は所得税・住民税ともに非課税で、**翌年の住民税・保育料の算定基礎にも入らない**。→ **復職後の保育料が下がる要因**になる。[01_hoikuryo-keisan](01-hoikuryo-keisan.md) へ送客する導線をここに置く。

### 9.3 罠（★実装前に必ず読む★）

1. **母親の賃金日額の起算点**: 育休開始前は産休中で賃金がない。**産前休業開始日の前6か月**が基準（`wageDailyFormula.note`）。「育休開始前6か月」で実装すると給付額が激減する。**テストケース #42 で固定。**
2. **出産日が予定日とずれた場合**: 産前期間の起算点は「出産の日（**出産の日が出産の予定日後であるときは、出産の予定日**）」。**遅れた場合、遅れた日数も産前期間に含まれ支給対象**。早まった場合は短くなる。**テストケース #40 / #41 で固定。**
3. **支給単位期間は暦月ではない**: 休業開始応当日ベース（`supportUnitPeriod.note`）。**テストケース #34 で固定。**
4. **80%は67%+13%の合算であり28日限定**: 給付率が80%になったのではない（`combinedRate.note`）。コンテンツで「育休給付が80%に引き上げられました」と**絶対に書かない**。
5. **社保料免除の対象年齢は3歳未満**（雇用保険の育休給付は原則1歳）。混同しない（`targetAge.note`）。
6. **出生後休業支援給付は減額されない**が、育休給付が80%以上の賃金で不支給になると**連動して不支給**になる（`wageAdjustment`）。
7. **`rate` の value は近似値**（0.6667 / 0.13 等）。法令上の分数で計算すること。
